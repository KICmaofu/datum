Windows 常用命令覆盖 **系统基础操作、文件管理、网络排查、进程服务、系统维护** 等核心场景，以下按功能分类整理，附 **功能说明+实战示例**，兼顾日常使用和应急排查，新手也能直接套用：


## 一、系统基础命令（必备）
### 1. 系统信息查询
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `winver`            | 查看 Windows 版本（如 Windows 10 22H2）   | 直接输入 `winver` 回车，弹出版本窗口       |
| `systeminfo`        | 查看系统详细信息（CPU、内存、补丁、网卡等）| 输入 `systeminfo` 回车，或导出文件：`systeminfo > C:\系统信息.txt` |
| `ver`               | 快速查看系统版本号（命令行输出）          | `ver` → 输出：`Microsoft Windows [版本 10.0.19045.3930]` |
| `hostname`          | 查看计算机名称                            | `hostname` → 输出：`DESKTOP-ABC123`        |

### 2. 用户与权限管理（需管理员权限）
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `net user`          | 查看所有本地用户                          | `net user`                                |
| `net user 用户名 密码` | 创建/修改用户密码                          | `net user test 123456` → 创建密码为123456的test用户 |
| `net user 用户名 /del` | 删除本地用户                              | `net user test /del` → 删除test用户        |
| `net localgroup administrators 用户名 /add` | 将用户添加到管理员组 | `net localgroup administrators test /add` → 让test用户获得管理员权限 |
| `whoami`            | 查看当前登录用户（含权限信息）            | `whoami` → 输出：`desktop-abc123\test`    |

### 3. 系统快捷操作
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `shutdown -s -t 0`  | 立即关机（-t 0 表示延迟0秒）              | `shutdown -s -t 0`                        |
| `shutdown -r -t 0`  | 立即重启                                  | `shutdown -r -t 0`                        |
| `shutdown -a`       | 取消关机/重启计划                          | 误操作后输入 `shutdown -a` 回车           |
| `logoff`            | 注销当前用户                              | `logoff`                                  |
| `control`           | 打开控制面板                              | `control`                                  |
| `services.msc`      | 打开服务管理窗口（启动/停止服务）          | `services.msc`                            |
| `taskmgr`           | 打开任务管理器                            | `taskmgr`（或直接按 Ctrl+Shift+Esc）       |


## 二、文件与目录管理（高频使用）
### 1. 基础导航与查看
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `cd`                | 切换目录（Change Directory）               | `cd C:\Users\Desktop` → 切换到桌面；`cd..` → 返回上一级目录 |
| `dir`               | 查看当前目录下的文件/文件夹               | `dir` → 显示详细信息（大小、修改时间）；`dir /a` → 显示隐藏文件 |
| `pwd`               | 查看当前目录的完整路径（PowerShell 可用）  | `pwd` → 输出：`C:\Users\Desktop`           |
| `tree`              | 以树形结构显示目录（直观查看文件夹层级）   | `tree C:\Users` → 显示Users目录的树形结构  |

### 2. 文件/文件夹操作
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `md 文件夹名`       | 创建文件夹（Make Directory）               | `md 测试文件夹` → 在当前目录创建“测试文件夹” |
| `rd 文件夹名`       | 删除空文件夹（Remove Directory）           | `rd 测试文件夹` → 仅删除空文件夹；`rd /s /q 测试文件夹` → 强制删除非空文件夹（/s递归，/q静默） |
| `copy 源文件 目标路径` | 复制文件                                  | `copy C:\test.txt D:\` → 把C盘的test.txt复制到D盘 |
| `xcopy 源目录 目标路径 /s` | 复制文件夹（含子目录）                    | `xcopy C:\测试文件夹 D:\备份 /s` → 复制文件夹及子文件 |
| `move 源文件 目标路径` | 移动/重命名文件/文件夹                    | `move C:\test.txt D:\` → 移动文件；`move C:\test.txt C:\new.txt` → 重命名 |
| `del 文件名`        | 删除文件（无法恢复，慎用）                | `del test.txt` → 删除当前目录的test.txt；`del *.txt` → 删除所有txt文件 |
| `type 文件名`       | 查看文本文件内容（命令行输出）            | `type C:\test.txt` → 显示文件内容          |
| `ren 旧文件名 新文件名` | 重命名文件/文件夹                        | `ren test.txt new.txt` → 重命名文件        |


## 三、网络操作命令（排查网络问题必备）
### 1. 网络配置与状态
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `ipconfig`          | 查看网卡IP、子网掩码、网关、DNS（基础）    | `ipconfig` → 快速查看；`ipconfig /all` → 查看详细信息（含MAC地址、DNS） |
| `ipconfig /release` | 释放DHCP分配的IP地址                      | 断网后重新获取IP时使用                    |
| `ipconfig /renew`   | 重新获取DHCP IP地址                       | `ipconfig /release && ipconfig /renew` → 释放+重新获取 |
| `ipconfig /flushdns` | 刷新DNS缓存（解决域名解析失败、网页打不开）| `ipconfig /flushdns` → 提示“成功刷新DNS解析缓存” |

### 2. 网络连通性测试
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `ping 目标IP/域名`  | 测试网络连通性（发送ICMP数据包）          | `ping 8.8.8.8`（测试谷歌DNS，外网）；`ping localhost`（测试本地）；`ping -n 4 baidu.com`（发送4个数据包后停止） |
| `tracert 目标IP/域名` | 跟踪数据包传输路径（排查网络延迟/中断点） | `tracert baidu.com` → 显示每一跳的延迟，红色超时表示该节点不可达 |
| `pathping 目标IP/域名` | 结合ping和tracert，统计丢包率（更精准）  | `pathping baidu.com` → 耗时稍长，适合深度排查 |

### 3. 端口与连接管理
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `netstat -ano`      | 查看所有端口连接（TCP/UDP）、进程PID       | `netstat -ano | findstr ":8080"` → 查找8080端口的占用进程（PID） |
| `netstat -tuln`     | 查看监听中的TCP/UDP端口（仅显示端口号）    | `netstat -tuln` → 过滤无关信息，快速看监听端口 |
| `netsh advfirewall firewall add rule name="允许8080端口" dir=in action=allow protocol=TCP localport=8080` | 开放防火墙端口（需管理员） | 开放8080/TCP端口，允许外部访问 |
| `netsh advfirewall firewall delete rule name="允许8080端口"` | 关闭防火墙端口（需管理员） | 删除上述开放的8080端口规则 |


## 四、进程与服务管理
### 1. 进程管理
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `tasklist`          | 查看所有运行的进程（名称、PID、内存占用）  | `tasklist` → 显示所有进程；`tasklist | findstr "chrome"` → 查找Chrome进程 |
| `taskkill /F /PID 进程PID` | 强制终止进程（需管理员，慎用）            | 先通过`tasklist`找到PID（如1234），执行 `taskkill /F /PID 1234` → 强制关闭该进程 |
| `taskkill /F /IM 进程名.exe` | 按进程名强制终止（更方便）                | `taskkill /F /IM chrome.exe` → 关闭所有Chrome窗口 |

### 2. 服务管理（需管理员权限）
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `net start`         | 查看所有正在运行的服务                    | `net start`                                |
| `net start 服务名`  | 启动服务                                  | `net start wuauserv` → 启动Windows Update服务 |
| `net stop 服务名`   | 停止服务                                  | `net stop wuauserv` → 停止Windows Update服务 |
| `sc query 服务名`   | 查看服务详细状态（启动类型、PID等）        | `sc query wuauserv` → 查看Windows Update服务状态 |


## 五、系统维护与故障修复（应急必备）
### 1. 系统文件修复
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `sfc /scannow`      | 扫描并修复损坏的系统文件（核心命令）      | 以管理员身份打开命令提示符，输入 `sfc /scannow` → 自动修复缺失/损坏的系统文件 |
| `DISM /Online /Cleanup-Image /RestoreHealth` | 修复系统映像（sfc修复失败时使用）        | 管理员身份执行，需联网（从微软服务器下载修复文件） |

### 2. 磁盘管理与清理
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `chkdsk C: /f /r`   | 检查并修复磁盘错误（需管理员，C盘为例）    | 管理员身份输入，重启后执行（/f修复错误，/r修复坏道） |
| `defrag C: /O`      | 磁盘碎片整理（优化性能）                  | `defrag C: /O` → 针对C盘进行碎片整理（SSD无需频繁执行） |
| `cleanmgr`          | 打开磁盘清理工具（删除临时文件、回收站等） | `cleanmgr` → 弹出图形界面，选择要清理的盘符 |
| `del /f /s /q %temp%` | 强制删除系统临时文件（快速清理）          | 输入命令后回车，删除C:\Users\用户名\AppData\Local\Temp下的所有文件 |

### 3. 注册表操作（慎用！修改错误可能导致系统崩溃）
| 命令                | 功能说明                                  | 示例                                      |
|---------------------|-------------------------------------------|-------------------------------------------|
| `regedit`           | 打开注册表编辑器                          | `regedit` → 谨慎修改，建议先备份注册表     |
| `reg export HKEY_CURRENT_USER C:\注册表备份.reg` | 备份注册表（示例：备份当前用户配置）      | 管理员身份执行，导出到C盘根目录           |

## 七、常见场景命令组合（直接套用）
### 1. 网络排查流程（网页打不开、连不上网）
```bash
ipconfig /all  # 查看IP/DNS是否配置正确
ping 8.8.8.8   # 测试外网连通性
ping baidu.com # 测试域名解析
tracert baidu.com # 排查延迟/中断点
ipconfig /flushdns # 刷新DNS缓存
```

### 2. 系统修复流程（系统卡顿、报错）
```bash
sfc /scannow  # 修复系统文件
DISM /Online /Cleanup-Image /RestoreHealth  # 修复系统映像
chkdsk C: /f /r  # 检查磁盘错误（重启后执行）
cleanmgr  # 清理系统垃圾
```

### 3. 端口占用排查（启动程序提示端口被占用）
```bash
netstat -ano | findstr ":8080"  # 查找8080端口的PID（假设端口8080被占用）
tasklist | findstr "PID号"  # 根据PID找到对应的进程名
taskkill /F /PID PID号  # 强制关闭该进程
```


## 重要提醒
1. 部分命令（如修改用户、开放端口、修复系统）需 **以管理员身份运行命令提示符**（右键开始菜单 → 命令提示符(管理员)）；
2. 命令大小写不敏感（如 `DIR` 和 `dir` 效果一致）；
3. 涉及删除、修改系统文件/注册表的命令（如 `del /s /q`、`regedit`）需谨慎，建议先备份数据；
4. 若命令提示“不是内部或外部命令”，可能是系统环境变量配置问题，可重启电脑或检查 `C:\Windows\System32` 目录下是否有对应exe文件（如 `ping.exe`）。
