Java的`if...else`语句是最常用的条件控制结构，用于根据**布尔表达式的结果（`true`或`false`）** 决定执行不同的代码块。它可以处理单条件判断、二选一判断、多条件分支判断，以及嵌套的复杂条件场景。


### 一、基本语法结构
`if...else`语句有三种常见形式，分别对应不同的判断场景：


#### 1. 单条件判断：`if` 语句
当只需要在**条件成立时执行某段代码**，不成立时不做任何操作（或执行默认逻辑），使用基本`if`结构。

**语法**：  
```java
if (布尔表达式) {
    // 条件为true时执行的代码块（可有多行）
}
```

**执行流程**：  
- 计算`布尔表达式`的值（结果为`true`或`false`）；  
- 若为`true`，执行`{}`中的代码块；  
- 若为`false`，跳过代码块，执行后续代码。  


**示例**：判断一个数是否为正数  
```java
int num = 5;
if (num > 0) { // 条件：num是否大于0
    System.out.println(num + "是正数"); // 条件为true，执行此句
}
// 输出：5是正数
```

```java
int num = -3;
if (num > 0) { 
    System.out.println(num + "是正数"); // 条件为false，不执行
}
// 无输出（跳过代码块）
```


#### 2. 二选一判断：`if...else` 语句
当需要在**条件成立时执行A代码块，不成立时执行B代码块**（非此即彼），使用`if...else`结构。

**语法**：  
```java
if (布尔表达式) {
    // 条件为true时执行的代码块A
} else {
    // 条件为false时执行的代码块B
}
```

**执行流程**：  
- 计算`布尔表达式`的值；  
- 若为`true`，执行代码块A，跳过代码块B；  
- 若为`false`，执行代码块B，跳过代码块A。  


**示例**：判断一个数是偶数还是奇数  
```java
int num = 6;
if (num % 2 == 0) { // 条件：num除以2的余数是否为0
    System.out.println(num + "是偶数"); // 条件true，执行A
} else {
    System.out.println(num + "是奇数"); // 条件false，执行B
}
// 输出：6是偶数
```

```java
int num = 7;
if (num % 2 == 0) {
    System.out.println(num + "是偶数");
} else {
    System.out.println(num + "是奇数"); // 条件false，执行B
}
// 输出：7是奇数
```


#### 3. 多条件分支：`if...else if...else` 语句
当需要判断**多个互斥条件**（只有一个条件可能成立），并执行对应代码块，使用`if...else if...else`结构。

**语法**：  
```java
if (条件1) {
    // 条件1为true时执行
} else if (条件2) {
    // 条件1为false，且条件2为true时执行
} else if (条件3) {
    // 条件1、2为false，且条件3为true时执行
} ... // 可多个else if
else {
    // 所有条件都为false时执行（可选）
}
```

**执行流程**：  
- 从上到下依次判断条件；  
- 一旦某个条件为`true`，执行对应代码块，**跳过后续所有条件判断**；  
- 若所有条件都为`false`，执行`else`块（若有）。  


**示例**：根据成绩评级（多条件分支）  
```java
int score = 85;
if (score >= 90) { // 条件1：90分及以上
    System.out.println("优秀");
} else if (score >= 60) { // 条件2：60-89分（隐含score<90）
    System.out.println("及格");
} else { // 所有条件不满足（score<60）
    System.out.println("不及格");
}
// 输出：及格（85满足条件2）
```

**关键**：条件判断有顺序性，需将**范围更小的条件写在前面**。例如，若先判断`score>=60`，则90分以上的情况会被提前拦截，无法进入“优秀”分支。


#### 4. 复杂条件：嵌套 `if` 语句
当条件判断需要**多层逻辑**（先满足外层条件，再判断内层条件），可在`if`、`else`块中嵌套另一个`if`语句。

**语法**：  
```java
if (外层条件) {
    // 外层条件为true
    if (内层条件) {
        // 内层条件也为true时执行
    } else {
        // 外层true，内层false时执行
    }
} else {
    // 外层条件为false时执行
}
```


**示例**：判断一个数是否为正偶数（先判断正数，再判断偶数）  
```java
int num = 4;
if (num > 0) { // 外层条件：是否为正数
    if (num % 2 == 0) { // 内层条件：是否为偶数
        System.out.println(num + "是正偶数");
    } else {
        System.out.println(num + "是正奇数");
    }
} else {
    System.out.println(num + "不是正数");
}
// 输出：4是正偶数
```


### 二、注意事项
1. **布尔表达式必须是`boolean`类型**：  
   Java是强类型语言，`if`后的条件必须直接返回`boolean`（`true`/`false`），不能像C语言那样用整数（0为假，非0为真）。  
   错误示例：`if (num) { ... }`（`num`是`int`类型，编译报错）。  


2. **代码块的括号`{}`**：  
   - 若代码块只有**一行语句**，括号可以省略（不推荐，易出错）；  
   - 多行语句必须用`{}`包裹，否则只有第一行受`if`控制。  

   ```java
   // 不推荐：省略括号（若后续添加语句，可能逻辑错误）
   if (num > 0) 
       System.out.println("正数"); 
   
   // 推荐：始终加括号，清晰且安全
   if (num > 0) {
       System.out.println("正数");
       System.out.println("处理完毕"); // 明确属于if块
   }
   ```


3. **避免混淆`=`和`==`**：  
   - `==`是**比较运算符**（判断相等），用于条件表达式；  
   - `=`是**赋值运算符**（赋值），若误写为`if (num = 5) { ... }`，编译报错（因为赋值结果是`int`，不是`boolean`）。  


4. **多条件的逻辑组合**：  
   可通过逻辑运算符（`&&`、`||`、`!`）组合多个条件，简化嵌套结构。  

   例如，判断`num`是正偶数：  
   ```java
   // 用逻辑与&&组合条件，替代嵌套if
   if (num > 0 && num % 2 == 0) {
       System.out.println(num + "是正偶数");
   }
   ```


### 三、适用场景总结
| 语句结构               | 适用场景                                  | 示例                          |
|------------------------|-------------------------------------------|-------------------------------|
| `if`                   | 单条件判断（只关心条件成立的情况）        | 判断是否成年（年龄>=18）      |
| `if...else`            | 二选一逻辑（条件成立/不成立对应不同操作）  | 判断奇偶数、真假值            |
| `if...else if...else`  | 多条件分支（多个互斥条件，选其一执行）    | 成绩评级、菜单选择（1/2/3选项）|
| 嵌套`if`               | 复杂条件（需先满足外层条件，再判断内层）  | 先判断是否为学生，再判断是否及格 |


### 总结
`if...else`语句是Java中控制程序分支的核心结构，通过布尔表达式的结果决定执行路径：  
- 单条件用`if`，二选一用`if...else`，多分支用`if...else if...else`，复杂逻辑用嵌套`if`；  
- 注意条件的顺序性、括号的使用，以及`==`和`=`的区别，可避免大部分逻辑错误。  

熟练使用`if...else`能处理各种场景的条件判断，是编写灵活程序的基础。
