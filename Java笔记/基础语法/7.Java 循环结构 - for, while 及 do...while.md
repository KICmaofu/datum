Java中的循环结构用于重复执行一段代码，直到满足终止条件。常用的循环结构有三种：`for`、`while` 和 `do...while`，它们的核心功能相同，但语法和适用场景略有不同。


### 一、for 循环：适合已知循环次数的场景
`for`循环的语法清晰，将“初始化、循环条件、迭代操作”集中在一行，适合**提前知道循环次数**的场景（如遍历数组、固定次数的重复操作）。


#### 1. 语法格式
```java
for (初始化语句; 循环条件; 迭代语句) {
    // 循环体（要重复执行的代码）
}
```

- **初始化语句**：循环开始前执行一次，通常用于声明和初始化循环变量（如`int i = 0`）；  
- **循环条件**：每次执行循环体前判断，结果为`boolean`类型（`true`则执行循环体，`false`则退出循环）；  
- **迭代语句**：每次循环体执行完后执行，通常用于更新循环变量（如`i++`）。  


#### 2. 执行流程
```
初始化语句 → 判断循环条件 → 条件为true → 执行循环体 → 执行迭代语句 → 再次判断条件...（重复）
                                  ↓
                            条件为false → 退出循环
```


#### 3. 示例
##### （1）基本用法：循环输出1~5
```java
// 输出1~5（循环5次）
for (int i = 1; i <= 5; i++) {
    System.out.print(i + " ");
}
// 输出：1 2 3 4 5 
```
- 初始化：`i=1`（只执行一次）；  
- 条件：`i<=5`（第1次判断`1<=5→true`，执行循环体；直到`i=6`时`6<=5→false`，退出）；  
- 迭代：`i++`（每次循环体执行后`i`加1）。  


##### （2）遍历数组（经典场景）
```java
int[] nums = {10, 20, 30, 40};
// 遍历数组（数组长度已知，nums.length=4）
for (int i = 0; i < nums.length; i++) {
    System.out.println(nums[i]); // 依次输出10、20、30、40
}
```


##### （3）增强for循环（foreach）：简化遍历
JDK5+引入增强for循环，专门用于遍历数组或集合（无需索引，直接获取元素），语法：  
```java
for (元素类型 变量名 : 数组/集合) {
    // 循环体（变量名代表当前元素）
}
```

示例：遍历数组  
```java
int[] nums = {10, 20, 30};
for (int num : nums) { // 每次循环，num依次取nums中的元素
    System.out.println(num); // 输出10、20、30
}
```


### 二、while 循环：适合循环次数不确定的场景
`while`循环的语法更简洁，仅包含循环条件，适合**循环次数不确定**（依赖动态条件）的场景（如等待用户输入正确值、处理未知长度的数据）。


#### 1. 语法格式
```java
// 先声明并初始化循环变量（可选，根据条件是否需要）
循环变量初始化;
while (循环条件) {
    // 循环体
    迭代语句; // 通常需要手动更新循环变量，避免死循环
}
```

- **循环条件**：每次执行循环体前判断（`true`则执行，`false`则退出）；  
- 注意：若循环条件初始为`false`，循环体**一次都不会执行**。  


#### 2. 执行流程
```
初始化循环变量 → 判断循环条件 → 条件为true → 执行循环体 → 执行迭代语句 → 再次判断条件...（重复）
                                  ↓
                            条件为false → 退出循环
```


#### 3. 示例
##### （1）基本用法：计算1~100的和
```java
int sum = 0;
int i = 1; // 初始化循环变量
while (i <= 100) { // 条件：i不超过100
    sum += i; // 累加
    i++; // 迭代（i+1，必须有，否则死循环）
}
System.out.println("1~100的和：" + sum); // 输出5050
```


##### （2）处理动态条件：等待用户输入“yes”
```java
import java.util.Scanner;

public class WhileDemo {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        String input = "";
        
        // 循环直到用户输入"yes"
        while (!input.equals("yes")) { 
            System.out.print("请输入'yes'继续：");
            input = scanner.next();
        }
        System.out.println("输入正确！");
    }
}
```
- 循环次数不确定（取决于用户输入），符合`while`的适用场景。  


### 三、do...while 循环：至少执行一次的场景
`do...while`循环与`while`循环类似，但**先执行一次循环体，再判断条件**，因此无论条件是否成立，循环体**至少执行一次**。适合需要“先执行，后判断”的场景（如菜单选择、输入验证）。


#### 1. 语法格式
```java
// 初始化循环变量（可选）
循环变量初始化;
do {
    // 循环体（至少执行一次）
    迭代语句;
} while (循环条件); // 注意：末尾有分号
```


#### 2. 执行流程
```
初始化循环变量 → 执行循环体 → 执行迭代语句 → 判断循环条件 → 条件为true → 再次执行循环体...（重复）
                                 ↓
                           条件为false → 退出循环
```


#### 3. 示例
##### （1）基本用法：至少输出一次
```java
int i = 10;
do {
    System.out.println(i); // 即使条件不成立，也会执行一次
    i++;
} while (i <= 5); // 条件：10<=5→false，循环结束
// 输出：10
```


##### （2）菜单选择（经典场景）
```java
import java.util.Scanner;

public class DoWhileDemo {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        int choice;
        
        do {
            // 先显示菜单（至少显示一次）
            System.out.println("1. 查看信息");
            System.out.println("2. 退出");
            System.out.print("请选择：");
            choice = scanner.nextInt();
            
            if (choice == 1) {
                System.out.println("显示信息...");
            }
        } while (choice != 2); // 选择2则退出
        
        System.out.println("程序结束");
    }
}
```
- 无论用户第一次选择什么，菜单都会先显示一次，符合“先执行，后判断”的需求。  


### 四、三种循环的对比与选择
| 循环类型       | 执行特点                          | 适用场景                                  | 注意事项                          |
|----------------|-----------------------------------|-------------------------------------------|-----------------------------------|
| `for`          | 初始化、条件、迭代集中管理        | 已知循环次数（如遍历数组、固定次数循环）    | 循环变量作用域仅限循环内          |
| `while`        | 先判断条件，不满足则不执行        | 循环次数不确定（依赖动态条件）            | 需在外部初始化循环变量，避免死循环 |
| `do...while`   | 先执行一次，再判断条件            | 至少需要执行一次的场景（如菜单、输入验证）  | 循环体后必须有分号                |  


### 五、循环控制：break 与 continue
在循环中，可通过`break`和`continue`控制流程：  

- **`break`**：立即终止当前循环，跳出循环体；  
- **`continue`**：跳过本次循环剩余部分，直接进入下一次循环。  


#### 示例
```java
// break：找到5就终止循环
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break; // 跳出循环
    }
    System.out.print(i + " "); // 输出：1 2 3 4 
}

// continue：跳过偶数，只输出奇数
for (int i = 1; i <= 5; i++) {
    if (i % 2 == 0) {
        continue; // 跳过本次循环剩余部分
    }
    System.out.print(i + " "); // 输出：1 3 5 
}
```


### 总结
- `for`循环适合已知次数的循环，结构清晰；  
- `while`循环适合次数不确定、依赖条件的场景，需先判断再执行；  
- `do...while`循环适合至少执行一次的场景，先执行再判断；  
- 可通过`break`（终止循环）和`continue`（跳过本次）控制循环流程。  

根据具体场景选择合适的循环结构，能让代码更简洁、易读。
