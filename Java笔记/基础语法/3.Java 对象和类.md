
类和对象是Java面向对象编程（OOP）的核心，**类是对象的模板（蓝图），定义对象的属性和行为；对象是类的实例（具体产物），拥有类定义的属性和行为**。理解两者的关系和使用方式，是掌握Java OOP的基础。


### 一、类的定义：创建对象的模板
类通过关键字`class`定义，包含两部分核心内容：  
- **成员变量（属性）**：描述对象的特征（如人的姓名、年龄）；  
- **成员方法（行为）**：描述对象能执行的操作（如人的吃饭、睡觉）；  
- 可选：**构造方法**：用于初始化对象（创建对象时自动调用）。

#### 类的定义语法
```java
[修饰符] class 类名 {
    // 1. 成员变量（属性）
    数据类型 变量名;
    
    // 2. 构造方法（与类名相同，无返回值）
    [修饰符] 类名(参数列表) {
        初始化逻辑;
    }
    
    // 3. 成员方法（行为）
    [修饰符] 返回值类型 方法名(参数列表) {
        方法体;
        return 返回值; // 无返回值（void）可省略
    }
}
```

#### 示例：定义一个`Person`类
```java
// 人类（类：模板）
public class Person {
    // 1. 成员变量（属性）：描述人的特征
    String name; // 姓名（引用类型，默认null）
    int age;     // 年龄（基本类型，默认0）
    
    // 2. 构造方法：初始化对象（创建对象时调用）
    // 无参构造（默认存在，若定义了带参构造，需手动写）
    public Person() {
        name = "未知";
        age = 0;
    }
    
    // 带参构造（自定义初始化值）
    public Person(String n, int a) {
        name = n; // 给成员变量赋值
        age = a;
    }
    
    // 3. 成员方法（行为）：描述人的操作
    // 无返回值方法
    public void eat() {
        System.out.println(name + "在吃饭");
    }
    
    // 有返回值方法
    public int getAge() {
        return age; // 返回年龄
    }
    
    // 带参数方法
    public void setAge(int newAge) {
        if (newAge > 0) { // 简单逻辑判断
            age = newAge;
        } else {
            System.out.println("年龄无效");
        }
    }
}
```


### 二、对象的创建与使用：类的实例化
对象是类的具体实例，创建对象需通过`new`关键字（调用构造方法），使用对象需通过“对象.成员”的语法访问属性和方法。

#### 1. 创建对象（实例化）
语法：`类名 对象名 = new 类名(参数);`  
- 若调用无参构造，括号内无参数；  
- 若调用带参构造，括号内传对应参数。

#### 2. 使用对象
- 访问成员变量：`对象名.变量名`；  
- 调用成员方法：`对象名.方法名(参数)`。

#### 示例：创建并使用`Person`对象
```java
public class TestPerson {
    public static void main(String[] args) {
        // 1. 调用无参构造，创建对象p1
        Person p1 = new Person();
        System.out.println("p1姓名：" + p1.name); // 访问属性：输出“未知”
        System.out.println("p1年龄：" + p1.getAge()); // 调用方法：输出0
        
        p1.name = "张三"; // 修改属性值
        p1.setAge(20); // 调用带参方法修改年龄
        p1.eat(); // 调用无参方法：输出“张三在吃饭”
        
        // 2. 调用带参构造，创建对象p2（直接初始化属性）
        Person p2 = new Person("李四", 25);
        System.out.println("p2姓名：" + p2.name); // 输出“李四”
        System.out.println("p2年龄：" + p2.getAge()); // 输出25
        p2.eat(); // 输出“李四在吃饭”
    }
}
```

#### 运行结果
```
p1姓名：未知
p1年龄：0
张三在吃饭
p2姓名：李四
p2年龄：25
李四在吃饭
```


### 三、类与对象的核心关系
可以用“蓝图与房子”理解：  
- **类（蓝图）**：只定义房子的结构（有卧室、客厅，能住人），不占用空间；  
- **对象（房子）**：根据蓝图建造的具体房子，有实际的空间，能执行“住人”的行为；  
- 一个类可以创建无数个对象，每个对象的属性值独立（如p1和p2的姓名、年龄不同），但行为一致（都能吃饭）。


### 四、核心补充：构造方法的细节
构造方法是创建对象的关键，有3个核心特点：  
1. 方法名与类名完全相同，无返回值（连`void`都不能写）；  
2. 若未手动定义构造方法，Java会自动生成一个**默认无参构造**；  
3. 若手动定义了构造方法（无论带参与否），默认无参构造会失效（需手动补写）。

#### 错误示例（构造方法问题）
```java
public class Student {
    String name;
    // 手动定义了带参构造，默认无参构造失效
    public Student(String n) {
        name = n;
    }
}

public class Test {
    public static void main(String[] args) {
        // 错误：无法调用无参构造（已失效）
        Student s = new Student(); 
    }
}
```

#### 解决：补写无参构造
```java
public class Student {
    String name;
    // 补写无参构造
    public Student() {
        name = "默认姓名";
    }
    // 带参构造
    public Student(String n) {
        name = n;
    }
}
```


### 五、核心补充：`this`关键字
`this`代表**当前对象**，用于区分成员变量和局部变量（如构造方法、成员方法中的参数与成员变量同名时）。

#### 示例：`this`的用法
```java
public class Person {
    String name;
    int age;
    
    // 带参构造（参数名与成员变量名相同）
    public Person(String name, int age) {
        this.name = name; // this.name：当前对象的成员变量name
        this.age = age;   // this.age：当前对象的成员变量age
    }
    
    public void printInfo() {
        // this可省略（默认指代当前对象）
        System.out.println("姓名：" + this.name + "，年龄：" + this.age);
    }
}
```


### 六、封装：类与对象的核心特性
封装是OOP的三大特性之一，核心是“隐藏对象的属性和实现细节，仅通过公共方法暴露接口”，目的是保证数据安全。

#### 实现封装的步骤
1. 成员变量用`private`修饰（私有，仅类内部可访问）；  
2. 提供公共的`getter`方法（获取属性值）和`setter`方法（修改属性值）；  
3. 在`setter`方法中添加逻辑校验（如年龄不能为负数）。

#### 示例：封装的`Person`类
```java
public class Person {
    // 1. 成员变量私有化（private）
    private String name;
    private int age;
    
    // 2. 构造方法（初始化）
    public Person(String name, int age) {
        this.name = name;
        this.setAge(age); // 调用setter进行校验
    }
    
    // 3. getter方法（获取属性）
    public String getName() {
        return name;
    }
    
    public int getAge() {
        return age;
    }
    
    // 4. setter方法（修改属性，加校验）
    public void setName(String name) {
        if (name != null && !name.isEmpty()) { // 姓名不能为null或空
            this.name = name;
        } else {
            System.out.println("姓名无效");
        }
    }
    
    public void setAge(int age) {
        if (age > 0 && age <= 120) { // 年龄范围校验
            this.age = age;
        } else {
            System.out.println("年龄无效，设为默认值18");
            this.age = 18;
        }
    }
}
```

#### 使用封装类
```java
public class Test {
    public static void main(String[] args) {
        Person p = new Person("张三", 200); // 年龄200无效
        System.out.println(p.getAge()); // 输出18（默认值）
        
        p.setAge(25); // 合法修改
        System.out.println(p.getAge()); // 输出25
        
        // p.age = 30; // 错误：age是private，无法直接访问
    }
}
```


### 总结
- **类**：是对象的模板，定义属性（成员变量）和行为（成员方法），用`class`声明；  
- **对象**：是类的实例，用`new`+构造方法创建，通过“对象.成员”访问属性和方法；  
- **核心关系**：一个类可创建多个对象，对象共享类的行为，属性值独立；  
- **关键特性**：封装（`private`+`getter/setter`）保证数据安全，`this`指代当前对象。
