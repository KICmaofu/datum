Java运算符是用于对数据（变量、常量）进行运算的符号，根据功能可分为**算术运算符**、**赋值运算符**、**比较运算符**、**逻辑运算符**、**位运算符**和**三元运算符**等。掌握运算符的用法是编写Java表达式和逻辑的基础。


### 一、算术运算符
用于执行基本的数学运算（加、减、乘、除等），适用于数值类型（整数、浮点）。

| 运算符 | 名称       | 示例（a=5, b=2） | 说明                                  |
|--------|------------|------------------|---------------------------------------|
| `+`    | 加法       | `a + b = 7`      | 数值相加，或字符串拼接（`"a"+"b"="ab"`） |
| `-`    | 减法       | `a - b = 3`      | 数值相减                              |
| `*`    | 乘法       | `a * b = 10`     | 数值相乘                              |
| `/`    | 除法       | `a / b = 2`      | 整数除法取商（5/2=2）；浮点除法取精确值（5.0/2=2.5） |
| `%`    | 取余（模） | `a % b = 1`      | 取除法的余数（结果符号与被除数一致：`-5%2=-1`，`5%-2=1`） |
| `++`   | 自增       | `++a=6`；`b++=2` | 前缀：先加1再使用；后缀：先使用再加1    |
| `--`   | 自减       | `--a=4`；`b--=2` | 前缀：先减1再使用；后缀：先使用再减1    |  


**关键示例**：  
1. 整数除法 vs 浮点除法：  
   ```java
   int x = 5 / 2;      // 结果2（整数除法，舍弃小数）
   double y = 5.0 / 2; // 结果2.5（浮点除法，保留小数）
   ```  

2. 自增自减的前缀/后缀区别：  
   ```java
   int a = 5;
   int b = ++a; // 前缀：a先+1（a=6），再赋值给b → b=6
   
   int c = 5;
   int d = c++; // 后缀：先赋值给d（d=5），c再+1 → c=6
   ```  

3. 字符串拼接（`+`的特殊用法）：  
   ```java
   System.out.println("年龄：" + 18); // 输出"年龄：18"（数字转字符串拼接）
   System.out.println(10 + 20 + "岁"); // 输出"30岁"（先算10+20，再拼接）
   System.out.println("岁：" + 10 + 20); // 输出"岁：1020"（先拼接"岁：10"，再拼接20）
   ```  


### 二、赋值运算符
用于将右侧的值赋给左侧的变量，包括基本赋值和复合赋值。

| 运算符 | 名称       | 示例（a=5）       | 等价于               |
|--------|------------|-------------------|----------------------|
| `=`    | 基本赋值   | `a = 10`          | `a = 10`             |
| `+=`   | 加后赋值   | `a += 3`          | `a = a + 3`（结果8） |
| `-=`   | 减后赋值   | `a -= 2`          | `a = a - 2`（结果3） |
| `*=`   | 乘后赋值   | `a *= 4`          | `a = a * 4`（结果20）|
| `/=`   | 除后赋值   | `a /= 2`          | `a = a / 2`（结果2） |
| `%=`   | 取余后赋值 | `a %= 3`          | `a = a % 3`（结果2） |  


**注意**：复合赋值会自动进行类型转换（针对基本类型）：  
```java
byte b = 1;
b += 2; // 正确：等价于 b = (byte)(b + 2)（自动强转）
// b = b + 2; // 错误：b+2结果为int类型，需手动强转 (byte)(b+2)
```  


### 三、比较运算符
用于判断两个数据的关系，返回`boolean`类型（`true`或`false`）。

| 运算符 | 名称       | 示例（a=5, b=2） | 结果  |
|--------|------------|------------------|-------|
| `==`   | 等于       | `a == b`         | false |
| `!=`   | 不等于     | `a != b`         | true  |
| `>`    | 大于       | `a > b`          | true  |
| `<`    | 小于       | `a < b`          | false |
| `>=`   | 大于等于   | `a >= 5`         | true  |
| `<=`   | 小于等于   | `b <= 1`         | false |  


**注意**：  
- 基本类型比较的是“值”（如`int a=5; int b=5; a==b → true`）；  
- 引用类型（如`String`）的`==`比较的是“内存地址”（是否为同一个对象），比较内容需用`equals()`方法：  
  ```java
  String s1 = new String("abc");
  String s2 = new String("abc");
  System.out.println(s1 == s2); // false（不同对象，地址不同）
  System.out.println(s1.equals(s2)); // true（内容相同）
  ```  


### 四、逻辑运算符
用于连接布尔表达式（返回`boolean`的表达式），结果仍为`boolean`类型。

| 运算符 | 名称       | 说明                                  | 示例（a=true, b=false） |
|--------|------------|---------------------------------------|-------------------------|
| `&&`   | 短路与     | 两边都为`true`则结果为`true`；左侧为`false`时，右侧不执行（短路） | `a && b → false`        |
| `||`   | 短路或     | 一边为`true`则结果为`true`；左侧为`true`时，右侧不执行（短路） | `a || b → true`         |
| `!`    | 逻辑非     | 取反（`true`变`false`，`false`变`true`） | `!a → false`            |
| `&`    | 逻辑与（非短路） | 两边都为`true`则结果为`true`；无论左侧结果如何，右侧都会执行 | `a & b → false`         |
| `|`    | 逻辑或（非短路） | 一边为`true`则结果为`true`；无论左侧结果如何，右侧都会执行 | `a | b → true`          |  


**核心：短路特性的作用**（避免无效计算或错误）：  
```java
int x = 0;
// 短路与：左侧x==0为true，才执行右侧（否则右侧/0会报错）
if (x != 0 && (10 / x) > 1) { ... } 

int y = 5;
// 短路或：左侧y>0为true，右侧不执行（无需判断）
if (y > 0 || (10 / 0) > 1) { ... } 
```  


### 五、位运算符
对数据的二进制位直接进行操作，适用于整数类型（`byte`、`short`、`int`、`long`）。

| 运算符 | 名称         | 说明（n为位数）                                  | 示例（a=6 → 0110，b=3 → 0011） |
|--------|--------------|-------------------------------------------------|--------------------------------|
| `&`    | 按位与       | 对应位都为1则为1，否则为0                        | `a & b → 0010 → 2`             |
| `|`    | 按位或       | 对应位有一个为1则为1，否则为0                    | `a | b → 0111 → 7`             |
| `^`    | 按位异或     | 对应位不同则为1，相同则为0（同0异1）              | `a ^ b → 0101 → 5`             |
| `~`    | 按位非       | 每一位取反（0变1，1变0）                         | `~a → 1001（补码）→ -7`        |
| `<<`   | 左移         | 整体左移n位，右侧补0（相当于乘以2ⁿ）              | `a << 1 → 1100 → 12`（6×2¹）   |
| `>>`   | 右移（带符号）| 整体右移n位，左侧补符号位（正数补0，负数补1）      | `a >> 1 → 0011 → 3`（6÷2¹）    |
| `>>>`  | 右移（无符号）| 整体右移n位，左侧补0（无论正负）                  | `a >>> 1 → 0011 → 3`           |  


**常用场景**：  
- 左移`<<`：快速计算乘以2的n次方（效率高于`*`）；  
- 右移`>>`：快速计算除以2的n次方（仅正数）；  
- 异或`^`：不借助临时变量交换两个整数（`a = a ^ b; b = a ^ b; a = a ^ b;`）。  


### 六、三元运算符（条件运算符）
Java中唯一的三目运算符，用于简化简单的`if-else`逻辑。  

**格式**：`条件表达式 ? 表达式1 : 表达式2`  
- 若条件为`true`，结果为表达式1的值；  
- 若条件为`false`，结果为表达式2的值。  


**示例**：  
```java
int score = 85;
String result = (score >= 60) ? "及格" : "不及格"; 
System.out.println(result); // 输出"及格"

int a = 10, b = 20;
int max = (a > b) ? a : b; // max=20（取较大值）
```  


### 七、运算符优先级（简化版）
当一个表达式包含多个运算符时，优先级决定执行顺序（优先级高的先执行）。无需死记，可通过**括号`()`** 强制改变优先级（推荐，提高可读性）。  

从高到低的核心优先级：  
1. 括号`()`  
2. 自增`++`、自减`--`、逻辑非`!`、按位非`~`  
3. 乘法`*`、除法`/`、取余`%`  
4. 加法`+`、减法`-`  
5. 位运算符（`<<`、`>>`、`>>>`）  
6. 比较运算符（`>`、`<`、`>=`、`<=`）  
7. 相等判断（`==`、`!=`）  
8. 位运算符（`&`、`^`、`|`）  
9. 逻辑运算符（`&&`、`||`）  
10. 三元运算符（`? :`）  
11. 赋值运算符（`=`、`+=`、`-=`等）  


**示例**：  
```java
int x = 10;
int y = (x++ > 10) ? (x + 5) : (x - 5); 
// 解析：x++是后缀，先判断10>10→false，再x=11；结果为11-5=6 → y=6
```  


### 总结
Java运算符按功能分为六大类，核心注意点：  
- 算术运算符中，`/`的整数除法和浮点除法区别，`++`/`--`的前缀/后缀区别；  
- 逻辑运算符中，`&&`/`||`的短路特性（提高效率，避免错误）；  
- 比较引用类型时，`==`比较地址，`equals()`比较内容；  
- 优先级复杂时，用括号明确执行顺序，增强代码可读性。  

熟练运用运算符是编写正确表达式和逻辑的基础，需结合实例多练习。
