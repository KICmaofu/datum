在Java中，**数组（Array）** 是一种用于存储**相同类型数据**的容器，它是一种**引用数据类型**，可以存储基本数据类型（如`int`、`char`）或对象（如`String`、自定义类）。数组的长度在创建后不可改变，适合存储固定数量的同类型元素。


### 一、数组的基本特性  
1. **同类型元素**：数组中所有元素必须是相同数据类型（如`int[]`只能存`int`，`String[]`只能存`String`）。  
2. **固定长度**：数组一旦创建，长度（元素个数）不可修改。  
3. **连续内存**：元素在内存中连续存储，通过索引（下标）快速访问。  
4. **引用类型**：数组变量存储的是数组对象在堆内存中的地址（类似对象引用）。  


### 二、数组的声明与初始化  
数组的使用分为**声明**和**初始化**两步，初始化时必须指定长度或元素（从而确定长度）。  


#### 1. 数组声明  
声明数组时，只需指定元素类型和数组名称，不指定长度。有两种语法（推荐第一种，更直观）：  
```java
// 语法1：类型[] 数组名;（推荐）
int[] nums;       // 声明int类型数组
String[] names;   // 声明String类型数组

// 语法2：类型 数组名[];（兼容C/C++风格，不推荐）
int ages[];       // 效果同上，但可读性差
```  


#### 2. 数组初始化  
初始化分为**静态初始化**（直接指定元素）和**动态初始化**（指定长度，元素用默认值）。  


##### （1）静态初始化  
直接列出所有元素，JVM会自动计算长度。语法：  
```java
// 完整写法：声明 + 初始化
int[] nums = new int[]{1, 2, 3, 4};  
String[] names = new String[]{"Alice", "Bob", "Charlie"};

// 简化写法（仅在声明时可用）
int[] nums2 = {1, 2, 3};  // 等价于 new int[]{1,2,3}
String[] names2 = {"A", "B"};
```  


##### （2）动态初始化  
只指定数组长度，元素由JVM赋予**默认值**（根据元素类型）。语法：  
```java
// 声明并初始化：指定长度，元素用默认值
int[] arr = new int[5];  // 长度为5的int数组，元素默认值为0
double[] dArr = new double[3]; // 默认值为0.0
boolean[] bArr = new boolean[2]; // 默认值为false
String[] sArr = new String[4]; // 引用类型默认值为null
```  

**不同类型的默认值**：  
- 整数类型（`byte`、`short`、`int`、`long`）：`0`  
- 浮点类型（`float`、`double`）：`0.0`  
- 字符类型（`char`）：`'\u0000'`（空字符）  
- 布尔类型（`boolean`）：`false`  
- 引用类型（对象、数组）：`null`  


### 三、数组的访问与遍历  
数组通过**索引（index）** 访问元素，索引从`0`开始，最大索引为`长度-1`（若超出范围，会抛出`ArrayIndexOutOfBoundsException`异常）。  


#### 1. 访问元素  
通过`数组名[索引]`读写元素：  
```java
int[] nums = {10, 20, 30};
System.out.println(nums[0]); // 读：10（访问第1个元素）
nums[1] = 200; // 写：修改第2个元素为200
System.out.println(nums[1]); // 200
```  


#### 2. 获取数组长度  
通过`数组名.length`属性获取长度（注意是**属性**，不是方法，无括号）：  
```java
int[] nums = {1, 2, 3};
System.out.println(nums.length); // 3（长度为3）
```  


#### 3. 遍历数组  
遍历是指依次访问数组中的每个元素，常用两种方式：  


##### （1）普通for循环（可修改元素）  
通过索引遍历，适合需要修改元素或获取索引的场景：  
```java
int[] nums = {1, 2, 3, 4};
for (int i = 0; i < nums.length; i++) {
    System.out.println(nums[i]); // 依次输出1、2、3、4
    nums[i] *= 2; // 修改元素（变为2、4、6、8）
}
```  


##### （2）增强for循环（foreach，仅读取）  
简化遍历语法，直接获取元素值，但无法修改元素（只能读），也无法获取索引：  
```java
int[] nums = {1, 2, 3};
for (int num : nums) { // 语法：元素类型 变量名 : 数组名
    System.out.println(num); // 依次输出1、2、3
    // num = 100; // 无效：仅修改临时变量，不影响原数组
}
```  


### 四、多维数组  
多维数组本质是“数组的数组”，最常用的是**二维数组**（可理解为“表格”，行和列）。  


#### 1. 二维数组的声明与初始化  
##### （1）静态初始化  
直接指定每行的元素（每行长度可不同，称为“不规则数组”）：  
```java
// 完整写法
int[][] matrix = new int[][]{{1, 2}, {3, 4, 5}, {6}};

// 简化写法
int[][] matrix2 = {{1, 2}, {3, 4}, {5}};
```  


##### （2）动态初始化  
- 方式1：指定行数和列数（规则数组，每行长度相同）：  
  ```java
  int[][] arr = new int[3][2]; // 3行2列，元素默认值为0
  ```  

- 方式2：先指定行数，再动态初始化每行（不规则数组）：  
  ```java
  int[][] arr = new int[3][]; // 3行，每行长度暂未指定
  arr[0] = new int[2]; // 第1行长度2
  arr[1] = new int[3]; // 第2行长度3
  arr[2] = new int[1]; // 第3行长度1
  ```  


#### 2. 二维数组的遍历  
通常用嵌套for循环：  
```java
int[][] matrix = {{1, 2}, {3, 4, 5}, {6}};

// 普通for循环（获取索引）
for (int i = 0; i < matrix.length; i++) { // 遍历行
    for (int j = 0; j < matrix[i].length; j++) { // 遍历每行的列
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println(); // 换行
}
// 输出：
// 1 2 
// 3 4 5 
// 6 

// 增强for循环（仅读取）
for (int[] row : matrix) { // 每行是一个int[]
    for (int num : row) { // 每行的元素
        System.out.print(num + " ");
    }
    System.out.println();
}
```  


### 五、数组的常用操作（工具类`Arrays`）  
`java.util.Arrays`类提供了大量静态方法，简化数组操作（需导入`java.util.Arrays`）。  


#### 1. 数组排序  
`Arrays.sort(数组)`：对数组升序排序（基本类型按值，对象按`Comparable`接口）。  
```java
int[] nums = {3, 1, 4, 2};
Arrays.sort(nums); // 排序后：[1, 2, 3, 4]

String[] names = {"Bob", "Alice", "Charlie"};
Arrays.sort(names); // 按字典序排序：["Alice", "Bob", "Charlie"]
```  


#### 2. 数组复制  
- `Arrays.copyOf(原数组, 新长度)`：复制原数组，返回新数组（长度不足则补默认值，超出则截断）。  
- `System.arraycopy(原数组, 原起始索引, 目标数组, 目标起始索引, 复制长度)`：高效复制指定范围元素（无返回值，直接修改目标数组）。  

```java
int[] oldArr = {1, 2, 3};

// Arrays.copyOf
int[] newArr1 = Arrays.copyOf(oldArr, 5); // 新长度5，补默认值0 → [1,2,3,0,0]
int[] newArr2 = Arrays.copyOf(oldArr, 2); // 新长度2，截断 → [1,2]

// System.arraycopy
int[] dest = new int[5];
System.arraycopy(oldArr, 0, dest, 1, 3); // 从oldArr[0]复制3个元素到dest[1]
// dest结果：[0,1,2,3,0]
```  


#### 3. 数组查找  
`Arrays.binarySearch(数组, 目标值)`：在**已排序**的数组中查找目标值，返回索引（未找到返回负数）。  
```java
int[] nums = {1, 2, 3, 4, 5}; // 必须先排序
int index = Arrays.binarySearch(nums, 3); // 找到，返回2
int index2 = Arrays.binarySearch(nums, 6); // 未找到，返回负数
```  


#### 4. 数组转字符串  
`Arrays.toString(数组)`：将数组转为字符串（方便打印）。  
```java
int[] nums = {1, 2, 3};
System.out.println(Arrays.toString(nums)); // 输出：[1, 2, 3]

int[][] matrix = {{1,2}, {3,4}};
System.out.println(Arrays.deepToString(matrix)); // 二维数组用deepToString → [[1, 2], [3, 4]]
```  


### 六、数组的注意事项  
1. **索引越界**：访问索引`<0`或`>=length`时，会抛出`ArrayIndexOutOfBoundsException`。  
2. **空指针异常**：若数组变量为`null`（未初始化），访问`length`或元素会抛出`NullPointerException`。  
   ```java
   int[] arr = null;
   // System.out.println(arr.length); // 抛NullPointerException
   ```  
3. **长度不可变**：数组长度一旦确定，无法修改（若需动态增减元素，需手动复制到新数组，或使用`ArrayList`等集合）。  


### 总结  
数组是Java中存储同类型数据的基础容器，核心特点是**固定长度、连续存储、通过索引访问**。主要用法包括：  
- 声明与初始化（静态/动态）；  
- 访问与遍历（普通for/foreach）；  
- 多维数组（尤其是二维数组的嵌套遍历）；  
- 借助`Arrays`工具类实现排序、复制、查找等操作。  

数组适合存储固定数量的同类型数据，若需动态调整长度，建议使用`ArrayList`等集合类。
