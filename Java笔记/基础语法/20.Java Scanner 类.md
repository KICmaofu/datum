在Java中，`java.util.Scanner`类是一个**便捷的输入处理工具**，用于从各种输入源（如键盘、文件、字符串等）读取数据，并支持自动解析为不同的数据类型（如`int`、`double`、`String`等）。它简化了输入操作，是Java中处理简单输入的常用类。


### 一、Scanner类的核心作用  
- 从输入源（`InputStream`、`File`、`String`等）读取数据。  
- 自动将输入的字符串解析为基本数据类型（`int`、`double`等）或字符串。  
- 支持通过**分隔符**（默认是空白字符，如空格、换行、制表符）拆分输入数据。  


### 二、创建Scanner对象  
使用`Scanner`类前需导入`java.util.Scanner`，然后通过构造方法关联输入源。最常用的输入源是**标准输入流`System.in`**（即键盘输入），也可关联文件、字符串等。  


#### 1. 关联键盘输入（`System.in`）  
```java
import java.util.Scanner;

// 创建Scanner对象，关联键盘输入
Scanner scanner = new Scanner(System.in);
```  


#### 2. 关联文件输入  
```java
import java.util.Scanner;
import java.io.File;
import java.io.FileNotFoundException;

// 创建Scanner对象，关联文件（需处理FileNotFoundException）
try {
    Scanner fileScanner = new Scanner(new File("input.txt"));
} catch (FileNotFoundException e) {
    e.printStackTrace();
}
```  


#### 3. 关联字符串输入  
```java
// 创建Scanner对象，关联字符串
Scanner strScanner = new Scanner("Hello 123 45.6");
```  


### 三、常用方法（读取数据）  
`Scanner`提供了一系列`nextXxx()`方法，用于读取不同类型的数据，核心方法如下：  


#### 1. 读取字符串  
- **`String next()`**：  
  读取**下一个完整的标记**（默认以空白字符为分隔符，不包含分隔符），自动跳过前导空白（如空格、换行）。  
  ```java
  Scanner sc = new Scanner(System.in);
  System.out.print("输入单词：");
  String word = sc.next(); // 输入"Hello World"，返回"Hello"（遇到空格停止）
  System.out.println(word); // 输出：Hello
  ```  

- **`String nextLine()`**：  
  读取**一整行内容**（包含空格），直到遇到换行符`\n`，返回的字符串不包含换行符。  
  ```java
  Scanner sc = new Scanner(System.in);
  System.out.print("输入一行话：");
  String line = sc.nextLine(); // 输入"Hello World"，返回完整的"Hello World"
  System.out.println(line); // 输出：Hello World
  ```  


#### 2. 读取基本数据类型  
`Scanner`支持直接读取`int`、`double`、`boolean`等基本类型，自动完成字符串到对应类型的转换：  

| 方法              | 说明                                  | 示例输入       | 返回值       |
|-------------------|---------------------------------------|----------------|--------------|
| `int nextInt()`   | 读取下一个整数                        | "123"          | `123`        |
| `long nextLong()` | 读取下一个长整数                      | "123456789"    | `123456789L` |
| `double nextDouble()` | 读取下一个双精度浮点数              | "3.14"         | `3.14`       |
| `float nextFloat()` | 读取下一个单精度浮点数               | "2.5f"         | `2.5f`       |
| `boolean nextBoolean()` | 读取下一个布尔值（仅识别"true"/"false"，不区分大小写） | "TRUE" | `true` |  


**示例：读取整数和浮点数**  
```java
Scanner sc = new Scanner(System.in);
System.out.print("输入整数：");
int num = sc.nextInt(); // 输入"100"，返回100

System.out.print("输入小数：");
double d = sc.nextDouble(); // 输入"3.14"，返回3.14

System.out.println("结果：" + (num + d)); // 103.14
```  


#### 3. 其他常用方法  
- **`boolean hasNextXxx()`**：判断输入中是否还有下一个对应类型的数据（避免读取时抛出异常）。  
  ```java
  Scanner sc = new Scanner(System.in);
  if (sc.hasNextInt()) { // 检查是否有整数输入
      int num = sc.nextInt();
  } else {
      System.out.println("输入不是整数");
  }
  ```  

- **`useDelimiter(String pattern)`**：自定义分隔符（默认是空白字符），支持正则表达式。  
  ```java
  Scanner sc = new Scanner("a,b,c,d");
  sc.useDelimiter(","); // 以逗号为分隔符
  while (sc.hasNext()) {
      System.out.println(sc.next()); // 依次输出a、b、c、d
  }
  ```  

- **`close()`**：关闭`Scanner`，释放关联的输入源资源（使用后建议关闭，避免资源泄漏）。  
  ```java
  scanner.close();
  ```  


### 四、注意事项  
1. **`next()`与`nextLine()`的冲突**：  
   若先调用`nextInt()`、`next()`等方法（不读取换行符），再调用`nextLine()`，`nextLine()`会读取到**空字符串**（因为前一个方法未消耗输入中的换行符`\n`）。  

   **解决方法**：在`nextInt()`后额外调用一次`nextLine()`消耗换行符。  
   ```java
   Scanner sc = new Scanner(System.in);
   System.out.print("输入年龄：");
   int age = sc.nextInt(); // 输入"20\n"，读取20后，换行符\n残留
   
   sc.nextLine(); // 消耗残留的\n
   
   System.out.print("输入姓名：");
   String name = sc.nextLine(); // 此时可正常读取姓名
   ```  


2. **输入不匹配异常（`InputMismatchException`）**：  
   若输入的数据类型与`nextXxx()`方法预期的类型不匹配（如用`nextInt()`读取"abc"），会抛出此异常。建议先用`hasNextXxx()`判断后再读取。  


3. **关闭`System.in`的影响**：  
   若`Scanner`关联的是`System.in`（键盘输入），调用`close()`后，`System.in`会被关闭，后续无法再通过其他输入流读取键盘输入（需谨慎）。  


4. **效率问题**：  
   `Scanner`适合处理**简单输入场景**（如控制台交互），但在读取大量数据（如大文件）时效率较低，此时推荐使用`BufferedReader`。  


### 五、综合示例  
```java
import java.util.Scanner;

public class ScannerDemo {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        // 读取姓名（整行）
        System.out.print("请输入姓名：");
        String name = sc.nextLine();

        // 读取年龄（整数）
        System.out.print("请输入年龄：");
        int age = 0;
        if (sc.hasNextInt()) {
            age = sc.nextInt();
        } else {
            System.out.println("年龄输入错误，默认0");
        }

        // 读取身高（小数）
        sc.nextLine(); // 消耗换行符
        System.out.print("请输入身高（米）：");
        double height = sc.nextDouble();

        // 输出结果
        System.out.printf("姓名：%s，年龄：%d，身高：%.2f米%n", name, age, height);

        sc.close(); // 关闭资源
    }
}
```  


### 总结  
`Scanner`类是Java处理输入的便捷工具，核心优势是**支持多种数据类型的自动解析**和**简单的API**。主要用于：  
- 控制台交互（读取键盘输入）。  
- 简单的文件或字符串解析（按分隔符拆分数据）。  

使用时需注意`next()`与`nextLine()`的区别、输入类型匹配问题，以及资源关闭的影响。对于复杂或大量数据的输入场景，可考虑`BufferedReader`等更高效的类。
