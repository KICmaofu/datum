Java是一种面向对象的编程语言，其语法严谨且具有跨平台性（通过JVM实现）。以下是Java基础语法的核心内容，涵盖从程序结构到控制流、数据类型等关键知识点，附带示例帮助理解。


### 一、Java程序的基本结构
Java程序的最小单位是**类（Class）**，所有代码必须放在类中。一个完整的Java程序至少包含一个类，且若包含`main`方法（程序入口），则该类为“主类”。

#### 示例：Hello World程序
```java
// 类名：HelloWorld（主类，文件名必须与类名一致：HelloWorld.java）
public class HelloWorld {
    // 程序入口：main方法（固定格式）
    public static void main(String[] args) {
        // 输出语句：打印字符串到控制台
        System.out.println("Hello, World!");
    }
}
```

**核心说明**：  
- 类名需与文件名一致（若类被`public`修饰），否则编译报错；  
- `main`方法是程序运行的入口，格式固定：`public static void main(String[] args)`；  
- `System.out.println()`用于控制台输出（`println`会自动换行，`print`不换行）。  


### 二、标识符与关键字
#### 1. 标识符
用于命名类、变量、方法等的名称，需遵循规则：  
- 由字母（A-Z/a-z）、数字（0-9）、下划线（_）、美元符号（$）组成；  
- 不能以数字开头；  
- 区分大小写（`name`和`Name`是不同标识符）；  
- 不能使用Java关键字（如`int`、`class`）。  

**示例**：合法标识符：`age`、`userName`、`_score`、`$total`；  
非法标识符：`123abc`（数字开头）、`user-name`（含减号）、`class`（关键字）。  


#### 2. 关键字
Java保留的特殊单词，具有特定含义，不能作为标识符。常见关键字：  
- 数据类型：`int`、`double`、`boolean`、`char`等；  
- 类与方法：`class`、`public`、`static`、`void`等；  
- 控制流：`if`、`else`、`for`、`while`、`break`等。  


### 三、数据类型
Java是**强类型语言**：变量必须先声明类型，且类型不可随意转换。数据类型分为两大类：  


#### 1. 基本数据类型（8种）
直接存储数据值，分为4类8种：  

| 类型         | 占用空间  | 取值范围                          | 示例                 |
|--------------|-----------|-----------------------------------|----------------------|
| **整数型**   |           |                                   |                      |
| `byte`       | 1字节     | -128 ~ 127                        | `byte b = 100;`      |
| `short`      | 2字节     | -32768 ~ 32767                    | `short s = 2000;`    |
| `int`（默认）| 4字节     | -2³¹ ~ 2³¹-1（约±21亿）           | `int i = 100000;`    |
| `long`       | 8字节     | -2⁶³ ~ 2⁶³-1                      | `long l = 1000000L;`（需加L/l） |
| **浮点型**   |           |                                   |                      |
| `float`      | 4字节     | 单精度浮点数（约±3.4×10³⁸）       | `float f = 3.14F;`（需加F/f） |
| `double`（默认）| 8字节   | 双精度浮点数（约±1.8×10³⁰⁸）      | `double d = 3.14159;` |
| **字符型**   |           |                                   |                      |
| `char`       | 2字节     | 单个Unicode字符（0 ~ 65535）      | `char c = 'A';`（单引号） |
| **布尔型**   | 1字节     | `true`（真）或`false`（假）       | `boolean flag = true;` |  


#### 2. 引用数据类型
存储的是对象的“引用”（内存地址），而非数据本身。包括：  
- 类（`class`）：如`String`、自定义类；  
- 接口（`interface`）；  
- 数组（`array`）。  

**示例**：  
```java
String str = "hello";  // String是类（引用类型）
int[] arr = {1, 2, 3};  // 数组是引用类型
```  


### 四、变量与常量
#### 1. 变量
用于存储可变数据，声明格式：`数据类型 变量名 = 初始值;`（初始值可省略，后续赋值）。  

**示例**：  
```java
int age;  // 声明变量（未初始化）
age = 18;  // 赋值
double height = 1.75;  // 声明并初始化
char gender = '男';
```  


#### 2. 常量
值不可修改的量，用`final`修饰，声明时必须初始化。  

**示例**：  
```java
final double PI = 3.14159;  // 常量名通常全大写
// PI = 3.14;  // 错误：常量不可修改
```  


### 五、运算符
用于对数据进行运算，按功能分为以下几类：  


#### 1. 算术运算符
| 运算符 | 描述       | 示例（a=5, b=2） |
|--------|------------|------------------|
| `+`    | 加         | `a + b = 7`      |
| `-`    | 减         | `a - b = 3`      |
| `*`    | 乘         | `a * b = 10`     |
| `/`    | 除（整数除法取商） | `a / b = 2`（5/2=2，非2.5） |
| `%`    | 取余（模运算） | `a % b = 1`      |
| `++`   | 自增（前++：先加后用；后++：先用后加） | `++a=6`，`b++=2`（执行后b=3） |
| `--`   | 自减（类似自增） | `--a=4`，`b--=2`（执行后b=1） |  


#### 2. 赋值运算符
`=`：将右侧值赋给左侧变量，可结合算术运算符简化：  
```java
int x = 10;
x += 5;  // 等价于 x = x + 5 → x=15
x *= 2;  // 等价于 x = x * 2 → x=30
```  


#### 3. 比较运算符
用于判断关系，返回`boolean`类型（`true`/`false`）：  
| 运算符 | 描述       | 示例（a=5, b=2） |
|--------|------------|------------------|
| `==`   | 等于       | `a == b → false` |
| `!=`   | 不等于     | `a != b → true`  |
| `>`    | 大于       | `a > b → true`   |
| `<`    | 小于       | `a < b → false`  |
| `>=`   | 大于等于   | `a >= 5 → true`  |
| `<=`   | 小于等于   | `b <= 1 → false` |  


#### 4. 逻辑运算符
用于连接布尔表达式，返回`boolean`类型：  
| 运算符 | 描述                 | 示例（a=true, b=false） |
|--------|----------------------|-------------------------|
| `&&`   | 短路与（两边都为真则真） | `a && b → false`        |
| `||`   | 短路或（一边为真则真） | `a || b → true`         |
| `!`    | 非（取反）           | `!a → false`            |  

**短路特性**：`&&`若左侧为`false`，右侧不执行；`||`若左侧为`true`，右侧不执行。  


#### 5. 位运算符（操作二进制位）
较少直接使用，用于底层操作：  
- `&`（与）、`|`（或）、`^`（异或）、`~`（非）、`<<`（左移）、`>>`（右移）。  


### 六、控制流语句
控制程序执行顺序，包括条件语句和循环语句。  


#### 1. 条件语句
##### （1）`if-else`
```java
int score = 85;
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 60) {  // 多条件判断
    System.out.println("及格");
} else {
    System.out.println("不及格");
}
// 输出：及格
```

##### （2）`switch`（多分支判断，JDK7+支持String）
```java
char grade = 'B';
switch (grade) {
    case 'A':
        System.out.println("优秀");
        break;  // 跳出switch
    case 'B':
        System.out.println("良好");
        break;
    default:  // 所有case不匹配时执行
        System.out.println("其他");
}
// 输出：良好
```


#### 2. 循环语句
##### （1）`for`循环（适合已知循环次数）
```java
// 遍历1~5
for (int i = 1; i <= 5; i++) {  // 初始化; 循环条件; 迭代
    System.out.print(i + " ");
}
// 输出：1 2 3 4 5 
```

##### （2）`while`循环（适合未知循环次数，先判断后执行）
```java
int i = 1;
while (i <= 5) {  // 条件为true时执行循环体
    System.out.print(i + " ");
    i++;
}
// 输出：1 2 3 4 5 
```

##### （3）`do-while`循环（先执行后判断，至少执行一次）
```java
int i = 1;
do {
    System.out.print(i + " ");
    i++;
} while (i <= 5);  // 条件为true时继续循环
// 输出：1 2 3 4 5 
```


#### 3. 跳转语句
- `break`：跳出当前循环或`switch`；  
- `continue`：跳过本次循环剩余部分，直接进入下一次循环；  
- `return`：结束当前方法。  


### 七、数组
存储相同类型数据的集合，长度固定（一旦创建不可修改）。  


#### 1. 一维数组
##### 声明与初始化：
```java
// 方式1：声明 + 初始化（指定长度）
int[] arr1 = new int[3];  // 长度3，默认值0
arr1[0] = 1;  // 赋值（索引从0开始）
arr1[1] = 2;
arr1[2] = 3;

// 方式2：声明 + 直接赋值（长度由元素数决定）
int[] arr2 = {1, 2, 3, 4};  // 长度4

// 方式3：动态初始化（先声明，后创建）
int[] arr3;
arr3 = new int[]{5, 6, 7};  // 注意：不可写new int[3]{5,6,7}
```

##### 访问与遍历：
```java
int[] arr = {10, 20, 30};
System.out.println(arr[1]);  // 访问索引1的元素 → 20

// 遍历数组（for循环）
for (int i = 0; i < arr.length; i++) {  // length是数组的属性（不是方法）
    System.out.print(arr[i] + " ");
}
// 输出：10 20 30 

// 增强for循环（foreach，适合只读遍历）
for (int num : arr) {  // 依次取arr中的元素赋值给num
    System.out.print(num + " ");
}
// 输出：10 20 30 
```


#### 2. 二维数组
本质是“数组的数组”，可理解为表格（行和列）。  
```java
// 初始化
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// 遍历（嵌套循环）
for (int i = 0; i < matrix.length; i++) {  // 行
    for (int j = 0; j < matrix[i].length; j++) {  // 列
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println();  // 换行
}
```


### 八、方法（函数）
封装一段可重复执行的代码，提高复用性。  

#### 方法的定义
格式：  
```java
修饰符 返回值类型 方法名(参数列表) {
    // 方法体（代码逻辑）
    return 返回值;  // 若返回值类型为void，可省略return
}
```

#### 示例：定义并调用方法
```java
public class MethodDemo {
    // 定义方法：计算两数之和（有参数，有返回值）
    public static int add(int a, int b) {  // static：静态方法，可直接调用
        return a + b;
    }

    // 定义方法：打印信息（无参数，无返回值）
    public static void printInfo() {
        System.out.println("这是一个方法示例");
    }

    public static void main(String[] args) {
        // 调用方法
        int sum = add(3, 5);
        System.out.println("和为：" + sum);  // 输出：和为：8

        printInfo();  // 输出：这是一个方法示例
    }
}
```

**说明**：  
- 方法名遵循“小驼峰命名法”（如`printInfo`）；  
- 静态方法（`static`修饰）可直接通过类名调用（如`MethodDemo.add(3,5)`）；  
- 非静态方法需先创建对象，再通过对象调用（面向对象部分详细讲解）。  


### 九、注释
用于解释代码，不影响程序执行，分为3种：  

1. 单行注释：`// 注释内容`（常用于单行说明）  
   ```java
   int age = 18;  // 定义年龄变量
   ```

2. 多行注释：`/* 注释内容 */`（常用于多行说明）  
   ```java
   /*
    * 这是一个计算和的方法
    * 参数：a和b为整数
    * 返回值：两数之和
    */
   int add(int a, int b) { ... }
   ```

3. 文档注释：`/** 注释内容 */`（用于生成API文档，可包含标签如`@param`、`@return`）  
   ```java
   /**
    * 计算两数之和
    * @param a 第一个整数
    * @param b 第二个整数
    * @return 两数相加的结果
    */
   int add(int a, int b) { ... }
   ```


### 总结
Java基础语法是编写程序的基石，核心包括：  
- 程序结构：类与`main`方法；  
- 数据类型：8种基本类型 + 引用类型；  
- 变量与常量：`final`修饰常量；  
- 运算符：算术、赋值、比较、逻辑等；  
- 控制流：`if-else`、`switch`、`for`、`while`；  
- 数组：存储相同类型数据的固定长度集合；  
- 方法：封装代码，提高复用性。  

掌握这些内容后，可进一步学习面向对象（类、对象、继承、多态等）、异常处理、集合框架等进阶知识。
