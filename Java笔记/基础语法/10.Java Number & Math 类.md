在Java中，`Number`类和`Math`类是处理数值运算的核心工具类，分别用于数值类型的包装与转换、以及复杂数学运算。下面详细介绍这两个类的核心特性和使用方法。


### 一、Number类  
`Number`类是**抽象类**（位于`java.lang`包），是所有数值型包装类的父类。它的主要作用是将基本数据类型（如`int`、`double`）包装为对象，并提供数值类型之间的转换方法。  


#### 1. Number类的核心方法  
`Number`类定义了一系列抽象方法，用于将包装类对象转换为对应的基本数据类型。子类（如`Integer`、`Double`）必须实现这些方法：  

| 方法               | 说明                                  |
|--------------------|---------------------------------------|
| `int intValue()`   | 转换为`int`类型                       |
| `long longValue()` | 转换为`long`类型                      |
| `float floatValue()` | 转换为`float`类型                   |
| `double doubleValue()` | 转换为`double`类型                 |
| `byte byteValue()` | 转换为`byte`类型（默认实现为`(byte)intValue()`） |
| `short shortValue()` | 转换为`short`类型（默认实现为`(short)intValue()`） |  


#### 2. Number的子类（数值包装类）  
`Number`的子类都是**final类**（不可继承），用于包装对应的基本数据类型，使其可以作为对象使用（如在集合中存储）。常见子类如下：  

| 包装类       | 对应基本类型 | 说明                                  |
|--------------|--------------|---------------------------------------|
| `Integer`    | `int`        | 32位整数                              |
| `Long`       | `long`       | 64位整数                              |
| `Float`      | `float`      | 32位浮点数                            |
| `Double`     | `double`     | 64位浮点数                            |
| `Byte`       | `byte`       | 8位整数                               |
| `Short`      | `short`      | 16位整数                              |  


#### 3. 自动装箱与拆箱  
Java 5+支持**自动装箱**（基本类型→包装类）和**自动拆箱**（包装类→基本类型），简化了包装类的使用：  
```java
// 自动装箱：int → Integer
Integer num = 100;  // 等价于 Integer num = Integer.valueOf(100);

// 自动拆箱：Integer → int
int val = num;      // 等价于 int val = num.intValue();
```  


### 二、Math类  
`Math`类（位于`java.lang`包）是一个**工具类**，包含大量静态方法和常量，用于执行常见的数学运算（如绝对值、取整、三角函数等）。它不能被实例化（构造方法为`private`），所有方法直接通过`Math`类名调用。  


#### 1. Math类的常量  
| 常量       | 说明                     |
|------------|--------------------------|
| `Math.PI`  | 圆周率（约3.1415926535） |
| `Math.E`   | 自然常数（约2.718281828）|  


#### 2. Math类的常用方法  
按功能分类如下：  


##### （1）基本运算  
- `abs(x)`：返回`x`的绝对值（支持`int`、`long`、`float`、`double`）  
  ```java
  Math.abs(-5);    // 5（int）
  Math.abs(-3.14); // 3.14（double）
  ```  

- `max(a, b)` / `min(a, b)`：返回两个数的最大值/最小值（支持多种数值类型）  
  ```java
  Math.max(3, 7);   // 7
  Math.min(2.5, 1.8); // 1.8
  ```  


##### （2）取整运算  
- `ceil(x)`：向上取整（返回大于等于`x`的最小整数，结果为`double`）  
  ```java
  Math.ceil(2.3);  // 3.0
  Math.ceil(-2.3); // -2.0
  ```  

- `floor(x)`：向下取整（返回小于等于`x`的最大整数，结果为`double`）  
  ```java
  Math.floor(2.7); // 2.0
  Math.floor(-2.7); // -3.0
  ```  

- `round(x)`：四舍五入（`float`返回`int`，`double`返回`long`）  
  ```java
  Math.round(2.3f); // 2（int）
  Math.round(2.6);  // 3（long）
  ```  


##### （3）幂运算与对数  
- `pow(a, b)`：返回`a`的`b`次幂（`a^b`）  
  ```java
  Math.pow(2, 3);  // 8.0（2^3）
  ```  

- `sqrt(x)`：返回`x`的平方根（`x≥0`）  
  ```java
  Math.sqrt(16);   // 4.0
  ```  

- `log(x)`：返回自然对数（以`e`为底）；`log10(x)`：返回以10为底的对数  
  ```java
  Math.log(Math.E); // 1.0（ln(e)=1）
  Math.log10(100);  // 2.0（log10(100)=2）
  ```  


##### （4）随机数  
- `random()`：返回`[0.0, 1.0)`之间的随机`double`值  
  ```java
  // 生成[1, 100]之间的随机整数
  int randomNum = (int)(Math.random() * 100) + 1;
  ```  


##### （5）三角函数  
支持正弦（`sin`）、余弦（`cos`）、正切（`tan`）等，参数为弧度：  
```java
Math.sin(Math.PI / 2); // 1.0（sin(π/2)=1）
Math.cos(0);           // 1.0（cos(0)=1）
```  


### 总结  
- `Number`类是数值包装类的父类，提供基本类型与包装类的转换能力，核心是`xxxValue()`方法。  
- `Math`类是数学运算工具类，通过静态方法提供绝对值、取整、幂运算等功能，无需实例化即可使用。  

这两个类是Java数值处理的基础，在数据转换、数学计算场景中频繁使用。
