在Java中，`String`类（位于`java.lang`包）是用于处理字符串的核心类，用于表示字符序列（如文本）。它是Java中最常用的类之一，具有**不可变性**等重要特性，同时提供了丰富的方法用于字符串操作。


### 一、String类的核心特性  
1. **不可变性（Immutable）**  
   String对象创建后，其字符序列（内容）不可被修改。任何看似“修改”字符串的操作（如拼接、替换）都会创建一个新的String对象，原对象保持不变。  
   ```java
   String s = "hello";
   s += " world"; // 实际创建了新对象"hello world"，原s（"hello"）未被修改
   ```  

2. **实现的接口**  
   - `Serializable`：支持序列化（可在网络传输或存储到文件）。  
   - `Comparable<String>`：支持字符串比较（通过`compareTo`方法）。  


### 二、String对象的创建  
String对象有两种常见创建方式，本质区别在于内存存储位置：  

#### 1. 直接赋值（推荐）  
```java
String s1 = "abc"; // 字符串常量池（方法区）中创建对象（若已存在则直接复用）
```  
- 字符串常量池：JVM为了节省内存，将重复的字符串常量存储在一个专用区域（常量池），相同内容的字符串直接复用同一对象。  


#### 2. 使用`new`关键字  
```java
String s2 = new String("abc"); // 堆内存中创建新对象（即使常量池已有"abc"）
```  
- 此时会生成两个对象（特殊情况）：常量池中的"abc"（若不存在）和堆中的`String`对象（`s2`指向堆对象）。  


#### 3. 两种方式的区别（`==`与`equals`）  
- `==`：比较对象的内存地址（是否为同一对象）。  
- `equals()`：`String`重写了`equals`，比较字符串的内容是否相同。  

```java
String s1 = "abc";
String s2 = "abc";
String s3 = new String("abc");

s1 == s2;      // true（同享常量池对象，地址相同）
s1 == s3;      // false（s1在常量池，s3在堆，地址不同）
s1.equals(s3); // true（内容相同）
```  


### 三、常用方法（重点）  
String类提供了大量方法用于字符串操作，按功能分类如下：  


#### 1. 基本信息获取  
- `int length()`：返回字符串长度（字符个数）。  
  ```java
  "hello".length(); // 5
  ```  

- `char charAt(int index)`：返回指定索引（0开始）的字符（索引越界会抛`StringIndexOutOfBoundsException`）。  
  ```java
  "hello".charAt(1); // 'e'
  ```  


#### 2. 字符串比较  
- `boolean equals(Object obj)`：比较内容是否相同（区分大小写）。  
  ```java
  "Hello".equals("hello"); // false
  ```  

- `boolean equalsIgnoreCase(String anotherString)`：比较内容是否相同（不区分大小写）。  
  ```java
  "Hello".equalsIgnoreCase("hello"); // true
  ```  

- `int compareTo(String anotherString)`：按字典顺序比较（返回差值：正数当前大，负数参数大，0相等）。  
  ```java
  "a".compareTo("b"); // -1（'a'比'b'小1）
  "abc".compareTo("ab"); // 1（长度长的大）
  ```  


#### 3. 字符串查找  
- `boolean contains(CharSequence s)`：判断是否包含子字符串`s`。  
  ```java
  "hello world".contains("world"); // true
  ```  

- `int indexOf(String str)`：返回子字符串`str`首次出现的索引（未找到返回-1）。  
  ```java
  "hello".indexOf("l"); // 2（首次出现的索引）
  ```  

- `int lastIndexOf(String str)`：返回子字符串`str`最后出现的索引（未找到返回-1）。  
  ```java
  "hello".lastIndexOf("l"); // 3
  ```  

- `boolean startsWith(String prefix)`：判断是否以`prefix`开头。  
- `boolean endsWith(String suffix)`：判断是否以`suffix`结尾。  
  ```java
  "hello".startsWith("he"); // true
  "hello".endsWith("lo");   // true
  ```  


#### 4. 字符串截取与拆分  
- `String substring(int beginIndex)`：从`beginIndex`截取到末尾。  
- `String substring(int beginIndex, int endIndex)`：截取`[beginIndex, endIndex)`范围的子串（包含开始，不包含结束）。  
  ```java
  "hello world".substring(6);      // "world"
  "hello world".substring(0, 5);   // "hello"
  ```  

- `String[] split(String regex)`：按正则表达式`regex`拆分字符串为数组。  
  ```java
  "a,b,c".split(","); // 结果：["a", "b", "c"]
  "a  b c".split("\\s+"); // 按1个或多个空格拆分：["a", "b", "c"]（注意正则转义）
  ```  


#### 5. 字符串修改（返回新对象）  
- `String replace(char oldChar, char newChar)`：替换所有旧字符为新字符。  
  ```java
  "hello".replace('l', 'x'); // "hexxo"
  ```  

- `String replace(CharSequence target, CharSequence replacement)`：替换所有子串`target`为`replacement`。  
  ```java
  "hello world".replace("world", "java"); // "hello java"
  ```  

- `String toUpperCase()`：转换为大写字母。  
- `String toLowerCase()`：转换为小写字母。  
  ```java
  "Hello".toUpperCase(); // "HELLO"
  "HELLO".toLowerCase(); // "hello"
  ```  

- `String trim()`：去除首尾空白字符（空格、制表符`\t`、换行符`\n`等）。  
  ```java
  "  hello  ".trim(); // "hello"
  ```  

- `String strip()`：JDK 11+新增，功能类似`trim()`，但支持更多Unicode空白字符。  


#### 6. 其他常用方法  
- `static String valueOf(...)`：将其他类型（如`int`、`Object`）转换为字符串（常用工具方法）。  
  ```java
  String.valueOf(123); // "123"
  String.valueOf(true); // "true"
  ```  

- `boolean isEmpty()`：判断字符串是否为空（长度为0）。  
  ```java
  "".isEmpty(); // true
  "a".isEmpty(); // false
  ```  


### 四、不可变性的影响与优化  
由于String的不可变性，频繁修改字符串（如循环拼接）会产生大量临时对象，浪费内存。此时推荐使用**可变字符串类**：  
- `StringBuilder`：线程不安全，效率高（单线程优先）。  
- `StringBuffer`：线程安全（有同步锁），效率稍低（多线程场景）。  

示例（高效拼接）：  
```java
StringBuilder sb = new StringBuilder();
for (int i = 0; i < 1000; i++) {
    sb.append(i); // 直接修改内部字符数组，不创建新对象
}
String result = sb.toString(); // 最终转换为String
```  


### 总结  
String类是Java中处理文本的核心类，核心特点：  
1. **不可变性**：内容一旦创建无法修改，修改操作会生成新对象。  
2. **常量池复用**：直接赋值的字符串会复用常量池中的对象，节省内存。  
3. **丰富方法**：提供比较、查找、截取、替换等大量工具方法，满足日常字符串操作需求。  

掌握String类的用法是Java开发的基础，尤其要注意`==`与`equals`的区别，以及不可变性带来的性能影响。
