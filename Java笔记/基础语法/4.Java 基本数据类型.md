Java是强类型语言，要求变量必须声明明确的数据类型。**基本数据类型（Primitive Types）** 是Java语言内置的基础类型，直接存储数据值（而非引用地址），共8种，分为四大类。它们是Java编程的基础，决定了数据的存储方式、取值范围和默认行为。


### 一、基本数据类型分类总览
8种基本数据类型按用途分为4类，核心信息如下表：

| **类别**       | **类型名称** | **占用空间** | **取值范围**                                  | **默认值**（成员变量） | **典型用途**                          |
|----------------|--------------|--------------|-----------------------------------------------|-------------------------|---------------------------------------|
| **整数型**     | `byte`       | 1字节（8位） | $-2^7$ ~ $2^7-1$（-128 ~ 127）                 | 0                       | 节省内存的小整数（如文件字节流）      |
|                | `short`      | 2字节（16位）| $-2^{15}$ ~ $2^{15}-1$（-32768 ~ 32767）       | 0                       | 较少使用（兼容旧系统）                |
|                | `int`（默认）| 4字节（32位）| $-2^{31}$ ~ $2^{31}-1$（约±21亿）              | 0                       | 大多数整数场景（计数、索引等）        |
|                | `long`       | 8字节（64位）| $-2^{63}$ ~ $2^{63}-1$（约±90亿亿）            | 0L                      | 大整数（时间戳、ID等）                |
| **浮点型**     | `float`      | 4字节（32位）| ±3.4×$10^{38}$（单精度，精度6-7位）            | 0.0f                    | 内存紧张的浮点场景（如游戏图形）      |
|                | `double`（默认）| 8字节（64位）| ±1.8×$10^{308}$（双精度，精度15-17位）         | 0.0d                    | 大多数浮点计算（科学计算、小数等）    |
| **字符型**     | `char`       | 2字节（16位）| 0 ~ 65535（Unicode字符集）                     | '\u0000'（空字符）      | 存储单个字符（字母、符号、汉字等）    |
| **布尔型**     | `boolean`    | 1字节（8位） | `true`（真）或`false`（假）                    | false                   | 逻辑判断（条件语句、开关等）          |  


### 二、各类型详解及示例


#### 1. 整数型（整数数据，无小数部分）
整数型用于存储没有小数的数值，区别在于取值范围和占用空间（空间越小，范围越窄）。

##### （1）`byte`（1字节）
- 最小的整数类型，范围极窄（-128 ~ 127），适合存储字节数据（如文件读写中的字节流）。  
- 示例：  
  ```java
  byte b1 = 100;    // 合法（在范围内）
  byte b2 = -128;   // 合法（最小值）
  // byte b3 = 128; // 错误（超出范围）
  ```


##### （2）`short`（2字节）
- 范围比`byte`大，但仍有限（-32768 ~ 32767），实际开发中较少使用（主要用于兼容早期系统或节省内存的数组）。  
- 示例：  
  ```java
  short s1 = 30000;  // 合法
  short s2 = -32768; // 合法（最小值）
  // short s3 = 32768; // 错误（超出范围）
  ```


##### （3）`int`（4字节，默认整数类型）
- 最常用的整数类型，范围覆盖大多数日常场景（±21亿），整数直接量（如`100`、`-50`）默认是`int`类型。  
- 示例：  
  ```java
  int num1 = 100000;    // 合法
  int num2 = -2000000000; // 合法（在范围内）
  // int num3 = 2147483648; // 错误（超出int最大值2147483647）
  ```


##### （4）`long`（8字节）
- 用于存储超大整数（如时间戳`System.currentTimeMillis()`返回的就是`long`类型）。  
- **注意**：`long`类型的直接量需加后缀`L`或`l`（推荐大写`L`，避免与数字`1`混淆），否则会被当作`int`处理，超出范围时编译报错。  
- 示例：  
  ```java
  long l1 = 10000000000L; // 正确（加L，否则10000000000超出int范围）
  long l2 = -9223372036854775808L; // 合法（long最小值）
  ```


#### 2. 浮点型（带小数的数值）
浮点型用于存储有小数部分的数值，分为单精度（`float`）和双精度（`double`），精度指有效数字的位数。

##### （1）`float`（4字节，单精度）
- 精度较低（6-7位有效数字），占用空间小，适合对精度要求不高且内存紧张的场景（如游戏中的坐标）。  
- **注意**：`float`类型的直接量需加后缀`F`或`f`，否则默认是`double`类型，直接赋值会编译报错。  
- 示例：  
  ```java
  float f1 = 3.14F;   // 正确（加F）
  float f2 = -0.0001f; // 正确（加f）
  // float f3 = 3.14;  // 错误（3.14默认是double，类型不兼容）
  ```


##### （2）`double`（8字节，双精度，默认浮点类型）
- 精度更高（15-17位有效数字），是浮点型的默认类型（如`3.14`、`-0.5`默认是`double`），适合大多数需要小数的场景（如科学计算、金额计算需谨慎，推荐用`BigDecimal`）。  
- 示例：  
  ```java
  double d1 = 3.1415926535; // 正确（默认double）
  double d2 = 1.2e3;        // 科学计数法（1.2×10³ = 1200.0）
  ```


#### 3. 字符型（`char`，2字节）
`char`用于存储单个字符，基于Unicode字符集（支持英文、中文、符号等全球字符），取值范围0 ~ 65535（对应Unicode编码值）。  

- 表示方式：  
  - 单引号包裹单个字符：`'A'`、`'中'`、`'$'`；  
  - Unicode转义字符：`'\uXXXX'`（XXXX是4位十六进制数，对应Unicode编码），如`'\u0041'`等价于`'A'`。  
- 示例：  
  ```java
  char c1 = 'A';       // 字母
  char c2 = '中';      // 汉字（Unicode支持）
  char c3 = '\u0031';  // Unicode编码，对应字符'1'
  char c4 = 65;        // 直接用ASCII码（65对应'A'）
  ```


#### 4. 布尔型（`boolean`，1字节）
`boolean`仅用于表示逻辑值，只有两个取值：`true`（真）和`false`（假），主要用于条件判断（如`if`语句、循环条件）。  

- 特点：  
  - 不能参与算术运算（如`true + 1`是错误的）；  
  - 没有明确的“数值对应”（不同于C语言的0和非0），仅能直接赋值`true`/`false`或通过逻辑表达式获取。  
- 示例：  
  ```java
  boolean isPass = true;
  boolean hasError = false;
  boolean isAdult = (age >= 18); // 逻辑表达式结果（true或false）
  ```


### 三、基本数据类型的核心特性
1. **直接存储值**：基本数据类型的变量存储的是“数据本身”，而非引用地址（与引用类型的核心区别）。  
   ```java
   int a = 10;
   int b = a; // b直接拷贝a的值（10），a和b相互独立
   ```

2. **默认值仅适用于成员变量**：  
   - 成员变量（类中方法外）未手动初始化时，会被赋予默认值（见表格）；  
   - 局部变量（方法内或代码块内）**必须手动初始化**，否则编译报错。  
   ```java
   public class Test {
       static int num; // 成员变量，默认0
       public static void main(String[] args) {
           // int localNum; // 错误：局部变量未初始化
           int localNum = 20; // 正确：手动初始化
       }
   }
   ```

3. **不可为`null`**：基本数据类型的变量必须存储具体值，不能赋值`null`（`null`是引用类型的“空引用”）。  
   ```java
   // int a = null; // 错误：基本类型不能为null
   String str = null; // 正确：引用类型可以为null
   ```

4. **自动类型提升与强制类型转换**：  
   - 小范围类型可自动转换为大范围类型（如`byte → short → int → long → float → double`）；  
   - 大范围类型转小范围类型需强制转换（可能丢失精度或溢出）。  
   ```java
   int i = 100;
   long l = i; // 自动提升（int → long）
   int j = (int)l; // 强制转换（long → int，l值需在int范围内）
   ```


### 四、使用建议
- 优先使用`int`和`double`：它们是整数和浮点的默认类型，覆盖大多数场景，且兼容性好；  
- 大整数用`long`：需加`L`后缀，避免溢出；  
- 内存敏感场景用`byte`/`short`：如大量整数存储（数组），可节省内存；  
- 字符用`char`：存储单个字符，注意Unicode编码支持全球字符；  
- 逻辑判断用`boolean`：仅`true`/`false`，清晰表达逻辑。  


### 总结
Java的8种基本数据类型是构建程序的基础，核心区别在于**占用空间**和**取值范围**。掌握它们的特点（如`long`需加`L`、`float`需加`F`、局部变量必须初始化等），能避免类型错误和内存浪费，是编写正确Java程序的前提。
