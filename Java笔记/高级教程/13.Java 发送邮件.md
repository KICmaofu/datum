Java 发送邮件是网络编程中非常实用的场景（本质是通过 SMTP 协议与邮件服务器通信），通常使用 **Jakarta Mail**（原 JavaMail）API 实现。以下是详细的实现案例，包括发送普通文本邮件、带附件的邮件、HTML 格式邮件等，并附带关键配置说明。


### 一、准备工作  
#### 1. 依赖配置  
Java 发送邮件需引入 `Jakarta Mail` 依赖（原 JavaMail，因 Java EE 迁移至 Eclipse 基金会后更名）。  

**Maven 依赖**（`pom.xml`）：  
```xml
<dependency>
    <groupId>com.sun.mail</groupId>
    <artifactId>jakarta.mail</artifactId>
    <version>2.0.1</version> <!-- 最新稳定版 -->
</dependency>
```  

**Gradle 依赖**：  
```groovy
implementation 'com.sun.mail:jakarta.mail:2.0.1'
```  


#### 2. 邮件服务器配置  
发送邮件需依赖 **SMTP 服务器**（如 QQ 邮箱、163 邮箱、企业邮箱的 SMTP 服务），需提前获取以下信息：  
- **SMTP 服务器地址**（如 QQ 邮箱：`smtp.qq.com`；163 邮箱：`smtp.163.com`）。  
- **SMTP 端口**（SSL 加密通常用 465；非加密或 STARTTLS 用 25，但多数邮箱强制 SSL）。  
- **邮箱账号**（如 `your-email@qq.com`）。  
- **授权码**（非登录密码！多数邮箱（如 QQ、163）需在设置中开启 SMTP 服务，并生成授权码）。  


#### 关键步骤：开启 SMTP 服务并获取授权码（以 QQ 邮箱为例）  
1. 登录 QQ 邮箱 → 进入「设置」→「账户」。  
2. 找到「POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV 服务」，开启「SMTP 服务」。  
3. 按提示完成验证（如短信验证），生成**授权码**（保存此授权码，发送邮件时用它代替密码）。  


### 二、核心类说明  
Jakarta Mail 中发送邮件的核心类：  
- `Session`：邮件会话，配置 SMTP 服务器信息、认证方式等。  
- `Message`：邮件消息对象，封装收件人、主题、内容等。  
- `Transport`：负责发送邮件的传输类。  
- `MimeMessage`：`Message` 的实现类，支持复杂邮件（带附件、HTML 等）。  
- `MimeMultipart`：多部分邮件内容（可包含文本、附件等）。  
- `MimeBodyPart`：邮件的一个部分（如文本内容、单个附件）。  


### 三、实际案例  

#### 案例 1：发送普通文本邮件  
发送简单的纯文本邮件（收件人、主题、正文）。  

```java
import jakarta.mail.*;
import jakarta.mail.internet.InternetAddress;
import jakarta.mail.internet.MimeMessage;
import java.util.Properties;

public class SendTextEmail {
    // 发件人信息
    private static final String FROM_EMAIL = "your-email@qq.com"; // 发件人邮箱
    private static final String FROM_PASSWORD = "your-auth-code"; // 授权码（非密码）
    // SMTP 服务器配置（QQ 邮箱示例）
    private static final String SMTP_HOST = "smtp.qq.com"; // SMTP 服务器地址
    private static final String SMTP_PORT = "465"; // SSL 端口

    public static void main(String[] args) {
        // 1. 配置邮件会话属性
        Properties props = new Properties();
        props.put("mail.smtp.host", SMTP_HOST); // SMTP 服务器地址
        props.put("mail.smtp.port", SMTP_PORT); // 端口
        props.put("mail.smtp.ssl.enable", "true"); // 启用 SSL 加密
        props.put("mail.smtp.auth", "true"); // 需要认证

        // 2. 创建会话（带认证）
        Session session = Session.getInstance(props, new Authenticator() {
            @Override
            protected PasswordAuthentication getPasswordAuthentication() {
                return new PasswordAuthentication(FROM_EMAIL, FROM_PASSWORD);
            }
        });
        session.setDebug(true); // 开启调试模式（可选，打印发送过程日志）

        try {
            // 3. 创建邮件消息
            Message message = new MimeMessage(session);
            message.setFrom(new InternetAddress(FROM_EMAIL)); // 发件人
            message.setRecipients(Message.RecipientType.TO, 
                    InternetAddress.parse("recipient@example.com")); // 收件人（可多个，用逗号分隔）
            message.setSubject("测试：Java 发送文本邮件"); // 主题
            message.setText("这是一封通过 Java 程序发送的普通文本邮件！"); // 正文

            // 4. 发送邮件
            Transport.send(message);
            System.out.println("邮件发送成功！");

        } catch (MessagingException e) {
            throw new RuntimeException("邮件发送失败：" + e.getMessage());
        }
    }
}
```  


#### 案例 2：发送带附件的邮件  
在文本邮件基础上添加附件（如图片、文档等）。  

```java
import jakarta.mail.*;
import jakarta.mail.internet.*;
import java.io.File;
import java.util.Properties;

public class SendEmailWithAttachment {
    // 发件人信息（同案例 1）
    private static final String FROM_EMAIL = "your-email@qq.com";
    private static final String FROM_PASSWORD = "your-auth-code";
    private static final String SMTP_HOST = "smtp.qq.com";
    private static final String SMTP_PORT = "465";

    public static void main(String[] args) {
        // 1. 配置会话（同案例 1）
        Properties props = new Properties();
        props.put("mail.smtp.host", SMTP_HOST);
        props.put("mail.smtp.port", SMTP_PORT);
        props.put("mail.smtp.ssl.enable", "true");
        props.put("mail.smtp.auth", "true");

        Session session = Session.getInstance(props, new Authenticator() {
            @Override
            protected PasswordAuthentication getPasswordAuthentication() {
                return new PasswordAuthentication(FROM_EMAIL, FROM_PASSWORD);
            }
        });

        try {
            // 2. 创建邮件消息
            MimeMessage message = new MimeMessage(session);
            message.setFrom(new InternetAddress(FROM_EMAIL));
            message.setRecipients(Message.RecipientType.TO, 
                    InternetAddress.parse("recipient@example.com"));
            message.setSubject("测试：Java 发送带附件的邮件");

            // 3. 构建多部分内容（文本 + 附件）
            MimeMultipart multipart = new MimeMultipart();

            // 3.1 添加文本部分
            MimeBodyPart textPart = new MimeBodyPart();
            textPart.setText("这是一封带附件的邮件，请查收！");
            multipart.addBodyPart(textPart);

            // 3.2 添加附件（示例：添加本地图片）
            MimeBodyPart attachmentPart = new MimeBodyPart();
            String filePath = "C:/test.jpg"; // 附件本地路径
            File file = new File(filePath);
            DataSource source = new FileDataSource(file);
            attachmentPart.setDataHandler(new DataHandler(source));
            // 设置附件文件名（解决中文乱码）
            attachmentPart.setFileName(MimeUtility.encodeWord(file.getName()));
            multipart.addBodyPart(attachmentPart);

            // 4. 将多部分内容设置为邮件内容
            message.setContent(multipart);

            // 5. 发送邮件
            Transport.send(message);
            System.out.println("带附件的邮件发送成功！");

        } catch (Exception e) {
            throw new RuntimeException("邮件发送失败：" + e.getMessage());
        }
    }
}
```  


#### 案例 3：发送 HTML 格式邮件  
支持 HTML 标签（如换行、加粗、链接等），适合发送富文本内容。  

```java
import jakarta.mail.*;
import jakarta.mail.internet.InternetAddress;
import jakarta.mail.internet.MimeMessage;
import java.util.Properties;

public class SendHtmlEmail {
    // 发件人信息（同案例 1）
    private static final String FROM_EMAIL = "your-email@qq.com";
    private static final String FROM_PASSWORD = "your-auth-code";
    private static final String SMTP_HOST = "smtp.qq.com";
    private static final String SMTP_PORT = "465";

    public static void main(String[] args) {
        // 1. 配置会话（同案例 1）
        Properties props = new Properties();
        props.put("mail.smtp.host", SMTP_HOST);
        props.put("mail.smtp.port", SMTP_PORT);
        props.put("mail.smtp.ssl.enable", "true");
        props.put("mail.smtp.auth", "true");

        Session session = Session.getInstance(props, new Authenticator() {
            @Override
            protected PasswordAuthentication getPasswordAuthentication() {
                return new PasswordAuthentication(FROM_EMAIL, FROM_PASSWORD);
            }
        });

        try {
            // 2. 创建邮件消息
            MimeMessage message = new MimeMessage(session);
            message.setFrom(new InternetAddress(FROM_EMAIL));
            message.setRecipients(Message.RecipientType.TO, 
                    InternetAddress.parse("recipient@example.com"));
            message.setSubject("测试：Java 发送 HTML 邮件");

            // 3. 设置 HTML 内容（支持 <b>、<br>、<a> 等标签）
            String htmlContent = """
                    <h1 style="color: red;">这是 HTML 邮件标题</h1>
                    <p>这是一段 <b>加粗</b> 的文本</p>
                    <p>点击访问：<a href="https://www.baidu.com">百度</a></p>
                    """;
            message.setContent(htmlContent, "text/html; charset=utf-8"); // 指定内容类型为 HTML

            // 4. 发送邮件
            Transport.send(message);
            System.out.println("HTML 邮件发送成功！");

        } catch (MessagingException e) {
            throw new RuntimeException("邮件发送失败：" + e.getMessage());
        }
    }
}
```  


### 四、常见问题与注意事项  
1. **认证失败（535 错误）**：  
   - 检查授权码是否正确（非登录密码）。  
   - 确认已开启 SMTP 服务（在邮箱设置中检查）。  

2. **连接超时/端口被拒绝**：  
   - 检查 SMTP 服务器地址和端口是否正确（如 QQ 用 `smtp.qq.com:465`，163 用 `smtp.163.com:465`）。  
   - 关闭防火墙或添加端口例外（部分环境会屏蔽 25、465 端口）。  

3. **中文乱码**：  
   - 附件文件名需用 `MimeUtility.encodeWord()` 编码（如案例 2）。  
   - HTML 内容或文本需指定编码（如 `text/html; charset=utf-8`）。  

4. **抄送/密送**：  
   - 抄送（CC）：`message.setRecipients(Message.RecipientType.CC, InternetAddress.parse("cc@example.com"))`  
   - 密送（BCC）：`message.setRecipients(Message.RecipientType.BCC, InternetAddress.parse("bcc@example.com"))`  


### 总结  
Java 发送邮件的核心是通过 Jakarta Mail API 与 SMTP 服务器通信，关键步骤包括：  
1. 配置 SMTP 服务器信息（地址、端口、SSL 等）。  
2. 创建带认证的邮件会话。  
3. 构建邮件内容（文本、HTML、附件等）。  
4. 通过 `Transport` 发送邮件。  

实际开发中，可根据需求扩展（如批量发送、模板邮件），并注意异常处理和安全配置（如避免硬编码授权码）。
