在C语言中，**循环结构**用于重复执行某段代码（循环体），直到满足终止条件。C语言提供了三种主要的循环语句：`for`循环、`while`循环和`do-while`循环，分别适用于不同的场景。


### 一、`for` 循环（最常用，适合次数已知的场景）
`for`循环的语法清晰，将循环的“初始化”“条件判断”“更新”三个关键操作集中在一行，适合循环次数可提前确定的情况（如从1循环到100）。


#### 语法
```c
for (初始化表达式; 条件表达式; 更新表达式) {
    // 循环体（重复执行的代码）
}
```


#### 执行流程
1. **初始化表达式**：循环开始前执行**一次**（通常用于定义和初始化循环变量，如`int i = 0`）。  
2. **条件表达式**：每次执行循环体前检查。若结果为**非0（真）**，执行循环体；若为**0（假）**，退出循环。  
3. **更新表达式**：执行完循环体后执行（通常用于更新循环变量，如`i++`）。  
4. 重复步骤2和3，直到条件表达式为假。  


#### 示例：计算1到10的和
```c
#include <stdio.h>

int main() {
    int sum = 0;
    // 初始化i=1；条件i<=10；每次循环后i+1
    for (int i = 1; i <= 10; i++) {
        sum += i;  // 累加i到sum中
    }
    printf("1到10的和为：%d\n", sum);  // 输出：55
    return 0;
}
```


#### 注意事项
- 初始化、条件、更新表达式均可省略，但分号`;`必须保留。例如：  
  - 省略初始化：`int i=0; for (; i<5; i++) { ... }`（循环变量在外部定义）。  
  - 省略条件：`for (int i=0; ; i++) { ... }`（条件恒为真，需在循环体内用`break`终止，否则死循环）。  
- 循环变量的作用域：若在`for`的初始化中定义（如`int i=0`），则仅在循环体内有效（C99及以上标准支持）。  


### 二、`while` 循环（适合次数未知的场景）
`while`循环先判断条件，再执行循环体，适合循环次数不确定（需根据动态条件终止）的场景（如等待用户输入特定值）。


#### 语法
```c
while (条件表达式) {
    // 循环体
}
```


#### 执行流程
1. 检查**条件表达式**：若为真（非0），执行循环体；若为假（0），退出循环。  
2. 执行完循环体后，再次回到步骤1检查条件，直到条件为假。  


#### 示例：计算用户输入的整数之和，直到输入0为止
```c
#include <stdio.h>

int main() {
    int num, sum = 0;
    printf("请输入整数（输入0结束）：");
    scanf("%d", &num);

    while (num != 0) {  // 条件：输入的数不等于0
        sum += num;
        printf("请输入整数（输入0结束）：");
        scanf("%d", &num);  // 更新条件（输入新的数）
    }

    printf("所有数的和为：%d\n", sum);
    return 0;
}
```


#### 注意事项
- 若循环体中未更新条件表达式的相关变量，可能导致**死循环**（条件永远为真）。例如：  
  ```c
  int i = 0;
  while (i < 5) {
      printf("循环中...");  // 未更新i，i始终为0，条件恒真，无限循环
  }
  ```  
- 循环体若只有一条语句，可省略`{}`，但建议保留以增强可读性。  


### 三、`do-while` 循环（至少执行一次的场景）
`do-while`循环与`while`循环的区别是：**先执行一次循环体，再判断条件**，因此循环体**至少执行一次**，适合“至少执行一次”的场景（如密码验证、菜单选择）。


#### 语法
```c
do {
    // 循环体
} while (条件表达式);  // 注意末尾有分号
```


#### 执行流程
1. 执行**循环体**（至少一次）。  
2. 检查**条件表达式**：若为真（非0），回到步骤1继续执行循环体；若为假（0），退出循环。  


#### 示例：验证密码（至少输入一次密码）
```c
#include <stdio.h>
#include <string.h>  // 用于strcmp函数

int main() {
    char password[20];
    const char correct_pwd[] = "123456";

    do {
        printf("请输入密码：");
        scanf("%s", password);  // 输入密码（假设不包含空格）
    } while (strcmp(password, correct_pwd) != 0);  // 条件：密码不正确则重复

    printf("密码正确，登录成功！\n");
    return 0;
}
```


#### 注意事项
- `do-while`循环的条件表达式后必须加**分号`;`**，否则编译错误。  
- 适合“先执行，后判断”的场景，避免条件初始化的冗余代码（如`while`循环需提前执行一次循环体相关操作）。  


### 四、循环控制语句（`break` 和 `continue`）
在循环体中，可通过`break`和`continue`控制循环的执行流程：


#### 1. `break`：终止整个循环
当`break`被执行时，立即跳出当前循环（不再执行后续循环体和更新操作），继续执行循环外的代码。

**示例**：找到100以内第一个能被13整除的数并停止  
```c
#include <stdio.h>

int main() {
    for (int i = 1; i <= 100; i++) {
        if (i % 13 == 0) {
            printf("找到：%d\n", i);  // 输出：13
            break;  // 找到后立即终止循环
        }
    }
    return 0;
}
```


#### 2. `continue`：跳过本次循环剩余部分
当`continue`被执行时，跳过当前循环体中`continue`后的代码，直接进入下一次循环（执行更新表达式或条件判断）。

**示例**：计算1到10中所有奇数的和（跳过偶数）  
```c
#include <stdio.h>

int main() {
    int sum = 0;
    for (int i = 1; i <= 10; i++) {
        if (i % 2 == 0) {
            continue;  // 若i是偶数，跳过后续累加操作
        }
        sum += i;  // 只累加奇数
    }
    printf("奇数和为：%d\n", sum);  // 输出：25（1+3+5+7+9）
    return 0;
}
```


#### `break` 与 `continue` 的区别
- `break`：直接终止**整个循环**，循环不再执行。  
- `continue`：仅跳过**本次循环的剩余部分**，下一次循环仍可能执行。  


### 五、三种循环的适用场景对比
| 循环类型   | 核心特点                          | 适用场景                                  |
|------------|-----------------------------------|-------------------------------------------|
| `for`      | 初始化、条件、更新集中管理        | 循环次数已知（如从1到n）                  |
| `while`    | 先判断条件，再执行循环体          | 循环次数未知（需根据动态条件终止）        |
| `do-while` | 先执行循环体，再判断条件（至少1次）| 需至少执行一次循环体（如密码验证、菜单）  |


### 总结
循环是C语言中处理重复操作的核心工具，选择合适的循环类型可使代码更简洁、逻辑更清晰。`for`适合次数固定的场景，`while`适合次数动态的场景，`do-while`适合至少执行一次的场景。通过`break`和`continue`可灵活控制循环流程，但需避免滥用导致逻辑混乱。
