C语言数据类型决定了数据的存储方式、取值范围和可执行的操作，核心分为**基本数据类型、构造数据类型、指针类型、空类型**四大类，其中基本数据类型是入门重点，以下是结构化讲解，含示例代码和关键注意事项：

### 一、基本数据类型（核心重点）
基本数据类型是C语言的基础，直接存储单一值，分为整型、字符型、浮点型三类，不同编译器（如GCC、VC）占用字节可能略有差异（以下以32位系统为例）。

#### 1. 整型（存储整数，正负均可）
| 类型        | 占用字节 | 取值范围                  | 说明                     |
|-------------|----------|---------------------------|--------------------------|
| `char`      | 1        | -128 ~ 127（有符号）      | 本质是8位整型，可存字符ASCII码 |
| `short`     | 2        | -32768 ~ 32767            | 短整型，节省内存         |
| `int`       | 4        | -2147483648 ~ 2147483647  | 最常用，默认整型         |
| `long`      | 4/8      | 32位系统4字节，64位8字节  | 长整型，存储大范围整数   |
| `unsigned 类型` | 同对应类型 | 0 ~ 2^n - 1（n为位数）    | 无符号整型，仅存非负数   |

**示例代码**：
```c
#include <stdio.h>
int main() {
    int age = 20;                  // 普通整型
    unsigned int score = 95;       // 无符号整型（不能存负数）
    long num = 1234567890;         // 长整型
    printf("年龄：%d\n", age);
    printf("无符号成绩：%u\n", score); // %u对应unsigned int
    printf("长整数：%ld\n", num);   // %ld对应long
    return 0;
}
```

**关键注意**：
- `char`可看作“最小整型”，存储字符时实际存ASCII码（如`'a'`对应97）。
- 无符号类型需用`%u`（unsigned int）、`%lu`（unsigned long）格式化输出，否则可能出错。

#### 2. 浮点型（存储小数，精度不同）
| 类型        | 占用字节 | 精度（有效数字） | 取值范围                  | 说明                     |
|-------------|----------|------------------|---------------------------|--------------------------|
| `float`     | 4        | 6~7位            | ±3.4×10^-38 ~ ±3.4×10^38  | 单精度浮点型             |
| `double`    | 8        | 15~16位          | ±1.7×10^-308 ~ ±1.7×10^308 | 双精度浮点型，精度更高   |

**示例代码**：
```c
#include <stdio.h>
int main() {
    float pi1 = 3.14159f;  // float需加f后缀，否则默认double
    double pi2 = 3.1415926535;
    printf("单精度圆周率：%f\n", pi1);   // %f对应float/double
    printf("双精度圆周率：%lf\n", pi2);  // %lf更规范（部分编译器兼容%f）
    return 0;
}
```

**关键注意**：
- `float`必须加`f`后缀，否则编译器会默认当作`double`处理。
- 浮点型存在精度误差（如`0.1 + 0.2 != 0.3`），需避免直接用`==`比较。

#### 3. 布尔型（C99标准新增）
- 需包含头文件`<stdbool.h>`，用`bool`定义布尔变量。
- 取值只有`true`（1）和`false`（0），本质是`unsigned char`的别名。

**示例代码**：
```c
#include <stdio.h>
#include <stdbool.h>
int main() {
    bool isPass = true;
    printf("是否通过：%d\n", isPass); // 输出1（true对应1）
    return 0;
}
```

### 二、构造数据类型（组合基本类型）
构造数据类型由基本类型组合而成，用于存储复杂数据，入门阶段重点掌握数组，其他类型简要了解。

#### 1. 数组（批量存储相同类型数据）
- 定义格式：`类型 数组名[长度];`（长度必须是常量）。
- 索引从0开始，未初始化元素默认值为0（全局数组）或随机值（局部数组）。

**示例代码**：
```c
#include <stdio.h>
int main() {
    int scores[5] = {85, 92, 78, 90, 88}; // 整型数组
    char chars[3] = {'a', 'b', 'c'};      // 字符数组
    printf("第一个成绩：%d\n", scores[0]); // 索引0是第一个元素
    printf("第二个字符：%c\n", chars[1]);  // 输出'b'
    return 0;
}
```

#### 2. 结构体（存储不同类型数据）
- 定义格式：`struct 结构体名 { 类型 成员1; 类型 成员2; ... };`。
- 用于封装多个不同类型的变量（如存储一个人的姓名、年龄、成绩）。

**示例代码**：
```c
#include <stdio.h>
// 定义结构体
struct Person {
    char name[20]; // 姓名（字符数组）
    int age;       // 年龄
    float score;   // 成绩
};

int main() {
    struct Person p = {"张三", 20, 95.5}; // 初始化结构体变量
    printf("姓名：%s，年龄：%d\n", p.name, p.age);
    return 0;
}
```

#### 3. 其他构造类型（简要了解）
- **共用体（union）**：所有成员共享同一块内存，同一时间只能存储一个成员的值，节省内存。
- **枚举（enum）**：定义一组有名字的常量（如`enum Color { RED, GREEN, BLUE };`），默认值从0开始。

### 三、指针类型（存储内存地址）
- 定义格式：`类型 *指针变量名;`（如`int *p;`表示指向int类型变量的指针）。
- 核心作用：通过指针访问内存地址，实现高效数据操作（如数组遍历、函数传参）。

**示例代码**：
```c
#include <stdio.h>
int main() {
    int a = 10;
    int *p = &a; // p存储a的内存地址（&是取地址符）
    printf("a的值：%d\n", a);
    printf("p指向的值：%d\n", *p); // *是解引用符，获取指针指向的值
    return 0;
}
```

### 四、空类型（void）
- 无具体取值，主要用于三个场景：
  1. 函数返回值：`void func()`表示函数无返回值。
  2. 函数参数：`void func(void)`表示函数无参数。
  3. 指针类型：`void *p`表示通用指针，可指向任意类型数据（需强转后使用）。

**示例代码**：
```c
#include <stdio.h>
// 无返回值函数
void printHello() {
    printf("Hello, void!\n");
}

int main() {
    printHello(); // 调用无返回值函数
    return 0;
}
```

### 五、常见误区总结
1. 混淆`char`的类型：`char`是整型，不是专门的“字符类型”，存储的是ASCII码。
2. 浮点型直接比较：`float a = 0.1, b = 0.2;`，`a + b`不等于`0.3`，需用差值判断（如`fabs(a+b-0.3) < 1e-6`）。
3. 数组长度动态改变：C语言数组长度必须是常量，不能用变量定义（如`int n=5; int arr[n];`是错误的）。
4. 无符号类型存负数：`unsigned int a = -1;`会溢出，结果是最大值（2^32-1）。
