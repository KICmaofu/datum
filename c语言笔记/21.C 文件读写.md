在C语言中，文件读写是程序与外部存储（如硬盘文件）交互的核心功能，主要通过标准库`stdio.h`提供的函数实现。与标准输入输出（键盘/屏幕）不同，文件读写需要显式打开文件、操作数据，最后关闭文件，核心是通过**文件指针**（`FILE*`）管理文件。


### 一、文件的基本概念
- **文件**：存储在外部介质（如硬盘）中的数据集合，C语言中以**流**（Stream）的形式处理（字节序列）。  
- **文本文件**：以字符ASCII码形式存储（如`.txt`），可被文本编辑器直接读取，换行符可能被系统转换（如Windows用`\r\n`，Linux用`\n`）。  
- **二进制文件**：以数据的二进制形式存储（如`.exe`、图片、音频），与内存中的数据格式一致，无转换，适合存储结构化数据（如结构体）。  
- **文件指针**：`FILE*`类型的指针，用于关联打开的文件，是所有文件操作的入口（由`fopen`函数返回）。  


### 二、文件操作的基本步骤
C语言文件操作遵循固定流程：  
1. **打开文件**：用`fopen`函数打开文件，获取文件指针（`FILE*`）。  
2. **读写文件**：通过文件指针调用读写函数（如`fread`、`fwrite`、`fprintf`等）。  
3. **关闭文件**：用`fclose`函数关闭文件，释放资源（必须执行，否则可能导致数据丢失）。  


### 三、文件的打开与关闭
#### 1. 打开文件：`fopen`函数
函数原型：  
```c
FILE *fopen(const char *filename, const char *mode);
```  
- **参数**：  
  - `filename`：文件名（含路径，如`"data.txt"`或`"C:/test/file.bin"`）。  
  - `mode`：打开模式（字符串），决定文件的操作权限（读/写/追加）和类型（文本/二进制）。  
- **返回值**：成功返回`FILE*`指针（关联文件）；失败返回`NULL`（需检查，避免空指针操作）。  


##### 常用打开模式（`mode`）：
| 模式   | 含义（文本模式）                          | 模式   | 含义（二进制模式）                  |
|--------|-------------------------------------------|--------|-------------------------------------|
| `"r"`  | 只读打开，文件必须存在（否则失败）        | `"rb"` | 只读打开二进制文件，文件必须存在    |
| `"w"`  | 只写打开，文件不存在则创建，存在则清空    | `"wb"` | 只写打开二进制文件，同上            |
| `"a"`  | 追加写，文件不存在则创建，新数据加在末尾  | `"ab"` | 追加写二进制文件，同上              |
| `"r+"` | 读写打开，文件必须存在                    | `"rb+"`| 读写打开二进制文件，文件必须存在    |
| `"w+"` | 读写打开，文件不存在则创建，存在则清空    | `"wb+"`| 读写打开二进制文件，同上            |
| `"a+"` | 读写打开，新数据加在末尾（读可从头开始）  | `"ab+"`| 读写打开二进制文件，同上            |

**说明**：带`b`的模式用于二进制文件（如`"rb"`），不带`b`则为文本模式（默认）。


#### 2. 关闭文件：`fclose`函数
函数原型：  
```c
int fclose(FILE *stream);  // stream是fopen返回的文件指针
```  
- **作用**：关闭文件，刷新缓冲区（确保数据写入磁盘），释放文件相关资源。  
- **返回值**：成功返回`0`；失败返回`EOF`（`-1`）。  

**注意**：程序结束前必须关闭所有打开的文件，否则缓冲区数据可能未写入磁盘，导致数据丢失。  


#### 示例：打开与关闭文件
```c
#include <stdio.h>

int main() {
    // 打开文件（文本模式，只写）
    FILE *fp = fopen("test.txt", "w");
    if (fp == NULL) {  // 检查是否打开成功（关键！）
        printf("文件打开失败！\n");
        return 1;  // 打开失败，退出程序
    }

    // ... 此处进行文件读写操作 ...

    // 关闭文件
    if (fclose(fp) != 0) {
        printf("文件关闭失败！\n");
    } else {
        printf("文件关闭成功！\n");
    }
    return 0;
}
```


### 四、文本文件的读写
文本文件以字符为单位读写，常用函数如下：  


#### 1. 单个字符读写：`fputc` 与 `fgetc`
- **写字符**：`fputc(int c, FILE *stream)`  
  向文件写入一个字符（`c`的ASCII码），成功返回该字符，失败返回`EOF`。  

- **读字符**：`fgetc(FILE *stream)`  
  从文件读取一个字符，成功返回该字符的ASCII码，到文件末尾或失败返回`EOF`。  


**示例：用`fputc`和`fgetc`读写文本文件**  
```c
#include <stdio.h>

int main() {
    FILE *fp;
    char c;

    // 写文件：向test.txt写入"Hello"
    fp = fopen("test.txt", "w");
    if (fp == NULL) { return 1; }
    fputc('H', fp);
    fputc('e', fp);
    fputc('l', fp);
    fputc('l', fp);
    fputc('o', fp);
    fclose(fp);

    // 读文件：从test.txt读取字符并输出
    fp = fopen("test.txt", "r");
    if (fp == NULL) { return 1; }
    while ((c = fgetc(fp)) != EOF) {  // 读到EOF结束
        printf("%c", c);  // 输出：Hello
    }
    fclose(fp);
    return 0;
}
```


#### 2. 字符串读写：`fputs` 与 `fgets`
- **写字符串**：`fputs(const char *str, FILE *stream)`  
  向文件写入字符串`str`（不自动添加换行符），成功返回非负数，失败返回`EOF`。  

- **读字符串**：`fgets(char *str, int n, FILE *stream)`  
  从文件读取最多`n-1`个字符到`str`（预留1位存`'\0'`），遇换行符`'\n'`或文件尾停止，成功返回`str`，失败/到尾返回`NULL`。  


**示例：用`fputs`和`fgets`读写字符串**  
```c
#include <stdio.h>

int main() {
    FILE *fp;
    char buf[100];

    // 写文件：写入两行字符串
    fp = fopen("test.txt", "w");
    if (fp == NULL) { return 1; }
    fputs("Hello World\n", fp);  // 手动添加换行符
    fputs("C File I/O", fp);
    fclose(fp);

    // 读文件：逐行读取并输出
    fp = fopen("test.txt", "r");
    if (fp == NULL) { return 1; }
    while (fgets(buf, 100, fp) != NULL) {  // 读一行
        printf("%s", buf);  // 输出：Hello World\nC File I/O
    }
    fclose(fp);
    return 0;
}
```


#### 3. 格式化读写：`fprintf` 与 `fscanf`
与`printf`/`scanf`用法类似，但操作对象是文件（通过文件指针指定）。  

- **格式化写**：`fprintf(FILE *stream, const char *format, ...)`  
  按格式向文件写入数据（如整数、字符串等）。  

- **格式化读**：`fscanf(FILE *stream, const char *format, ...)`  
  按格式从文件读取数据到变量（需传变量地址）。  


**示例：用`fprintf`和`fscanf`读写结构化数据**  
```c
#include <stdio.h>

int main() {
    FILE *fp;
    int id;
    char name[20];
    float score;

    // 写文件：写入学生信息（格式化）
    fp = fopen("student.txt", "w");
    if (fp == NULL) { return 1; }
    fprintf(fp, "%d %s %.1f\n", 1001, "Alice", 95.5);  // 格式：id 姓名 成绩
    fprintf(fp, "%d %s %.1f\n", 1002, "Bob", 88.0);
    fclose(fp);

    // 读文件：读取学生信息（格式化）
    fp = fopen("student.txt", "r");
    if (fp == NULL) { return 1; }
    while (fscanf(fp, "%d %s %f", &id, name, &score) != EOF) {
        printf("学号：%d，姓名：%s，成绩：%.1f\n", id, name, score);
    }
    fclose(fp);
    return 0;
}
```

输出：  
```
学号：1001，姓名：Alice，成绩：95.5
学号：1002，姓名：Bob，成绩：88.0
```


### 五、二进制文件的读写
二进制文件以字节为单位读写（与内存数据格式一致），适合存储结构体、图片等非文本数据，核心函数是`fread`和`fwrite`。  


#### 1. 块写入：`fwrite`函数
函数原型：  
```c
size_t fwrite(const void *ptr, size_t size, size_t nmemb, FILE *stream);
```  
- **参数**：  
  - `ptr`：待写入数据的内存地址（如结构体变量地址）。  
  - `size`：每个数据块的字节数（如`sizeof(int)`、`sizeof(Student)`）。  
  - `nmemb`：要写入的数据块数量。  
  - `stream`：文件指针。  
- **返回值**：成功写入的**数据块数量**（若小于`nmemb`，表示写入失败）。  


#### 2. 块读取：`fread`函数
函数原型：  
```c
size_t fread(void *ptr, size_t size, size_t nmemb, FILE *stream);
```  
- **参数**：与`fwrite`类似，`ptr`是接收数据的内存地址。  
- **返回值**：成功读取的**数据块数量**（到文件尾时可能小于`nmemb`）。  


#### 示例：用`fread`和`fwrite`读写结构体（二进制文件）
```c
#include <stdio.h>

// 定义学生结构体
typedef struct {
    int id;
    char name[20];
    float score;
} Student;

int main() {
    FILE *fp;
    Student s1 = {1001, "Alice", 95.5};
    Student s2;  // 用于接收读取的数据

    // 写二进制文件：写入s1
    fp = fopen("student.bin", "wb");  // 二进制模式
    if (fp == NULL) { return 1; }
    fwrite(&s1, sizeof(Student), 1, fp);  // 写1个Student大小的块
    fclose(fp);

    // 读二进制文件：读取到s2
    fp = fopen("student.bin", "rb");
    if (fp == NULL) { return 1; }
    fread(&s2, sizeof(Student), 1, fp);  // 读1个Student大小的块
    fclose(fp);

    // 输出读取的内容
    printf("学号：%d，姓名：%s，成绩：%.1f\n", s2.id, s2.name, s2.score);
    // 输出：学号：1001，姓名：Alice，成绩：95.5
    return 0;
}
```

**优势**：二进制读写无需格式转换，速度快，且能完整保存结构体的内存布局（文本模式可能因格式转换丢失信息）。


### 六、文件定位（随机访问）
默认情况下，文件读写按顺序进行（从文件头到尾），但可通过**文件指针位置**（记录下一次读写的位置）实现随机访问，核心函数：  


#### 1. `fseek`：移动文件指针
函数原型：  
```c
int fseek(FILE *stream, long offset, int whence);
```  
- **作用**：将文件指针移动到指定位置。  
- **参数**：  
  - `offset`：偏移量（字节数，正数向后移，负数向前移）。  
  - `whence`：基准位置：  
    - `SEEK_SET`：从文件开头（0字节处）开始计算。  
    - `SEEK_CUR`：从当前指针位置开始计算。  
    - `SEEK_END`：从文件末尾开始计算。  
- **返回值**：成功返回`0`；失败返回非`0`。  


#### 2. `ftell`：获取当前指针位置
函数原型：  
```c
long ftell(FILE *stream);
```  
- **作用**：返回当前文件指针相对于文件开头的偏移量（字节数）。  
- **返回值**：成功返回偏移量；失败返回`-1L`。  


#### 3. `rewind`：重置指针到文件开头
函数原型：  
```c
void rewind(FILE *stream);
```  
- **作用**：等价于`fseek(stream, 0, SEEK_SET)`，将指针移到文件开头。  


#### 示例：随机访问文件
```c
#include <stdio.h>

int main() {
    FILE *fp = fopen("test.txt", "w+");  // 读写模式
    if (fp == NULL) { return 1; }

    // 写入数据："0123456789"
    fputs("0123456789", fp);

    // 移动指针到第3个字符（索引2，0-based）
    fseek(fp, 2, SEEK_SET);
    char c = fgetc(fp);
    printf("第3个字符：%c\n", c);  // 输出：2

    // 移动指针到当前位置后2个字节
    fseek(fp, 2, SEEK_CUR);
    c = fgetc(fp);
    printf("当前位置后2个字符：%c\n", c);  // 输出：4

    // 移动指针到文件末尾前3个字节
    fseek(fp, -3, SEEK_END);
    c = fgetc(fp);
    printf("末尾前3个字符：%c\n", c);  // 输出：7

    fclose(fp);
    return 0;
}
```


### 七、文件操作的错误处理
文件操作可能因各种原因失败（如文件不存在、权限不足），必须通过返回值判断并处理：  
1. `fopen`返回`NULL`：用`perror`函数打印具体错误信息（需包含`stdio.h`）。  
   ```c
   #include <stdio.h>
   int main() {
       FILE *fp = fopen("nonexistent.txt", "r");
       if (fp == NULL) {
           perror("文件打开失败");  // 输出：文件打开失败: No such file or directory
           return 1;
       }
       fclose(fp);
       return 0;
   }
   ```  
2. 读写函数（`fread`、`fwrite`等）返回值异常：判断是否达到文件尾或操作失败。  


### 总结
C语言文件读写的核心是通过`FILE*`指针操作，流程为“打开→读写→关闭”：  
- **文本文件**：用`fputc`/`fgetc`（字符）、`fputs`/`fgets`（字符串）、`fprintf`/`fscanf`（格式化）。  
- **二进制文件**：用`fread`/`fwrite`（块数据），适合存储结构体等二进制数据。  
- **随机访问**：通过`fseek`、`ftell`、`rewind`移动文件指针。  

关键注意事项：始终检查`fopen`的返回值（避免空指针），操作完成后用`fclose`关闭文件（确保数据写入），并通过返回值处理错误。
