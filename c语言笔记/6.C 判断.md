在C语言中，**判断结构**用于根据条件的真假执行不同的代码分支，是控制程序流程的核心机制之一。C语言提供了两种主要的判断结构：`if`语句（用于条件判断）和`switch`语句（用于多分支选择）。


### 一、`if` 语句（条件判断）
`if`语句根据指定条件的真假（非0为真，0为假）决定是否执行某段代码。其基本形式和扩展形式如下：


#### 1. 基本`if`语句（单分支）
**语法**：  
```c
if (条件表达式) {
    // 条件为真（非0）时执行的语句块
}
```

**说明**：  
- `条件表达式`：可以是任何返回整数（或可隐式转换为整数）的表达式（如关系表达式、逻辑表达式、变量等）。若表达式结果为**非0**（真），则执行`{}`中的语句块；若为**0**（假），则跳过语句块。  
- 若语句块中只有**一条语句**，`{}`可以省略（但建议保留，避免逻辑错误）。  


**示例**：判断一个数是否为正数  
```c
#include <stdio.h>

int main() {
    int num = 5;
    if (num > 0) {  // 条件：num大于0（真）
        printf("%d是正数\n", num);  // 执行此语句
    }
    return 0;
}
// 输出：5是正数
```


#### 2. `if-else`语句（双分支）
当需要在“条件真”和“条件假”时执行不同代码时，使用`if-else`。

**语法**：  
```c
if (条件表达式) {
    // 条件为真时执行的语句块
} else {
    // 条件为假时执行的语句块
}
```


**示例**：判断一个数是正数还是非正数  
```c
#include <stdio.h>

int main() {
    int num = -3;
    if (num > 0) {
        printf("%d是正数\n", num);
    } else {
        printf("%d不是正数\n", num);  // 条件为假，执行此语句
    }
    return 0;
}
// 输出：-3不是正数
```


#### 3. `if-else if-else`语句（多分支）
当需要判断多个条件（按顺序检查）时，使用`if-else if-else`，适用于“多选一”的场景。

**语法**：  
```c
if (条件1) {
    // 条件1为真时执行
} else if (条件2) {
    // 条件1为假，且条件2为真时执行
} else if (条件3) {
    // 前两个条件为假，且条件3为真时执行
} else {
    // 所有条件都为假时执行（可省略）
}
```

**说明**：  
- 程序会**依次检查条件**，一旦某个条件为真，执行对应语句块后，跳过后续所有`else if`和`else`。  
- 若所有条件都为假，且存在`else`，则执行`else`块；若没有`else`，则什么都不做。  


**示例**：根据分数评级（0-100分）  
```c
#include <stdio.h>

int main() {
    int score = 85;
    if (score >= 90) {
        printf("评级：A\n");
    } else if (score >= 80) {  // 前面条件（>=90）为假，检查此条件
        printf("评级：B\n");  // 85>=80为真，执行此语句
    } else if (score >= 60) {
        printf("评级：C\n");
    } else {
        printf("评级：D\n");
    }
    return 0;
}
// 输出：评级：B
```

**注意**：条件顺序很重要。例如，若将`score >=60`放在`score >=80`前面，会导致80-89分的情况错误匹配到`>=60`。


### 二、`switch`语句（多分支选择）
`switch`语句用于根据一个**整数/字符/枚举类型**的表达式，从多个预定义分支中选择一个执行，比`if-else if`更适合“判断值是否等于某个常量”的场景。


#### 语法
```c
switch (表达式) {  // 表达式必须是整数、字符或枚举类型
    case 常量1:
        // 当表达式的值等于常量1时执行的语句
        break;  // 跳出switch块（关键，否则会继续执行下一个case）
    case 常量2:
        // 当表达式的值等于常量2时执行的语句
        break;
    // ... 更多case
    default:  // 可选，所有case都不匹配时执行
        // 语句
        break;
}
```


#### 关键说明
1. **`switch`后的表达式**：必须是整数类型（`int`、`char`、`short`等）或枚举类型（浮点数、字符串等不允许）。  
2. **`case`后的常量**：必须是**整数常量**（如`10`、`'a'`、枚举成员），且不能重复。  
3. **`break`的作用**：执行完当前`case`的语句后，用`break`跳出`switch`块；若省略`break`，程序会继续执行下一个`case`（称为“穿透”），直到遇到`break`或`switch`结束。  
4. **`default`**：可选分支，当表达式的值与所有`case`的常量都不匹配时执行，通常放在最后（可省略）。  


#### 示例：根据数字判断星期几
```c
#include <stdio.h>

int main() {
    int day = 3;
    switch (day) {
        case 1:
            printf("星期一\n");
            break;  // 执行后跳出
        case 2:
            printf("星期二\n");
            break;
        case 3:
            printf("星期三\n");  // day=3，执行此语句
            break;
        case 4:
            printf("星期四\n");
            break;
        default:  // 若day为5-7或其他值，执行这里
            printf("其他星期\n");
            break;
    }
    return 0;
}
// 输出：星期三
```


#### 利用“穿透”特性的场景
有时可故意省略`break`，实现多个`case`共享同一段代码：  
```c
// 判断输入的字符是否为元音字母（a,e,i,o,u,A,E,I,O,U）
#include <stdio.h>

int main() {
    char c = 'E';
    switch (c) {
        case 'a':
        case 'A':
        case 'e':
        case 'E':  // c='E'，匹配此case
        case 'i':
        case 'I':
        case 'o':
        case 'O':
        case 'u':
        case 'U':
            printf("%c是元音字母\n", c);  // 执行此语句
            break;
        default:
            printf("%c不是元音字母\n", c);
            break;
    }
    return 0;
}
// 输出：E是元音字母
```


### 三、判断中的常见问题
1. **`if`条件中误用`=`（赋值）代替`==`（比较）**：  
   错误：`if (a = 5)` → 实际是将5赋值给a，表达式结果为5（非0，即“真”），导致条件永远成立。  
   正确：`if (a == 5)` → 判断a是否等于5。  

2. **`switch`表达式类型错误**：  
   错误：`switch(3.14)` → 浮点数不能作为`switch`的表达式。  

3. **`case`后使用变量**：  
   错误：`case a:` → `case`后必须是常量，不能是变量。  

4. **遗漏`break`导致穿透**：  
   若无意省略`break`，会导致多个`case`被执行，需特别注意。  


### 总结
- `if`语句适合**范围判断**或**复杂条件**（如`score > 80 && score <= 90`）。  
- `switch`语句适合**离散值判断**（如等于1、2、3等），结构更清晰。  

根据具体场景选择合适的判断结构，可使代码更易读、逻辑更清晰。
