在C语言中，**字符串（String）** 是由多个字符组成的序列，但其本质是**以空字符`'\0'`（ASCII码为0）结尾的字符数组**。`'\0'`是字符串的“结束标志”，用于告知字符串处理函数（如`printf`、`strlen`）哪里是字符串的末尾。

C语言没有专门的“字符串类型”，所有字符串操作都通过字符数组和指针实现，这一点与其他高级语言（如Python、Java）不同。


### 一、字符串的定义与初始化
字符串的核心是“字符数组 + `'\0'`结尾”，初始化方式主要有以下几种：


#### 1. 用字符串常量初始化（推荐）
字符串常量用双引号`""`括起，编译器会自动在末尾添加`'\0'`。  
```c
// 方式1：省略数组长度，编译器自动计算（包含'\0'）
char str1[] = "hello";  // 长度为6：'h','e','l','l','o','\0'

// 方式2：指定数组长度（需确保能容纳所有字符+'\0'）
char str2[6] = "hello";  // 正确：5个字符 + '\0'，共6位
char str3[5] = "hello";  // 警告：长度不足（"hello"需6位），'\0'可能被截断
```


#### 2. 用字符列表初始化（需手动添加`'\0'`）
若用单引号分隔的字符列表初始化，必须手动添加`'\0'`，否则只是普通字符数组（不是字符串）。  
```c
// 正确：手动添加'\0'，是字符串
char str4[] = {'h', 'i', '\0'};  // 长度3：'h','i','\0'

// 错误：无'\0'，是普通字符数组（不是字符串）
char str5[] = {'h', 'i'};  // 不是字符串，字符串处理函数会出错
```


#### 关键区别：字符串 vs 字符数组
- **字符串**：以`'\0'`结尾的字符数组，可被所有字符串函数（如`strlen`、`printf("%s")`）正确处理。  
- **普通字符数组**：无`'\0'`，字符串函数会因找不到结束标志而越界访问（读取垃圾值）。  


### 二、字符串的输入与输出
C语言通过标准库函数实现字符串的输入输出，核心函数如下：


#### 1. 输出字符串：`printf("%s", str)` 或 `puts(str)`
- `printf("%s", str)`：按`%s`格式输出字符串，直到`'\0'`为止（不输出`'\0'`）。  
- `puts(str)`：输出字符串后自动换行（同样以`'\0'`为结束标志）。  

**示例**：  
```c
#include <stdio.h>

int main() {
    char str[] = "hello world";
    printf("用printf输出：%s\n", str);  // 输出：用printf输出：hello world
    puts("用puts输出：hello");          // 输出：用puts输出：hello（自动换行）
    return 0;
}
```


#### 2. 输入字符串：`scanf("%s", str)` 或 `fgets(str, len, stdin)`
- **`scanf("%s", str)`**：  
  - 从键盘读取字符串，遇到**空格、回车、制表符**时停止，自动在末尾添加`'\0'`。  
  - 缺点：无法读取带空格的字符串（如“hello world”会被截断为“hello”），且不检查缓冲区大小，可能导致溢出。  

- **`fgets(str, max_len, stdin)`**（推荐）：  
  - 从指定输入流（`stdin`表示键盘）读取字符串，最多读取`max_len-1`个字符（预留1位存`'\0'`），遇到回车时停止（会保留回车符`'\n'`）。  
  - 优点：可读取带空格的字符串，且限制最大长度，避免溢出。  

**示例**：  
```c
#include <stdio.h>

int main() {
    char str1[20], str2[20];

    printf("用scanf输入（不含空格）：");
    scanf("%s", str1);  // 输入：hello world → 只读取"hello"
    printf("str1: %s\n", str1);  // 输出：str1: hello

    // 注意：scanf读取后，缓冲区可能残留回车，需清空（否则影响fgets）
    getchar();  // 吸收scanf残留的回车

    printf("用fgets输入（可含空格）：");
    fgets(str2, 20, stdin);  // 输入：hello world → 读取完整字符串（含空格）
    printf("str2: %s", str2);  // 输出：str2: hello world（含回车）
    return 0;
}
```


### 三、字符串处理函数（`string.h`库）
C标准库`string.h`提供了一系列字符串操作函数，需包含头文件`#include <string.h>`才能使用。以下是最常用的函数：


#### 1. 求字符串长度：`strlen(str)`
返回字符串中**有效字符的个数**（不包括`'\0'`）。  

```c
#include <stdio.h>
#include <string.h>

int main() {
    char str[] = "hello";
    printf("长度：%zu\n", strlen(str));  // 输出：5（'h','e','l','l','o'共5个）
    return 0;
}
```

**注意**：`strlen`的返回值类型是`size_t`（无符号整数），打印时用`%zu`格式。


#### 2. 复制字符串：`strcpy(dest, src)` 与 `strncpy(dest, src, n)`
- `strcpy(dest, src)`：将`src`字符串复制到`dest`缓冲区（包括`'\0'`）。  
  **风险**：若`dest`缓冲区大小小于`src`长度，会导致溢出（覆盖其他内存）。  

- `strncpy(dest, src, n)`：安全版本，最多复制`n`个字符到`dest`，若`src`长度小于`n`，剩余位置用`'\0'`填充。  

```c
#include <stdio.h>
#include <string.h>

int main() {
    char src[] = "hello";
    char dest[20];

    strcpy(dest, src);  // 复制src到dest → dest = "hello"
    printf("strcpy后：%s\n", dest);  // 输出：hello

    char dest2[4];
    strncpy(dest2, src, 3);  // 最多复制3个字符 → dest2 = {'h','e','l'}
    dest2[3] = '\0';  // 手动添加结束标志（否则可能不是字符串）
    printf("strncpy后：%s\n", dest2);  // 输出：hel
    return 0;
}
```


#### 3. 拼接字符串：`strcat(dest, src)` 与 `strncat(dest, src, n)`
- `strcat(dest, src)`：将`src`字符串拼接在`dest`末尾（覆盖`dest`原有的`'\0'`，并在新末尾添加`'\0'`）。  
  **风险**：`dest`需足够大，否则溢出。  

- `strncat(dest, src, n)`：安全版本，最多拼接`n`个字符。  

```c
#include <stdio.h>
#include <string.h>

int main() {
    char dest[20] = "hello";
    char src[] = " world";

    strcat(dest, src);  // 拼接 → dest = "hello world"
    printf("strcat后：%s\n", dest);  // 输出：hello world
    return 0;
}
```


#### 4. 比较字符串：`strcmp(s1, s2)` 与 `strncmp(s1, s2, n)`
- `strcmp(s1, s2)`：按ASCII码值逐个比较字符，直到不同或遇到`'\0'`。  
  - 返回值：`0`（s1 == s2）；`>0`（s1 > s2）；`<0`（s1 < s2）。  

- `strncmp(s1, s2, n)`：比较前`n`个字符。  

```c
#include <stdio.h>
#include <string.h>

int main() {
    char s1[] = "apple", s2[] = "app", s3[] = "banana";

    printf("strcmp(s1, s2) = %d\n", strcmp(s1, s2));  // s1长于s2 → 1（>0）
    printf("strcmp(s1, s3) = %d\n", strcmp(s1, s3));  // 'a' < 'b' → -1（<0）
    printf("strcmp(s1, s1) = %d\n", strcmp(s1, s1));  // 相等 → 0
    return 0;
}
```


### 四、字符串的遍历与修改
字符串本质是字符数组，可通过下标遍历或修改字符（注意不能修改字符串常量）。


#### 1. 遍历字符串（直到`'\0'`）
```c
#include <stdio.h>

int main() {
    char str[] = "hello";
    for (int i = 0; str[i] != '\0'; i++) {  // 循环到'\0'结束
        printf("str[%d] = %c\n", i, str[i]);
    }
    return 0;
}
```


#### 2. 修改字符串（字符数组可修改，字符串常量不可修改）
- **字符数组**：存储在栈区，可修改字符。  
- **字符串常量**：存储在只读内存区，通过指针指向时不可修改（否则程序崩溃）。  

```c
#include <stdio.h>

int main() {
    // 字符数组：可修改
    char str1[] = "hello";
    str1[0] = 'H';  // 合法 → str1 = "Hello"
    printf("%s\n", str1);  // 输出：Hello

    // 字符串常量：不可修改（指针指向只读内存）
    char *str2 = "hello";
    // str2[0] = 'H';  // 错误：修改只读内存，程序崩溃
    return 0;
}
```


### 五、常见错误与注意事项
1. **忘记添加`'\0'`**：用字符列表初始化时，若漏写`'\0'`，`strlen`、`printf`等函数会读取到垃圾值（越界）。  
2. **缓冲区溢出**：`strcpy`、`strcat`不检查目标缓冲区大小，需确保`dest`足够大，或改用`strncpy`、`strncat`。  
3. **混淆`strlen`与`sizeof`**：  
   - `strlen(str)`：有效字符数（不含`'\0'`）。  
   - `sizeof(str)`：数组总字节数（含`'\0'`和未使用的空间）。  
   示例：`char str[] = "abc";` → `strlen(str)=3`，`sizeof(str)=4`。  
4. **修改字符串常量**：字符串常量（如`"hello"`）是只读的，不能通过指针修改。  


### 总结
C语言字符串是以`'\0'`结尾的字符数组，依赖`string.h`库函数进行操作。核心要点：  
- 字符串必须以`'\0'`结尾，否则无法被正确处理。  
- 输入用`fgets`更安全（可读空格、防溢出），输出用`printf("%s")`或`puts`。  
- 常用函数：`strlen`（长度）、`strcpy`（复制）、`strcat`（拼接）、`strcmp`（比较），注意安全版本的使用。  
- 字符数组可修改，字符串常量不可修改。  

掌握字符串操作是C语言处理文本数据的基础，需特别注意边界和溢出问题。
