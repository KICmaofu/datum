在C语言中，**常量**是指程序运行过程中其值不能被修改的量。与变量（值可修改）不同，常量一旦定义，其值在整个程序生命周期中保持不变。


### C语言常量的分类
C语言中的常量主要分为以下几类，每种类型有特定的表示形式和使用场景：


#### 1. 整数常量（Integer Constants）
整数常量是没有小数部分的数值，可表示为十进制、八进制或十六进制形式。

- **十进制**：最常用的形式，直接写数字（不能以0开头，除了0本身）。  
  例：`123`、`-456`、`0`  

- **八进制**：以数字`0`开头，后续数字为0-7。  
  例：`012`（对应十进制的10）、`077`（对应十进制的63）  

- **十六进制**：以`0x`或`0X`开头，后续数字为0-9、a-f（或A-F，不区分大小写）。  
  例：`0x1A`（对应十进制的26）、`0Xf`（对应十进制的15）  


**整数常量的后缀**（用于指定类型）：  
- `u`或`U`：表示无符号整数（unsigned）。  
  例：`123u`（无符号整数）、`0x20U`  

- `l`或`L`：表示长整数（long）。  
  例：`456L`（长整数）、`-077l`  

- 组合使用：`ul`或`UL`（无符号长整数）。  
  例：`100UL`  


#### 2. 浮点常量（Floating-point Constants）
浮点常量是带小数部分的数值，或用指数形式表示的数值（即科学计数法）。

- **十进制形式**：包含小数点（`.`），整数部分和小数部分可省略其一，但不能都省略。  
  例：`3.14`、`.5`（等价于0.5）、`5.`（等价于5.0）  

- **指数形式**：用`e`或`E`表示“乘以10的n次方”，`e`前后必须有数字，`e`后必须是整数（可正可负）。  
  例：`1.2e3`（等价于1.2×10³=1200）、`3.4E-2`（等价于3.4×10⁻²=0.034）  


**浮点常量的后缀**（用于指定类型）：  
- 默认是`double`类型（双精度浮点）。  
- `f`或`F`：表示`float`类型（单精度浮点）。  
  例：`3.14f`（单精度）、`0.5F`  


#### 3. 字符常量（Character Constants）
字符常量是单个字符，用单引号`' '`括起来，在内存中存储的是该字符的ASCII码值。

- **普通字符**：直接写字符本身。  
  例：`'a'`（ASCII码97）、`'5'`（ASCII码53）、`'+'`（ASCII码43）  

- **转义字符**：用反斜杠`\`开头的特殊字符（表示无法直接输入的字符，如换行、制表符等）。  
  常见转义字符：  
  | 转义字符 | 含义 | ASCII码 |  
  |----------|------|---------|  
  | `\n`     | 换行 | 10      |  
  | `\t`     | 水平制表符（Tab） | 9 |  
  | `\\`     | 反斜杠本身 | 92 |  
  | `\'`     | 单引号 | 39 |  
  | `\"`     | 双引号 | 34 |  
  | `\0`     | 空字符（字符串结束标志） | 0 |  
  | `\ddd`   | 八进制数表示的字符（ddd为1-3位八进制数） | 例：`\141`等价于'a'（八进制141=十进制97） |  
  | `\xhh`   | 十六进制数表示的字符（hh为1-2位十六进制数） | 例：`\x61`等价于'a'（十六进制61=十进制97） |  


#### 4. 字符串常量（String Literals）
字符串常量是由多个字符组成的序列，用双引号`" "`括起来。编译器会自动在字符串末尾添加一个`\0`作为结束标志（因此字符串的实际长度比可见字符数多1）。

例：  
- `"hello"`：包含字符`h、e、l、l、o、\0`（共6个字符）  
- `"123"`：包含`1、2、3、\0`（共4个字符）  
- `""`：空字符串（仅包含`\0`）  


#### 5. 宏常量（Macro Constants）
用`#define`预处理指令定义的常量，本质是“文本替换”（无类型，预处理阶段会被替换为对应值）。

格式：  
```c
#define 常量名 常量值  // 注意：末尾没有分号
```

例：  
```c
#define PI 3.14159  // 定义宏常量PI，值为3.14159
#define MAX 100     // 定义宏常量MAX，值为100
```

使用时，`PI`会被直接替换为`3.14159`，`MAX`替换为`100`。  
**注意**：宏常量没有类型检查，替换时会原样代入，需避免因格式错误导致的问题（如末尾加了分号）。


#### 6. 枚举常量（Enumeration Constants）
用`enum`关键字定义的枚举类型中的成员，其值为整数（默认从0开始，可手动指定）。

格式：  
```c
enum 枚举名 { 常量1, 常量2, ..., 常量n };
```

例：  
```c
// 定义枚举类型Week，成员为星期几（枚举常量）
enum Week {
    Mon,  // 默认值0
    Tue,  // 默认值1（前一个+1）
    Wed,  // 默认值2
    Thu=5,  // 手动指定为5
    Fri   // 自动为6（前一个+1）
};
```

枚举常量的值一旦定义，不能被修改。


### 常量与变量的核心区别
- 常量：值不可修改，定义后无法重新赋值。  
- 变量：值可修改，通过赋值语句更新（如`int a=5; a=10;`是合法的）。


通过理解以上常量类型，可在C程序中更规范地使用固定值，提高代码的可读性和安全性。
