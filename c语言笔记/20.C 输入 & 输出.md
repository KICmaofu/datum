在C语言中，输入（Input）和输出（Output）是程序与用户交互的核心方式，主要通过标准输入输出库（`stdio.h`）提供的函数实现。C语言本身没有内置的输入输出语句，所有操作都依赖库函数，最常用的包括`printf`（输出）、`scanf`（输入）、`putchar`、`getchar`、`puts`、`fgets`等。


### 一、标准输出函数（Output）
标准输出通常指**显示器**，常用函数有`printf`、`putchar`、`puts`。


#### 1. `printf`：格式化输出（最常用）
`printf`用于按指定格式输出数据到屏幕，函数原型：  
```c
int printf(const char *format, ...);
```  
- **`format`**：格式控制字符串，包含普通字符（直接输出）和**格式说明符**（指定输出数据的类型和格式）。  
- **`...`**：可变参数列表，即要输出的变量或值（数量与格式说明符一一对应）。  


##### （1）常用格式说明符
格式说明符以`%`开头，用于指定数据类型，常见如下：  

| 格式符 | 含义                  | 示例                          |
|--------|-----------------------|-------------------------------|
| `%d`   | 输出十进制整数        | `printf("%d", 100);` → `100`  |
| `%c`   | 输出单个字符          | `printf("%c", 'A');` → `A`    |
| `%s`   | 输出字符串（以`'\0'`结尾） | `printf("%s", "hello");` → `hello` |
| `%f`   | 输出浮点数（默认6位小数） | `printf("%f", 3.14);` → `3.140000` |
| `%lf`  | 输出双精度浮点数（`double`） | `printf("%lf", 3.1415);` → `3.141500` |
| `%x`   | 输出十六进制整数（小写） | `printf("%x", 15);` → `f`     |
| `%X`   | 输出十六进制整数（大写） | `printf("%X", 15);` → `F`     |
| `%%`   | 输出`%`本身           | `printf("%%");` → `%`         |  


##### （2）格式修饰符（控制输出格式）
可在`%`和格式符之间添加修饰符，控制宽度、精度、对齐方式等：  

- **宽度**：指定输出的最小字符数，不足则补空格（默认右对齐）。  
  ```c
  printf("|%5d|", 123);  // 宽度5，右对齐 → |  123|（补2个空格）
  printf("|%-5d|", 123); // 宽度5，左对齐（-表示左对齐）→ |123  |
  ```  

- **精度**：对浮点数指定小数位数，对字符串指定最大输出长度。  
  ```c
  printf("%.2f", 3.1415);  // 保留2位小数 → 3.14
  printf("%.3s", "hello"); // 最多输出3个字符 → hel
  ```  


##### （3）示例：`printf`的基本用法
```c
#include <stdio.h>

int main() {
    int a = 10;
    char c = 'B';
    float f = 3.1415f;
    char str[] = "world";

    // 混合输出普通字符和变量
    printf("整数：%d，字符：%c\n", a, c);  // 整数：10，字符：B
    printf("浮点数：%.2f，字符串：%s\n", f, str);  // 浮点数：3.14，字符串：world
    return 0;
}
```


#### 2. `putchar`：输出单个字符
`putchar`用于输出一个字符，函数原型：  
```c
int putchar(int c);  // 参数是字符的ASCII码（char会自动转换为int）
```  

**示例**：  
```c
#include <stdio.h>

int main() {
    putchar('A');  // 输出字符'A'
    putchar(10);   // 10是'\n'的ASCII码，输出换行
    putchar('B');  // 输出'B'
    // 输出结果：A\nB → 即A在下一行B的前面
    return 0;
}
```


#### 3. `puts`：输出字符串并自动换行
`puts`用于输出字符串（以`'\0'`结尾），并自动在末尾添加换行符`'\n'`，函数原型：  
```c
int puts(const char *str);
```  

**示例**：  
```c
#include <stdio.h>

int main() {
    puts("hello");    // 输出"hello"并换行
    puts("world");    // 输出"world"并换行
    // 输出结果：
    // hello
    // world
    return 0;
}
```

**注意**：`puts`只能输出字符串，且会自动换行，适合简单的字符串输出。


### 二、标准输入函数（Input）
标准输入通常指**键盘**，常用函数有`scanf`、`getchar`、`fgets`。


#### 1. `scanf`：格式化输入（最常用）
`scanf`用于按指定格式从键盘读取数据，存储到变量中，函数原型：  
```c
int scanf(const char *format, ...);
```  
- **`format`**：格式控制字符串，包含格式说明符（与`printf`类似）。  
- **`...`**：可变参数列表，必须是**变量的地址**（用`&`运算符获取，字符串数组除外）。  
- **返回值**：成功读取并赋值的变量个数（用于判断输入是否有效）。  


##### （1）常用格式说明符（与`printf`对应）
| 格式符 | 含义                  | 示例                          |
|--------|-----------------------|-------------------------------|
| `%d`   | 读取十进制整数        | `int a; scanf("%d", &a);`     |
| `%c`   | 读取单个字符          | `char c; scanf("%c", &c);`    |
| `%s`   | 读取字符串（遇空格/回车停止） | `char str[20]; scanf("%s", str);`（str是数组名，即地址，无需&） |
| `%f`   | 读取浮点数到`float`   | `float f; scanf("%f", &f);`   |
| `%lf`  | 读取浮点数到`double`  | `double d; scanf("%lf", &d);` |  


##### （2）输入注意事项
- **必须传变量地址**：`scanf`需要修改变量的值，因此必须传递变量的地址（`&变量名`），字符串数组除外（数组名本身就是地址）。  
  ```c
  int a;
  scanf("%d", &a);  // 正确：&a是a的地址
  // scanf("%d", a);  // 错误：传递的是a的值，不是地址
  ```  

- **跳过空白字符**：`scanf`读取非`%c`格式时，会自动跳过空格、回车、制表符（`\t`）等空白字符。  
  例如，输入`  10 20`（10前有空格，10和20间有空格），`scanf("%d%d", &a, &b)`会正确读取10和20。  

- **`%c`不跳过空白**：`%c`会读取任何字符（包括空格、回车），需注意缓冲区残留的空白。  
  ```c
  char c1, c2;
  scanf("%c", &c1);  // 输入'A'后回车，c1='A'，缓冲区残留'\n'
  scanf("%c", &c2);  // 直接读取'\n'，而非下一个输入的字符
  ```  
  解决方法：用`getchar()`吸收残留的空白，或在`%c`前加空格（`scanf(" %c", &c2)`，空格会跳过空白）。  

- **字符串输入截断**：`%s`读取字符串时，遇空格/回车停止，且不会检查目标数组大小，可能导致缓冲区溢出（不安全）。  


##### （3）示例：`scanf`的基本用法
```c
#include <stdio.h>

int main() {
    int a;
    float f;
    char str[20];

    printf("请输入一个整数、一个浮点数和一个字符串：");
    int count = scanf("%d%f%s", &a, &f, str);  // 读取数据

    if (count == 3) {  // 检查是否成功读取3个数据
        printf("你输入的是：%d, %.2f, %s\n", a, f, str);
    } else {
        printf("输入格式错误！\n");
    }
    return 0;
}
```

输入示例：`10 3.14 hello` → 输出：`你输入的是：10, 3.14, hello`  


#### 2. `getchar`：读取单个字符
`getchar`从键盘读取一个字符（包括空格、回车），函数原型：  
```c
int getchar(void);  // 返回读取的字符的ASCII码（失败返回EOF）
```  

**示例**：读取字符直到输入回车  
```c
#include <stdio.h>

int main() {
    char c;
    printf("请输入字符（按回车结束）：");
    while ((c = getchar()) != '\n') {  // 读取到回车则停止
        printf("你输入了：%c\n", c);
    }
    return 0;
}
```

**用途**：常用于吸收`scanf`残留的换行符，或逐个读取字符。


#### 3. `fgets`：安全读取字符串（推荐）
`fgets`从指定输入流（键盘用`stdin`）读取字符串，可指定最大长度，避免溢出，函数原型：  
```c
char *fgets(char *str, int n, FILE *stream);
```  
- **`str`**：存储字符串的缓冲区（字符数组）。  
- **`n`**：最大读取字符数（实际最多读`n-1`个，预留1位存`'\0'`）。  
- **`stream`**：输入流（`stdin`表示键盘）。  
- **特点**：会读取换行符`'\n'`并包含在字符串中，遇回车或达到最大长度时停止。  


**示例**：用`fgets`读取带空格的字符串  
```c
#include <stdio.h>
#include <string.h>  // 用于strcspn函数（移除换行符）

int main() {
    char str[20];
    printf("请输入字符串（可含空格）：");
    fgets(str, 20, stdin);  // 最多读19个字符，存到str

    // 移除fgets读取的换行符（若存在）
    str[strcspn(str, "\n")] = '\0';  // strcspn找'\n'的位置，替换为'\0'

    printf("你输入的是：%s\n", str);
    return 0;
}
```

输入示例：`hello world` → 输出：`你输入的是：hello world`  

**优势**：比`scanf("%s")`更安全（限制长度），且能读取带空格的字符串，推荐优先使用。


### 三、输入输出缓冲区（Buffer）
C语言的输入输出默认使用**缓冲区**（临时存储数据的内存区域），而非立即读写设备，目的是提高效率：  
- **输出缓冲区**：`printf`、`puts`等输出的数据会先存到缓冲区，满足以下条件之一时才刷新到屏幕：  
  - 缓冲区满；  
  - 遇到换行符`'\n'`；  
  - 程序结束或调用`fflush(stdout)`手动刷新。  

- **输入缓冲区**：键盘输入的字符会先存到输入缓冲区，`scanf`、`getchar`等函数从缓冲区读取数据，若缓冲区有残留数据（如未读完的字符、回车），会影响下一次输入。  


#### 示例：缓冲区影响输入
```c
#include <stdio.h>

int main() {
    int a;
    char c;

    printf("输入一个整数：");
    scanf("%d", &a);  // 输入10后回车，缓冲区残留'\n'
    printf("输入一个字符：");
    c = getchar();    // 直接读取残留的'\n'，而非新输入的字符
    printf("你输入的字符是：%d（ASCII码）\n", c);  // 输出10（'\n'的ASCII码）
    return 0;
}
```

**解决方法**：读取前清空输入缓冲区，例如用`getchar()`吸收残留字符，或用`fflush(stdin)`（部分编译器支持）。  


### 四、常见问题与最佳实践
1. **`scanf`读取字符串溢出**：改用`fgets`读取字符串，指定最大长度。  
2. **`%c`读取到空白字符**：在`%c`前加空格（`scanf(" %c", &c)`），或用`getchar()`先吸收空白。  
3. **输出不立即显示**：确保有`'\n'`，或用`fflush(stdout)`手动刷新缓冲区。  
4. **输入格式错误**：检查`scanf`的返回值，判断是否成功读取预期数量的变量。  


### 总结
C语言的输入输出依赖`stdio.h`库函数，核心函数包括：  
- **输出**：`printf`（格式化输出）、`putchar`（单个字符）、`puts`（字符串+换行）。  
- **输入**：`scanf`（格式化输入）、`getchar`（单个字符）、`fgets`（安全读字符串）。  

理解格式说明符、缓冲区机制和安全注意事项（如避免溢出）是掌握输入输出的关键。实际开发中，优先用`fgets`读取字符串，用`scanf`的返回值检查输入有效性，以编写健壮的代码。
