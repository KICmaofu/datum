在C语言中，**数组**是一组具有相同数据类型的元素的集合，这些元素在内存中连续存储，通过**下标**（索引）可以快速访问单个元素。数组的核心作用是“批量管理相同类型的数据”，避免为每个元素单独定义变量，大幅简化代码。


### 一、数组的基本概念
- **特点**：元素类型相同、内存连续、通过下标访问。  
- **下标**：数组元素的编号，从`0`开始（第一个元素下标为0，第二个为1，以此类推）。  
- **长度**：数组中元素的总个数，定义后不可修改（C语言中数组长度固定）。  


### 二、一维数组（最常用）
一维数组是最简单的数组形式，元素按线性顺序排列。


#### 1. 定义格式
```c
数据类型 数组名[长度];
```
- **数据类型**：数组中所有元素的类型（如`int`、`float`、`char`等）。  
- **数组名**：遵循标识符命名规则（见名知意，如`scores`、`nums`）。  
- **长度**：必须是**常量表达式**（整数常量或用`#define`定义的宏，C99及以上支持变量长度，但不推荐）。  


#### 2. 初始化（定义时赋值）
数组初始化有多种方式，需注意未显式赋值的元素会被自动初始化（数值型为0，字符型为`'\0'`）。

- **完全初始化**：为所有元素赋值（长度可省略，编译器自动计算）。  
  ```c
  int nums[5] = {1, 2, 3, 4, 5};  // 长度5，元素分别为1,2,3,4,5
  int nums[] = {1, 2, 3};         // 长度省略，自动计算为3（推荐）
  ```

- **部分初始化**：只给前n个元素赋值，剩余元素自动为0。  
  ```c
  int nums[5] = {1, 2};  // 元素为1,2,0,0,0
  ```

- **全部初始化为0**：只写一个0，剩余元素自动为0。  
  ```c
  int nums[5] = {0};  // 所有元素均为0
  ```


#### 3. 访问数组元素
通过**下标**访问，格式：`数组名[下标]`。  
**注意**：下标范围是`0 ~ 长度-1`，超出范围会导致“下标越界”（编译器不报错，但会访问非法内存，可能导致程序崩溃）。

**示例**：遍历并输出数组元素  
```c
#include <stdio.h>

int main() {
    int scores[] = {85, 92, 78, 90, 88};
    int len = 5;  // 数组长度（可通过sizeof计算，见下文）

    // 遍历数组（下标从0到4）
    for (int i = 0; i < len; i++) {
        printf("scores[%d] = %d\n", i, scores[i]);
    }
    return 0;
}
```
输出：  
```
scores[0] = 85
scores[1] = 92
scores[2] = 78
scores[3] = 90
scores[4] = 88
```


#### 4. 计算数组长度
通过`sizeof`运算符可计算数组总字节数，再除以单个元素的字节数，得到长度：  
```c
int nums[] = {1, 2, 3, 4};
int len = sizeof(nums) / sizeof(nums[0]);  // 总字节数 / 单个元素字节数 = 4
```


### 三、多维数组（以二维数组为例）
多维数组可理解为“数组的数组”，最常用的是二维数组（如矩阵、表格），格式为`行×列`。


#### 1. 定义格式
```c
数据类型 数组名[行数][列数];
```
- 行数和列数均为常量表达式，**列数不能省略**（行数在初始化时可省略）。  


#### 2. 初始化
- **按行初始化**：用大括号分组，每组对应一行元素。  
  ```c
  int matrix[2][3] = {
      {1, 2, 3},  // 第0行
      {4, 5, 6}   // 第1行
  };
  ```

- **连续初始化**：不分组，按行优先顺序赋值（不推荐，可读性差）。  
  ```c
  int matrix[2][3] = {1, 2, 3, 4, 5, 6};  // 效果同上
  ```

- **部分初始化**：未赋值的元素自动为0，行数可省略（编译器根据列数和元素总数计算）。  
  ```c
  int matrix[][3] = {1, 2, 3, 4};  // 列数3，元素共4个 → 行数=2（第0行1,2,3；第1行4,0,0）
  ```


#### 3. 访问二维数组元素
格式：`数组名[行下标][列下标]`（行、列下标均从0开始）。  

**示例**：输出二维数组（矩阵）  
```c
#include <stdio.h>

int main() {
    int matrix[2][3] = {{1,2,3}, {4,5,6}};
    // 外层循环控制行，内层循环控制列
    for (int i = 0; i < 2; i++) {  // 行下标0~1
        for (int j = 0; j < 3; j++) {  // 列下标0~2
            printf("%d ", matrix[i][j]);
        }
        printf("\n");  // 每行结束换行
    }
    return 0;
}
```
输出：  
```
1 2 3 
4 5 6 
```


#### 4. 内存存储
二维数组在内存中**按行优先连续存储**（先存第0行所有元素，再存第1行，以此类推）。例如`matrix[2][3]`的内存布局为：  
`matrix[0][0] → matrix[0][1] → matrix[0][2] → matrix[1][0] → matrix[1][1] → matrix[1][2]`  


### 四、数组作为函数参数
数组作为函数参数时，**传递的是首元素的地址**（而非整个数组的副本），因此函数内部修改数组元素会影响原数组（与普通变量的“传值调用”不同）。


#### 1. 一维数组作为参数
函数定义时，参数列表中数组的长度可省略（通常需额外传递长度）：  
```c
// 函数：打印一维数组元素
void printArray(int arr[], int len) {  // 数组名arr接收首地址，len是数组长度
    for (int i = 0; i < len; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");
}

int main() {
    int nums[] = {10, 20, 30};
    int len = sizeof(nums) / sizeof(nums[0]);
    printArray(nums, len);  // 传递数组名（首地址）和长度
    return 0;
}
```


#### 2. 二维数组作为参数
函数定义时，**列数不能省略**（行数可省略）：  
```c
// 函数：打印二维数组（2行3列）
void printMatrix(int matrix[][3], int rows) {  // 列数3必须写，rows是行数
    for (int i = 0; i < rows; i++) {
        for (int j = 0; j < 3; j++) {
            printf("%d ", matrix[i][j]);
        }
        printf("\n");
    }
}

int main() {
    int matrix[2][3] = {{1,2,3}, {4,5,6}};
    printMatrix(matrix, 2);  // 传递数组名和行数
    return 0;
}
```


### 五、字符数组与字符串
字符数组是元素类型为`char`的数组，**字符串是特殊的字符数组**（以空字符`'\0'`结尾，`'\0'`的ASCII码为0，用于标记字符串结束）。


#### 1. 字符数组初始化（字符串）
- 用字符串常量初始化（自动添加`'\0'`）：  
  ```c
  char str1[] = "hello";  // 长度6（h,e,l,l,o,'\0'），推荐
  ```

- 用字符列表初始化（需手动添加`'\0'`）：  
  ```c
  char str2[] = {'h', 'i', '\0'};  // 长度3（必须加'\0'，否则不是字符串）
  ```


#### 2. 字符串的输入输出
- 输出：用`printf("%s", str)`（自动打印到`'\0'`为止）。  
- 输入：用`scanf("%s", str)`（读取到空格/回车为止，自动添加`'\0'`）。  

**示例**：  
```c
#include <stdio.h>

int main() {
    char name[20];  // 定义可存19个字符+1个'\0'的数组
    printf("请输入姓名：");
    scanf("%s", name);  // 无需&，name是首地址
    printf("你好，%s！\n", name);
    return 0;
}
```


### 六、数组的注意事项
1. **长度固定**：数组定义后长度不可修改（如需动态调整，需用指针和动态内存分配）。  
2. **下标越界**：访问`下标 < 0`或`下标 >= 长度`的元素会导致未定义行为（程序崩溃、数据错误等），需严格控制下标范围。  
3. **数组名不是指针**：数组名在大多数情况下会被转换为指向首元素的指针（如作为函数参数时），但数组名本身不是指针（不能被赋值，如`arr = &x`是错误的）。  
4. **初始化限制**：只能在定义时初始化，定义后不能整体赋值（如`int nums[3]; nums = {1,2,3};`是错误的，需逐个元素赋值）。  


### 总结
数组是C语言中管理批量数据的基础工具，一维数组适合存储线性数据（如成绩、温度），二维数组适合存储表格类数据（如矩阵）。字符数组通过`'\0'`扩展为字符串，是文本处理的核心。使用时需注意下标越界和长度固定的限制，合理利用函数参数传递数组可提高代码复用性。
