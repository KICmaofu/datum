C语言基础语法是编程入门的核心，核心围绕“数据存储（变量/类型）、逻辑控制（分支/循环）、代码组织（函数/数组）”三大模块，以下是结构化讲解，含核心知识点、示例代码和关键注意事项：

### 一、环境搭建与第一个C程序
#### 核心知识点
- 编译运行流程：编写`.c`源文件 → 编译器（GCC/Dev-C++）编译生成`.exe`可执行文件 → 运行程序。
- 基本程序结构：包含头文件、主函数（程序入口）、语句结束符`;`。

#### 示例代码（Hello World）
```c
#include <stdio.h>  // 头文件：包含输入输出函数（如printf）

int main() {  // 主函数：程序入口，int表示返回值类型
    printf("Hello, C Language!\n");  // 输出语句，\n是换行符
    return 0;  // 主函数返回0，表示程序正常结束
}
```

#### 关键注意
- 所有语句必须以`;`结尾，遗漏会报错。
- `main()`是程序唯一入口，不可省略。
- 头文件用`#include <头文件名>`引入，常用`stdio.h`（输入输出）、`stdlib.h`（标准库函数）。

---

### 二、变量与数据类型
#### 核心知识点
- 变量：存储数据的容器，需先定义后使用（格式：`类型 变量名;`）。
- 基本数据类型（常用）：
  - 整型：`int`（4字节，存储整数，如`10、-5`）。
  - 字符型：`char`（1字节，存储单个字符，用单引号`' '`包裹，如`'a'、'1'`）。
  - 实型（浮点型）：`float`（4字节，单精度小数，如`3.14`）、`double`（8字节，双精度，精度更高）。

#### 示例代码
```c
#include <stdio.h>
int main() {
    // 变量定义（可初始化，也可先定义后赋值）
    int age = 20;          // 整型变量
    char gender = 'M';     // 字符型变量
    float score = 95.5f;   // float需加f后缀，否则默认double
    double pi = 3.1415926; // 双精度浮点型

    // 输出变量值（%d对应int，%c对应char，%f对应float/double）
    printf("年龄：%d\n", age);
    printf("性别：%c\n", gender);
    printf("成绩：%f\n", score);
    printf("圆周率：%lf\n", pi); // %lf专门用于double
    return 0;
}
```

#### 关键注意
- 变量名遵循命名规则：由字母、数字、下划线组成，不能以数字开头，区分大小写（如`age`和`Age`是两个变量）。
- 字符型变量存储的是ASCII码值，可直接参与运算（如`'a' + 1`结果为`'b'`）。

### 三、运算符与表达式
#### 核心知识点
- 运算符分类：算术运算符、关系运算符、逻辑运算符、赋值运算符。
- 表达式：运算符连接变量/常量组成的式子，有返回值（如`3+5`、`a>b`）。

#### 常用运算符示例
| 类型         | 运算符                | 示例                  | 结果（假设a=5, b=2） |
|--------------|-----------------------|-----------------------|----------------------|
| 算术运算符   | `+、-、*、/、%`（取余）| `a/b`、`a%b`          | 2、1（整数除法取整） |
| 关系运算符   | `>、<、==（等于）、!=` | `a==b`、`a>b`         | 0（假）、1（真）     |
| 逻辑运算符   | `&&（与）、||（或）、!（非）` | `(a>3)&&(b<5)` | 1（真）              |
| 赋值运算符   | `=、+=、-=`           | `a+=3`（等价a=a+3）   | a=8                  |

#### 示例代码
```c
#include <stdio.h>
int main() {
    int a = 5, b = 2;
    int sum = a + b;          // 算术运算
    int isGreater = a > b;    // 关系运算（结果1）
    int logic = (a>3) && (b<1); // 逻辑运算（结果0）

    printf("和：%d\n", sum);
    printf("a>b？%d\n", isGreater); // C语言中1表示真，0表示假
    return 0;
}
```

#### 关键注意
- 优先级：算术运算符 > 关系运算符 > 逻辑运算符 > 赋值运算符，可加`()`改变优先级。
- 除法`/`：两个整数相除结果为整数（如`5/2=2`），需用浮点型转换（如`(float)5/2=2.5`）。
- 取余`%`：仅用于整型，结果符号与被除数一致（如`-5%2=-1`）。

### 四、控制语句（分支与循环）
#### 1. 分支语句（if-else、switch）
##### if-else（条件判断）
```c
#include <stdio.h>
int main() {
    int score = 85;
    if (score >= 90) {
        printf("优秀\n");
    } else if (score >= 60) {
        printf("及格\n");
    } else {
        printf("不及格\n");
    }
    return 0;
}
```

##### switch（多分支选择）
```c
#include <stdio.h>
int main() {
    char grade = 'B';
    switch (grade) {
        case 'A': printf("90-100分\n"); break; // break跳出switch，否则继续执行下一个case
        case 'B': printf("80-89分\n"); break;
        case 'C': printf("60-79分\n"); break;
        default: printf("不及格\n"); // 所有case不匹配时执行
    }
    return 0;
}
```

#### 2. 循环语句（for、while、do-while）
##### for（固定次数循环，最常用）
```c
#include <stdio.h>
int main() {
    // 格式：for(初始化; 循环条件; 迭代)
    for (int i = 1; i <= 5; i++) { // i从1到5，循环5次
        printf("第%d次循环\n", i);
    }
    return 0;
}
```

##### while（先判断条件，再执行）
```c
#include <stdio.h>
int main() {
    int i = 1;
    while (i <= 5) { // 条件为真时执行循环体
        printf("i=%d\n", i);
        i++; // 迭代（必须写，否则死循环）
    }
    return 0;
}
```

##### do-while（先执行一次，再判断条件）
```c
#include <stdio.h>
int main() {
    int i = 1;
    do {
        printf("i=%d\n", i);
        i++;
    } while (i <= 5); // 至少执行一次
    return 0;
}
```

#### 关键注意
- `break`：跳出当前循环或`switch`语句；`continue`：跳过本次循环剩余语句，进入下一次循环。
- 循环条件必须有终止条件，否则会陷入死循环（如`while(1)`需内部用`break`退出）。

### 五、数组（批量存储相同类型数据）
#### 核心知识点
- 定义格式：`类型 数组名[长度];`（长度必须是常量，如`int arr[5]`）。
- 索引：从0开始（如`arr[0]`是第一个元素，`arr[4]`是第五个元素）。
- 初始化：可在定义时赋值（如`int arr[5] = {1,2,3,4,5}`），未赋值元素默认是0。

#### 示例代码（一维数组遍历）
```c
#include <stdio.h>
int main() {
    int scores[5] = {85, 92, 78, 90, 88}; // 存储5个学生成绩
    int sum = 0;

    // 循环遍历数组，计算总分
    for (int i = 0; i < 5; i++) {
        sum += scores[i];
        printf("第%d个学生成绩：%d\n", i+1, scores[i]);
    }

    printf("平均分：%f\n", sum / 5.0); // 用5.0确保浮点除法
    return 0;
}
```

#### 关键注意
- 数组长度不能动态改变（C语言不支持动态数组，需提前确定长度）。
- 避免索引越界（如`arr[5]`对于长度为5的数组是非法的，会导致内存错误）。

### 六、函数（代码模块化）
#### 核心知识点
- 定义格式：`返回值类型 函数名(参数列表) { 函数体; return 返回值; }`。
- 调用：`函数名(实参);`（实参与形参类型、个数必须一致）。
- 作用：拆分复杂代码，提高复用性（如封装计算、输出等功能）。

#### 示例代码（自定义加法函数）
```c
#include <stdio.h>

// 自定义函数：计算两个整数的和，返回int类型
int add(int a, int b) { // a、b是形参（形式参数）
    return a + b; // 返回计算结果
}

int main() {
    int x = 10, y = 20;
    int result = add(x, y); // 调用add函数，x、y是实参（实际参数）
    printf("%d + %d = %d\n", x, y, result);
    return 0;
}
```

#### 关键注意
- 函数需先声明或定义，再调用（若函数定义在`main`之后，需先声明，如`int add(int a, int b);`）。
- 无返回值的函数用`void`表示（如`void printInfo() { ... }`），无需`return`语句。

### 七、核心重点总结
1. 语法规范：`;`不可漏、变量先定义后使用、区分大小写。
2. 数据存储：用变量存单个值，数组存批量相同类型值。
3. 逻辑控制：分支（if-else/switch）做选择，循环（for/while）重复执行。
4. 代码组织：函数拆分功能，提高可读性和复用性。
