在C语言中，**运算符**是用于执行各种运算（如算术、比较、逻辑操作等）的符号。根据功能不同，C语言的运算符可分为多个类别，每个类别有特定的用途和优先级。以下是详细分类及说明：


### 一、算术运算符
用于执行基本的数学运算（加、减、乘、除等），操作数通常为数值型（整数、浮点数）。

| 运算符 | 名称       | 示例                  | 说明                                  |
|--------|------------|-----------------------|---------------------------------------|
| `+`    | 加法       | `a + b`               | 计算a与b的和                          |
| `-`    | 减法       | `a - b`               | 计算a与b的差                          |
| `*`    | 乘法       | `a * b`               | 计算a与b的积                          |
| `/`    | 除法       | `a / b`               | 整数除法：结果取整数部分（如`5/2=2`）；浮点数除法：结果为精确值（如`5.0/2=2.5`） |
| `%`    | 取模（余数） | `a % b`               | 仅用于整数，返回a除以b的余数（结果符号与被除数一致，如`-5%3=-2`，`5%-3=2`） |


### 二、关系运算符
用于比较两个值的关系，返回**逻辑值**（`0`表示“假”，`1`表示“真”）。

| 运算符 | 名称       | 示例                  | 说明                                  |
|--------|------------|-----------------------|---------------------------------------|
| `==`   | 等于       | `a == b`              | 若a与b相等，返回1；否则返回0          |
| `!=`   | 不等于     | `a != b`              | 若a与b不相等，返回1；否则返回0        |
| `>`    | 大于       | `a > b`               | 若a大于b，返回1；否则返回0            |
| `<`    | 小于       | `a < b`               | 若a小于b，返回1；否则返回0            |
| `>=`   | 大于等于   | `a >= b`              | 若a大于或等于b，返回1；否则返回0      |
| `<=`   | 小于等于   | `a <= b`              | 若a小于或等于b，返回1；否则返回0      |

**注意**：区分`==`（比较）和`=`（赋值），例如`if(a == 5)`是判断a是否为5，而`if(a = 5)`会将5赋值给a，且表达式结果为5（非0，即“真”），容易出错。


### 三、逻辑运算符
用于对逻辑值（真/假）进行运算，返回`0`（假）或`1`（真）。

| 运算符 | 名称       | 示例                  | 说明（短路求值特性）                  |
|--------|------------|-----------------------|---------------------------------------|
| `&&`   | 逻辑与     | `exp1 && exp2`        | 当exp1为假时，**不再计算exp2**，整体返回0；否则返回exp2的结果 |
| `||`   | 逻辑或     | `exp1 || exp2`        | 当exp1为真时，**不再计算exp2**，整体返回1；否则返回exp2的结果 |
| `!`    | 逻辑非     | `!exp`                | 若exp为真（非0），返回0；若exp为假（0），返回1 |

**示例**：  
- `3 > 5 && 2 < 4`：先算`3>5`为0（假），不再算`2<4`，整体返回0。  
- `3 < 5 || 2 > 4`：先算`3<5`为1（真），不再算`2>4`，整体返回1。  


### 四、赋值运算符
用于给变量赋值，基础赋值符为`=`，另有复合赋值符（简化运算）。

| 运算符 | 名称       | 示例                  | 等价表达式                            |
|--------|------------|-----------------------|---------------------------------------|
| `=`    | 基础赋值   | `a = b`               | 将b的值赋给a                          |
| `+=`   | 加后赋值   | `a += b`              | `a = a + b`                           |
| `-=`   | 减后赋值   | `a -= b`              | `a = a - b`                           |
| `*=`   | 乘后赋值   | `a *= b`              | `a = a * b`                           |
| `/=`   | 除后赋值   | `a /= b`              | `a = a / b`                           |
| `%=`   | 取模后赋值 | `a %= b`              | `a = a % b`                           |

**注意**：赋值表达式的结果是“赋值后变量的值”，例如`a = 3 + 5`中，表达式结果为8，且a的值为8。


### 五、自增/自减运算符
用于对变量进行+1或-1操作，分为**前缀**（先变后用）和**后缀**（先用后变）。

| 运算符 | 名称       | 示例                  | 说明                                  |
|--------|------------|-----------------------|---------------------------------------|
| `++`   | 自增       | `++a`（前缀）         | 先将a的值+1，再使用a的值              |
|        |            | `a++`（后缀）         | 先使用a的值，再将a的值+1              |
| `--`   | 自减       | `--a`（前缀）         | 先将a的值-1，再使用a的值              |
|        |            | `a--`（后缀）         | 先使用a的值，再将a的值-1              |

**示例**：  
```c
int a = 5, b;
b = ++a;  // a先+1变为6，再赋值给b → b=6，a=6
b = a++;  // 先将a=6赋值给b，再a+1 → b=6，a=7
```

**注意**：避免在同一表达式中多次使用自增/自减（如`a = a++ + ++a`），可能导致未定义行为（结果依赖编译器）。


### 六、条件运算符（三目运算符）
C语言中唯一的三目运算符，用于简化条件判断，格式为：  
`表达式1 ? 表达式2 : 表达式3`  

**功能**：若表达式1为真（非0），则整个表达式结果为表达式2的值；否则为表达式3的值。  

**示例**：  
```c
int max = (a > b) ? a : b;  // 若a>b，max=a；否则max=b
```

可嵌套使用（但可读性较差），例如：  
`(a > b) ? (a > c ? a : c) : (b > c ? b : c)`  // 求a、b、c的最大值。


### 七、逗号运算符
用于将多个表达式分隔为一个表达式，整个表达式的结果为**最后一个表达式的值**。  

格式：`表达式1, 表达式2, ..., 表达式n`  

**示例**：  
```c
int a = (3+5, 2*4, 6-1);  // 表达式依次计算，结果为最后一个6-1=5 → a=5
for (i=0, j=10; i<j; i++, j--)  // for循环中常用，分隔初始化和更新操作
```


### 八、位运算符
对操作数的二进制位进行逐位操作（仅用于整数）。

| 运算符 | 名称       | 示例                  | 说明                                  |
|--------|------------|-----------------------|---------------------------------------|
| `&`    | 按位与     | `a & b`               | 逐位与：对应位均为1则为1，否则为0     |
| `|`    | 按位或     | `a | b`               | 逐位或：对应位有一个为1则为1，否则为0 |
| `^`    | 按位异或   | `a ^ b`               | 逐位异或：对应位不同则为1，相同则为0   |
| `~`    | 按位取反   | `~a`                  | 逐位取反：0变1，1变0（包括符号位）    |
| `<<`   | 左移       | `a << n`              | 将a的二进制位左移n位，右边补0（等价于a×2ⁿ，溢出需注意） |
| `>>`   | 右移       | `a >> n`              | 将a的二进制位右移n位：无符号数左边补0；有符号数通常补符号位（算术右移）（等价于a÷2ⁿ，向下取整） |

**示例**：  
- `3 & 5`：3是`011`，5是`101`，逐位与得`001`（即1）。  
- `3 << 1`：`011`左移1位为`110`（即6）。  


### 九、其他常用运算符
1. **指针运算符**：  
   - `&`：取地址符，`&a`表示获取变量a的内存地址。  
   - `*`：解引用符，`*p`表示获取指针p指向的变量的值。  

2. **成员运算符**：  
   - `.`：访问结构体/联合体的成员，`s.name`表示结构体变量s的name成员。  
   - `->`：通过指针访问结构体/联合体的成员，`p->age`表示指针p指向的结构体的age成员。  

3. **sizeof运算符**：  
   计算变量或类型所占的字节数，格式：`sizeof(变量/类型)` 或 `sizeof 变量`（对类型必须加括号）。  
   示例：`sizeof(int)`（通常为4）、`sizeof(a)`（若a是int，结果为4）。  


### 运算符优先级与结合性
当多个运算符同时出现时，优先级决定运算顺序（优先级高的先算）；优先级相同时，结合性（左结合/右结合）决定顺序。  

- **优先级大致顺序**（从高到低）：  
  单目运算符（`!`、`~`、`++`、`--`、`*`、`&`、`sizeof`）→ 算术运算符（`*`、`/`、`%` 高于 `+`、`-`）→ 位运算符（`<<`、`>>` 高于 `&`、`^`、`|`）→ 关系运算符 → 逻辑运算符（`&&` 高于 `||`）→ 条件运算符 → 赋值运算符 → 逗号运算符。  

- **结合性**：  
  大部分运算符左结合（从左到右计算，如`a + b + c`）；单目运算符、条件运算符、赋值运算符右结合（从右到左计算，如`a = b = c`等价于`a = (b = c)`）。  


通过掌握运算符的分类、功能及优先级，可正确编写和理解复杂表达式，避免逻辑错误。
