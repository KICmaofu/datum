# Python pyecharts 模块完全指南：交互式可视化图表实战
pyecharts 是基于百度 ECharts 开发的 Python 可视化库，核心优势是 **生成交互式、可动态渲染的图表**（支持鼠标hover提示、缩放、数据筛选等），且支持直接导出为 HTML 文件（便于 Web 展示）、图片等格式。相比 Matplotlib/Seaborn 的静态图表，pyecharts 更适合做数据看板、Web 可视化报告、动态数据展示等场景，支持折线图、柱状图、饼图、地图、雷达图等 30+ 图表类型。


## 一、核心定位与优势
### 1. 模块特点
- **交互式体验**：图表支持鼠标hover显示详情、缩放、拖拽、数据筛选，用户体验远超静态图表；
- **Web 友好**：默认导出为 HTML 文件，可直接嵌入网页，无需依赖后端服务；
- **配置灵活**：支持自定义图表标题、坐标轴、图例、颜色、样式等所有细节；
- **支持多维度数据**：轻松实现组合图表（如折线图+柱状图）、地图可视化、3D 图表；
- **轻量化**：无需复杂环境配置，纯 Python 代码编写，学习成本低。

### 2. 版本说明（关键！）
pyecharts 分为 **v0.5.x** 和 **v1.x** 两个主要版本，两者 API 不兼容（v1.x 更简洁、功能更全），目前主流使用 **v1.x**（本文基于 v1.1.0+ 版本）。

### 3. 适用场景
- 数据看板、Web 可视化报告；
- 动态数据展示（如实时监控数据）；
- 地图可视化（如省份/城市数据分布）；
- 演示汇报用交互式图表。

## 三、核心概念与基础流程
pyecharts v1.x 采用「链式调用」语法，核心流程为：
1. 导入对应图表类（如 `Line` 折线图、`Bar` 柱状图）；
2. 初始化图表（设置宽高、主题等）；
3. 添加数据（`add_xaxis` 横坐标 + `add_yaxis` 纵坐标/系列数据）；
4. 配置全局样式（标题、坐标轴、图例等）；
5. 导出/展示图表（`render()` 导出 HTML，或直接在 Notebook 中显示）。

### 核心模块导入
```python
# 1. 导入图表类（按需导入）
from pyecharts.charts import Line, Bar, Pie, Map, Radar, Scatter
# 2. 导入配置项（标题、坐标轴、图例等）
from pyecharts import options as opts
# 3. 导入辅助工具（颜色、标记点、数据转换等）
from pyecharts.globals import ThemeType, SymbolType
from pyecharts.commons.utils import JsCode
```


## 四、常用图表实战（含代码示例）
### 1. 折线图（Line）：展示趋势变化
适合展示时间序列数据的趋势（如销售额月度变化、温度变化等）。

#### 基础示例：月度销售额趋势
```python
from pyecharts.charts import Line
from pyecharts import options as opts
from pyecharts.globals import ThemeType

# 1. 准备数据
x_data = ["1月", "2月", "3月", "4月", "5月", "6月"]
y_data1 = [5800, 6200, 7500, 8100, 9500, 12000]  # 产品A销售额
y_data2 = [4200, 4800, 5300, 6400, 7200, 8800]  # 产品B销售额

# 2. 构建折线图（链式调用）
line_chart = (
    Line(
        init_opts=opts.InitOpts(
            theme=ThemeType.LIGHT,  # 主题（LIGHT/DARK/CHALK等）
            width="1200px",  # 图表宽度
            height="600px"   # 图表高度
        )
    )
    # 添加横坐标
    .add_xaxis(xaxis_data=x_data)
    # 添加产品A数据系列（标记点：最大值/最小值）
    .add_yaxis(
        series_name="产品A",
        y_axis=y_data1,
        symbol=SymbolType.CIRCLE,  # 标记点样式（圆形）
        symbol_size=8,  # 标记点大小
        markpoint_opts=opts.MarkPointOpts(
            data=[
                opts.MarkPointItem(type_="max", name="最大值"),
                opts.MarkPointItem(type_="min", name="最小值")
            ]
        ),
        linestyle_opts=opts.LineStyleOpts(width=3)  # 线条宽度
    )
    # 添加产品B数据系列（标记线：平均值）
    .add_yaxis(
        series_name="产品B",
        y_axis=y_data2,
        symbol=SymbolType.DIAMOND,  # 标记点样式（菱形）
        symbol_size=8,
        markline_opts=opts.MarkLineOpts(
            data=[opts.MarkLineItem(type_="average", name="平均值")]
        )
    )
    # 配置全局样式
    .set_global_opts(
        # 标题配置
        title_opts=opts.TitleOpts(
            title="2025年上半年产品销售额趋势",
            subtitle="单位：元",
            title_textstyle_opts=opts.TextStyleOpts(font_size=18),
            subtitle_textstyle_opts=opts.TextStyleOpts(font_size=14)
        ),
        # 坐标轴配置
        xaxis_opts=opts.AxisOpts(axislabel_opts=opts.LabelOpts(font_size=12)),
        yaxis_opts=opts.AxisOpts(
            name="销售额（元）",
            name_textstyle_opts=opts.TextStyleOpts(font_size=12),
            axislabel_opts=opts.LabelOpts(font_size=12)
        ),
        # 图例配置（位置：右上）
        legend_opts=opts.LegendOpts(pos_top="5%", pos_right="10%"),
        # 工具箱配置（支持下载图片、数据视图等）
        toolbox_opts=opts.ToolboxOpts(
            feature={
                "saveAsImage": {},  # 下载图片
                "dataView": {},    # 查看数据
                "restore": {},      # 重置图表
                "zoom": {}          # 缩放
            }
        )
    )
)

# 3. 导出为HTML文件（在当前目录生成 line_chart.html）
line_chart.render("line_chart.html")
```

#### 关键配置说明：
- `ThemeType`：支持 `LIGHT`（浅色）、`DARK`（深色）、`CHALK`（粉笔风）、`ESSOS`（权游风）等 10+ 主题；
- `markpoint_opts`：标记点（最大值、最小值、自定义点）；
- `markline_opts`：标记线（平均值、中位数、自定义线）；
- `toolbox_opts`：工具箱（必备，支持下载、数据查看等交互功能）。


### 2. 柱状图（Bar）：对比分类数据
适合对比不同分类的数据（如各部门业绩、各产品销量等），支持横向柱状图、堆叠柱状图。

#### 示例：各部门季度业绩对比（横向柱状图）
```python
from pyecharts.charts import Bar
from pyecharts import options as opts
from pyecharts.globals import ThemeType

# 数据准备
departments = ["销售部", "技术部", "市场部", "运营部", "财务部"]
q1_data = [120, 85, 95, 70, 45]  # Q1业绩（万元）
q2_data = [150, 92, 110, 85, 50]  # Q2业绩（万元）

# 构建横向柱状图（rotate=90 为纵向，默认横向需设置 yaxis 为分类轴）
bar_chart = (
    Bar(init_opts=opts.InitOpts(theme=ThemeType.DARK, width="1000px", height="600px"))
    .add_xaxis(xaxis_data=departments)
    # 添加Q1数据系列
    .add_yaxis(series_name="Q1业绩", y_axis=q1_data, color="#4CAF50")
    # 添加Q2数据系列
    .add_yaxis(series_name="Q2业绩", y_axis=q2_data, color="#2196F3")
    # 配置全局样式
    .set_global_opts(
        title_opts=opts.TitleOpts(title="2025年各部门季度业绩对比", subtitle="单位：万元"),
        xaxis_opts=opts.AxisOpts(name="业绩（万元）", name_textstyle_opts=opts.TextStyleOpts(color="white")),
        yaxis_opts=opts.AxisOpts(axislabel_opts=opts.TextStyleOpts(color="white", font_size=12)),
        legend_opts=opts.LegendOpts(textstyle_opts=opts.TextStyleOpts(color="white")),
        toolbox_opts=opts.ToolboxOpts()
    )
    # 配置系列样式（hover时显示数值）
    .set_series_opts(
        label_opts=opts.LabelOpts(
            is_show=True, 
            position="right",  # 数值显示在柱状图右侧
            color="white"
        )
    )
)

# 导出HTML
bar_chart.render("bar_chart.html")
```

#### 进阶：堆叠柱状图（只需添加 `stack` 参数）
```python
# 在 add_yaxis 中添加 stack 参数（同一 stack 名称的系列会堆叠）
.add_yaxis(series_name="Q1业绩", y_axis=q1_data, color="#4CAF50", stack="total")
.add_yaxis(series_name="Q2业绩", y_axis=q2_data, color="#2196F3", stack="total")
```


### 3. 饼图（Pie）：展示占比数据
适合展示各部分占总体的比例（如市场份额、用户分布、费用占比等），支持环形饼图、玫瑰图。

#### 示例：用户来源分布（环形饼图）
```python
from pyecharts.charts import Pie
from pyecharts import options as opts
from pyecharts.globals import ThemeType

# 数据准备（格式：[(类别1, 数值1), (类别2, 数值2), ...]）
user_source = [
    ("抖音推广", 3500),
    ("微信公众号", 2800),
    ("官网", 1500),
    ("朋友推荐", 1200),
    ("其他", 800)
]

# 构建环形饼图
pie_chart = (
    Pie(init_opts=opts.InitOpts(theme=ThemeType.CHALK, width="800px", height="600px"))
    # 添加数据（radius：内外半径，控制环形厚度）
    .add(
        series_name="用户来源占比",
        data_pair=user_source,
        radius=["40%", "70%"],  # 内半径40%，外半径70%（环形）
        center=["50%", "50%"],  # 图表居中
        label_opts=opts.LabelOpts(is_show=False, position="center")  # 隐藏默认标签
    )
    # 配置全局样式
    .set_global_opts(
        title_opts=opts.TitleOpts(title="2025年新用户来源分布"),
        legend_opts=opts.LegendOpts(orient="vertical", pos_top="15%", pos_left="2%")  # 图例垂直排列在左侧
    )
    # 配置系列样式（hover时显示详情，中心显示总人数）
    .set_series_opts(
        tooltip_opts=opts.TooltipOpts(
            trigger="item", formatter="{b}: {c} ({d}%)"  # hover提示格式：类别：数值（占比）
        ),
        # 中心文本（总用户数）
        label_opts=opts.LabelOpts(
            is_show=True,
            position="center",
            formatter=f"总用户数\n{sum([x[1] for x in user_source])}",
            font_size=16,
            font_weight="bold"
        )
    )
)

pie_chart.render("pie_chart.html")
```

#### 进阶：玫瑰图（设置 `rosetype` 参数）
```python
# 在 add 中添加 rosetype 参数（玫瑰图样式）
.add(
    series_name="用户来源占比",
    data_pair=user_source,
    radius=["40%", "70%"],
    center=["50%", "50%"],
    rosetype="radius",  # 玫瑰图（按半径缩放）
    label_opts=opts.LabelOpts(is_show=False)
)
```


### 4. 地图（Map）：地理数据可视化
pyecharts 地图功能强大，支持世界地图、中国地图、省份地图、城市地图，适合展示区域数据分布（如各省份销售额、人口分布等）。

#### 示例：中国各省份销售额分布
```python
from pyecharts.charts import Map
from pyecharts import options as opts
from pyecharts.globals import ThemeType

# 数据准备（格式：[(省份名称, 数值), ...]，省份名称需与地图数据一致）
province_sales = [
    ("北京", 1200), ("上海", 1500), ("广东", 1800), ("江苏", 1600), ("浙江", 1400),
    ("山东", 1100), ("四川", 950), ("湖北", 850), ("河南", 900), ("河北", 800)
]

# 构建中国地图
map_chart = (
    Map(init_opts=opts.InitOpts(theme=ThemeType.LIGHT, width="1200px", height="800px"))
    .add(
        series_name="销售额（万元）",
        data_pair=province_sales,
        maptype="china"  # 地图类型：china（中国地图）、省份名称（如"广东"为广东省地图）
    )
    .set_global_opts(
        title_opts=opts.TitleOpts(title="2025年各省份销售额分布", subtitle="单位：万元"),
        # 视觉映射配置（颜色梯度，数值对应颜色）
        visualmap_opts=opts.VisualMapOpts(
            min_=800,  # 最小值
            max_=1800,  # 最大值
            range_color=["#e0f7fa", "#4dd0e1", "#0097a7", "#006064"],  # 颜色梯度
            is_piecewise=True  # 分段显示（False为连续渐变）
        ),
        legend_opts=opts.LegendOpts(is_show=False)
    )
    .set_series_opts(
        tooltip_opts=opts.TooltipOpts(formatter="{b}: {c}万元")  # hover提示格式
    )
)

map_chart.render("map_china.html")
```

#### 关键说明：
- `maptype` 参数：`"china"`（中国地图）、省份名称（如 `"广东"` 生成广东省地图）、城市名称（如 `"深圳"` 生成深圳市地图）；
- 数据匹配：省份/城市名称必须与地图数据一致（如"广西壮族自治区"不能简写为"广西"）；
- 视觉映射：`VisualMapOpts` 控制颜色梯度，`is_piecewise=True` 分段显示（更清晰）。


### 5. 组合图表：多图表叠加（如折线图+柱状图）
适合在同一图表中展示不同类型的数据（如销售额+增长率）。

#### 示例：销售额（柱状图）+ 增长率（折线图）
```python
from pyecharts.charts import Bar, Line
from pyecharts import options as opts
from pyecharts.commons.utils import JsCode
from pyecharts.globals import ThemeType

# 数据准备
months = ["1月", "2月", "3月", "4月", "5月", "6月"]
sales = [5800, 6200, 7500, 8100, 9500, 12000]  # 销售额（万元）
growth_rates = [0, 6.9, 20.9, 8.0, 17.3, 26.3]  # 同比增长率（%）

# 构建组合图表（Bar + Line）
bar_line_chart = (
    Bar(init_opts=opts.InitOpts(theme=ThemeType.LIGHT, width="1200px", height="600px"))
    .add_xaxis(xaxis_data=months)
    # 柱状图：销售额
    .add_yaxis(
        series_name="销售额",
        y_axis=sales,
        color="#2196F3",
        label_opts=opts.LabelOpts(is_show=True, position="top")  # 数值显示在柱子顶部
    )
    # 配置全局样式
    .set_global_opts(
        title_opts=opts.TitleOpts(title="2025年上半年销售额及增长率", subtitle="销售额：万元，增长率：%"),
        xaxis_opts=opts.AxisOpts(axislabel_opts=opts.LabelOpts(font_size=12)),
        yaxis_opts=opts.AxisOpts(
            name="销售额（万元）",
            name_textstyle_opts=opts.TextStyleOpts(font_size=12),
            max_=15000  # y轴最大值
        ),
        legend_opts=opts.LegendOpts(pos_top="5%"),
        toolbox_opts=opts.ToolboxOpts()
    )
)

# 添加折线图（增长率，共用x轴，独立y轴）
line = (
    Line()
    .add_xaxis(xaxis_data=months)
    .add_yaxis(
        series_name="同比增长率",
        y_axis=growth_rates,
        color="#F44336",
        symbol=SymbolType.CIRCLE,
        symbol_size=8,
        # 折线图使用右侧y轴
        yaxis_index=1,
        label_opts=opts.LabelOpts(is_show=True, position="right")
    )
)

# 组合图表（添加右侧y轴）
bar_line_chart.extend_axis(
    yaxis=opts.AxisOpts(
        name="增长率（%）",
        name_textstyle_opts=opts.TextStyleOpts(font_size=12),
        max_=30,  # 增长率最大值
        axislabel_opts=opts.LabelOpts(formatter="{value}%")
    )
)
bar_line_chart.overlap(line)  # 叠加折线图

bar_line_chart.render("bar_line_chart.html")
```


## 五、高级功能
### 1. 图表导出为图片
pyecharts 默认导出 HTML，如需导出为 PNG/JPG，需借助 `snapshot` 插件和 Chrome 驱动：
```python
# 方法1：使用 render_notebook() + snapshot（Notebook 中）
from pyecharts.render import make_snapshot
from snapshot_selenium import snapshot

# 对已构建的图表导出为图片
make_snapshot(snapshot, line_chart.render(), "line_chart.png")
```

### 2. 动态数据加载（JavaScript 注入）
支持通过 JavaScript 动态加载数据（适合实时监控场景），需使用 `JsCode` 工具：
```python
# 示例：动态生成y轴数据（模拟实时更新）
from pyecharts.charts import Line
from pyecharts import options as opts
from pyecharts.commons.utils import JsCode

line_chart = (
    Line(init_opts=opts.InitOpts(width="1200px", height="600px"))
    .add_xaxis(xaxis_data=["1s", "2s", "3s", "4s", "5s"])
    .add_yaxis(
        series_name="实时数据",
        y_axis=JsCode("""
            function() {
                return [Math.random()*100, Math.random()*100, Math.random()*100, Math.random()*100, Math.random()*100];
            }
        """),
        symbol=SymbolType.CIRCLE
    )
    .set_global_opts(title_opts=opts.TitleOpts(title="实时数据监控"))
)

line_chart.render("dynamic_chart.html")
```

### 3. 响应式设计（适配不同屏幕）
通过 `responsive=True` 让图表自适应屏幕宽度：
```python
Line(
    init_opts=opts.InitOpts(
        responsive=True,  # 响应式设计
        width="100%"  # 宽度占满容器
    )
)
```


## 六、常见问题与避坑指南
### 1. 地图数据加载失败
- 问题：运行地图代码时提示「地图数据不存在」；
- 解决方案：确保已安装对应地图包（如 `echarts-china-provinces-pypkg`），且省份/城市名称与地图数据一致（如"内蒙古自治区"不能简写为"内蒙古"）。

### 2. 图表导出图片失败
- 问题：`make_snapshot` 报错，无法生成图片；
- 解决方案：
  1. 确保已安装 `pyecharts-snapshot` 和 `selenium`；
  2. 下载对应 Chrome 版本的 `chromedriver`，并配置到系统环境变量；
  3. 确保 Chrome 浏览器已安装。

### 3. 数据格式错误
- 问题：图表无数据显示，控制台报错；
- 解决方案：
  - 饼图/地图数据格式必须是「列表套元组」：`[(name1, value1), (name2, value2)]`；
  - 折线图/柱状图的 x 轴和 y 轴数据长度必须一致。

### 4. 版本冲突
- 问题：代码报错「AttributeError: module 'pyecharts' has no attribute 'xxx'」；
- 解决方案：确认安装的是 v1.x 版本（`pip show pyecharts` 查看版本），v0.5.x 与 v1.x API 不兼容，需卸载旧版本重新安装。


## 七、核心配置速查表
| 配置项          | 作用                                  | 示例                                  |
|-----------------|---------------------------------------|---------------------------------------|
| `InitOpts`      | 初始化配置（主题、宽高）              | `opts.InitOpts(theme=ThemeType.DARK, width="1200px")` |
| `TitleOpts`     | 标题配置（主标题、副标题、样式）      | `opts.TitleOpts(title="图表标题", subtitle="副标题")` |
| `AxisOpts`      | 坐标轴配置（名称、标签样式、范围）    | `opts.AxisOpts(name="销售额", max_=2000)` |
| `LegendOpts`    | 图例配置（位置、排列方式、样式）      | `opts.LegendOpts(pos_top="5%", orient="vertical")` |
| `TooltipOpts`   | 鼠标hover提示配置（格式、触发方式）   | `opts.TooltipOpts(formatter="{b}: {c}")` |
| `VisualMapOpts` | 地图/热力图颜色映射配置               | `opts.VisualMapOpts(min_=0, max_=2000, range_color=["#e0f7fa", "#006064"])` |
| `LabelOpts`     | 数据标签配置（是否显示、位置、样式）  | `opts.LabelOpts(is_show=True, position="top")` |


## 八、总结
pyecharts 是 Python 交互式可视化的首选工具，核心优势是「交互式体验+Web 友好」，适合数据看板、Web 报告、地理数据展示等场景。其 v1.x 版本采用链式调用语法，配置灵活、学习成本低，支持 30+ 图表类型和丰富的自定义功能。
