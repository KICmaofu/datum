## 一、什么是 SMTP 与 Python 实现？
SMTP（Simple Mail Transfer Protocol，简单邮件传输协议）是用于发送电子邮件的标准网络协议。Python3 内置 `smtplib` 模块，无需第三方依赖即可实现 SMTP 协议的邮件发送功能；配合 `email` 模块（用于构建邮件内容），可轻松实现纯文本、HTML 格式、带附件等多种类型的邮件发送。

### 核心优势
- 纯内置模块，无需额外安装
- 支持多种邮件格式（文本、HTML、附件）
- 兼容主流邮箱服务商（QQ、163、Gmail 等）
- 支持 SSL/TLS 加密传输，安全可靠
- 语法简洁，易集成到英语学习工具（如单词推送、学习报告）

### 核心模块说明
| 模块名       | 功能                          |
|--------------|-------------------------------|
| `smtplib`    | 建立 SMTP 连接、登录、发送邮件 |
| `email.mime.text` | 构建文本/HTML 格式邮件内容    |
| `email.mime.multipart` | 构建多部分邮件（含附件）      |
| `email.mime.base` | 处理邮件附件                  |

## 二、准备工作
### 1. 开启邮箱 SMTP 服务
发送邮件前，需先在邮箱中开启 SMTP 服务并获取 **授权码**（替代密码登录，更安全）。以下是主流邮箱的设置方法：

#### （1）QQ 邮箱
1. 登录 QQ 邮箱 → 进入「设置」→「账户」
2. 找到「POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV 服务」
3. 开启「SMTP 服务」，按提示验证手机
4. 生成「授权码」（保存好，仅显示一次）

#### （2）163 邮箱
1. 登录 163 邮箱 → 进入「设置」→「POP3/SMTP/IMAP」
2. 开启「SMTP 服务」
3. 验证手机后，使用邮箱密码或单独设置的授权码登录

#### （3）Gmail 邮箱
1. 登录 Gmail → 进入「账户设置」→「安全性」
2. 开启「两步验证」
3. 创建「应用专用密码」（用于 SMTP 登录）

### 2. 常见邮箱 SMTP 配置表
| 邮箱类型   | SMTP 服务器地址       | 端口（SSL） | 端口（非 SSL） |
|------------|-----------------------|-------------|---------------|
| QQ 邮箱    | `smtp.qq.com`         | 465         | 25（部分网络禁用） |
| 163 邮箱   | `smtp.163.com`        | 465/994     | 25            |
| 新浪邮箱   | `smtp.sina.com`       | 465         | 25            |
| Gmail      | `smtp.gmail.com`      | 465         | 587           |
| 企业微信邮箱 | `smtp.exmail.qq.com`  | 465         | 25            |

> 💡 推荐使用 SSL 加密端口（如 465），传输更安全，不易被拦截。

## 三、基础案例：发送纯文本邮件（每日英语句子推送）
以「每日一句英语」推送为例，实现纯文本邮件发送，适合作为英语学习提醒工具。

### 完整代码
```python
import smtplib
from email.mime.text import MIMEText
from email.header import Header
from datetime import datetime

def send_daily_english():
    # 1. 邮件配置信息
    smtp_server = "smtp.qq.com"       # SMTP 服务器地址（按自己邮箱修改）
    smtp_port = 465                   # SSL 端口
    sender_email = "your_qq@qq.com"   # 发件人邮箱
    sender_pass = "your_auth_code"    # 授权码（不是邮箱密码！）
    receiver_email = "recipient@example.com"  # 收件人邮箱

    # 2. 构建邮件内容（每日英语句子）
    today = datetime.now().strftime("%Y-%m-%d")
    english_sentence = "The only way to do great work is to love what you do. — Steve Jobs"
    chinese_trans = "成就伟大事业的唯一途径是热爱你所做的事。—— 史蒂夫·乔布斯"
    
    mail_content = f"""
    🌟 每日英语句子 ({today}) 🌟
    
    英文：{english_sentence}
    中文：{chinese_trans}
    
    🔍 生词解析：
    - recipient: 收件人 (n.)
    -途径：way (n.) / approach (n.)
    - 热爱：love (v.) / passion (n.)
    
    📌 今日学习建议：尝试用这个句子造一个自己的英语句子，回复邮件即可保存到你的学习记录~
    """

    # 3. 构建邮件对象（MIMEText 用于纯文本）
    # 参数1：邮件内容；参数2：格式（plain=纯文本）；参数3：编码（utf-8 避免中文乱码）
    msg = MIMEText(mail_content, "plain", "utf-8")
    
    # 设置邮件主题（Header 处理中文主题，避免乱码）
    msg["Subject"] = Header(f"每日英语学习 · {today}", "utf-8")
    # 设置发件人（格式："显示名称" <邮箱地址>）
    msg["From"] = Header("英语学习助手", "utf-8") + f" <{sender_email}>"
    # 设置收件人
    msg["To"] = Header("英语学习者", "utf-8") + f" <{receiver_email}>"

    try:
        # 4. 连接 SMTP 服务器并发送邮件
        # SMTP_SSL：使用 SSL 加密连接（推荐）
        with smtplib.SMTP_SSL(smtp_server, smtp_port) as server:
            # 登录邮箱
            server.login(sender_email, sender_pass)
            # 发送邮件（from_addr, to_addrs, msg.as_string()）
            server.sendmail(
                sender_email,          # 发件人邮箱
                receiver_email,        # 收件人邮箱（可多个，用列表：[a@xx.com, b@xx.com]）
                msg.as_string()        # 邮件内容转为字符串
            )
        print(f"✅ 每日英语邮件发送成功！收件人：{receiver_email}")

    except Exception as e:
        print(f"❌ 邮件发送失败：{str(e)}")

if __name__ == "__main__":
    send_daily_english()
```

### 代码关键说明
- `Header` 类：处理中文主题和发件人/收件人名称，避免乱码
- `SMTP_SSL`：直接建立加密连接，比 `SMTP()` + `starttls()` 更简洁
- `with` 语句：自动关闭 SMTP 连接，无需手动调用 `server.quit()`
- 邮件内容可动态生成（如从数据库读取每日句子，结合之前的 MySQL 教程）

## 五、高级案例：发送带附件的邮件（英语资料推送）
发送英语学习附件（如单词表 Excel、PDF 教程、听力音频），适合分享学习资源。

### 完整代码
```python
import smtplib
import os
from email.mime.text import MIMEText
from email.mime.multipart import MIMEMultipart
from email.mime.base import MIMEBase
from email import encoders
from email.header import Header

def send_english_material():
    # 1. 邮件配置
    smtp_server = "smtp.qq.com"
    smtp_port = 465
    sender_email = "your_qq@qq.com"
    sender_pass = "your_auth_code"
    receiver_email = "learner@example.com"

    # 2. 构建多部分邮件（MIMEMultipart 支持文本+附件）
    msg = MIMEMultipart()
    msg["Subject"] = Header("英语学习资料推送 · 中级词汇表+听力素材", "utf-8")
    msg["From"] = Header("英语资源助手", "utf-8") + f" <{sender_email}>"
    msg["To"] = Header("英语学习者", "utf-8") + f" <{receiver_email}>"

    # 3. 邮件正文（纯文本或 HTML）
    text_content = """
    亲爱的学习者：
    
    本次为你推送中级英语学习资料，包含以下内容：
    1. 中级核心词汇表（Excel 格式，含释义和例句）
    2. 日常对话听力素材（MP3 格式，共 10 篇）
    
    建议学习计划：
    - 每日记忆 20 个核心词汇，结合例句理解
    - 每日听 1 篇听力素材，跟读模仿发音
    - 周末整理错题，回复邮件可获取个性化学习建议
    
    祝你学习进步！
    —— 英语学习助手
    """
    msg.attach(MIMEText(text_content, "plain", "utf-8"))

    # 4. 添加附件（支持多个文件）
    def add_attachment(file_path):
        """添加单个附件的工具函数"""
        if not os.path.exists(file_path):
            print(f"⚠️  附件文件不存在：{file_path}")
            return
        
        # 创建附件对象
        part = MIMEBase("application", "octet-stream")
        # 读取文件内容
        with open(file_path, "rb") as f:
            part.set_payload(f.read())
        # 编码为 base64（邮件传输标准）
        encoders.encode_base64(part)
        # 设置附件头（文件名用 Header 处理中文，避免乱码）
        file_name = os.path.basename(file_path)
        part.add_header(
            "Content-Disposition",
            f"attachment; filename={Header(file_name, 'utf-8').encode()}"
        )
        # 将附件添加到邮件
        msg.attach(part)
        print(f"✅ 已添加附件：{file_name}")

    # 调用函数添加附件（替换为你的文件路径）
    add_attachment("中级英语核心词汇表.xlsx")
    add_attachment("日常对话听力素材.zip")

    # 5. 发送邮件
    try:
        with smtplib.SMTP_SSL(smtp_server, smtp_port) as server:
            server.login(sender_email, sender_pass)
            server.sendmail(sender_email, receiver_email, msg.as_string())
        print(f"\n🎉 带附件的英语资料邮件发送成功！")
    except Exception as e:
        print(f"❌ 邮件发送失败：{str(e)}")

if __name__ == "__main__":
    send_english_material()
```

### 关键说明
- `MIMEMultipart`：用于组合多部分内容（正文+附件）
- `MIMEBase` + `encoders.encode_base64`：处理二进制附件，确保传输兼容
- 中文文件名需用 `Header` 编码，避免接收方显示乱码
- 支持多种附件格式（Excel、PDF、MP3、ZIP 等）

## 六、扩展功能：批量发送与抄送/密送
### 1. 批量发送邮件（给多个学习者推送）

def send_batch_english():

## 七、常见问题与解决方案
### 1. 授权码错误：`smtplib.SMTPAuthenticationError`
- 检查授权码是否正确（不是邮箱登录密码！）
- 重新生成授权码（部分邮箱授权码有有效期）
- 确认 SMTP 服务已开启（按「准备工作」步骤重新检查）

### 2. 端口/服务器错误：`smtplib.SMTPServerDisconnected`
- 核对 SMTP 服务器地址和端口（参考「配置表」）
- 确保使用 SSL 端口时用 `SMTP_SSL()`，非 SSL 端口用 `SMTP()`
- 部分网络（如公司内网）禁用 25 端口，优先使用 465 端口

### 3. 中文乱码问题
- 邮件内容、主题、附件名均需用 `utf-8` 编码
- 中文主题和名称必须用 `Header` 类处理
- 避免在 HTML 中使用特殊中文字体，优先用系统默认字体

### 4. 邮件被判定为垃圾邮件
- 邮件内容避免大量广告、敏感词汇
- 发件人名称和邮箱保持一致，不要频繁更换
- 控制发送频率，批量发送时添加间隔（如 `time.sleep(1)`）
- 收件人列表中避免无效邮箱

### 5. 超时错误：`smtplib.SMTPServerTimeoutError`
- 检查网络连接是否正常
- 增加超时参数：`smtplib.SMTP_SSL(smtp_server, smtp_port, timeout=30)`
- 更换更稳定的 SMTP 服务器（如 QQ 邮箱换 163 邮箱）

## 八、学习小结
Python3 SMTP 发送邮件的核心流程可总结为：  
**开启 SMTP 服务 → 构建邮件内容（文本/HTML/附件）→ 建立加密连接 → 登录发送 → 关闭连接**

### 适用场景
- 英语学习提醒（每日单词、句子推送）
- 学习资料分享（单词表、语法 PDF、听力素材）
- 学习报告发送（成绩、进度统计）
- 批量通知（课程安排、作业布置）

### 进阶建议
1. 结合 MySQL 数据库：存储收件人列表、学习资料路径，实现动态发送
2. 定时发送：用 `schedule` 库（`pip install schedule`）设置每日/每周自动推送
3. 接收回复：结合 POP3/IMAP 协议（`poplib`/`imaplib` 模块）读取收件人回复，实现互动
4. 模板化邮件：将 HTML 邮件内容保存为模板文件，动态替换变量（如学习者姓名、今日主题）
5. 错误重试：添加异常重试机制（如 `tenacity` 库），提高发送成功率
