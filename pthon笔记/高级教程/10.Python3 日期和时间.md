## 一、核心概念与应用场景
### 1. 日期时间的核心术语
在Python中处理日期时间，需先明确几个关键概念，结合英语学习场景理解更直观：
- **日期（Date）**：年、月、日（如 `2025-11-20`，对应英语学习计划的“每日任务日期”）。
- **时间（Time）**：时、分、秒、微秒（如 `19:30:00`，对应“晚上7点半开始英语听力练习”）。
- **日期时间（Datetime）**：日期+时间（如 `2025-11-20 19:30:00`，完整的学习记录时间戳）。
- **时间戳（Timestamp）**：从1970-01-01 00:00:00 UTC到当前时间的秒数（如 `1755732600`，用于学习记录排序）。
- **时间差（Timedelta）**：两个日期时间的间隔（如 `2小时30分钟`，对应英语学习的“累计时长”）。
- **时区（Timezone）**：不同地区的时间偏移（如 UTC+8 北京时间，用于解析国外英语新闻的发布时间）。

### 2. Python 处理日期时间的核心模块
Python3 内置 3 个核心模块，无需第三方依赖，覆盖绝大多数场景：

| 模块名       | 核心功能                          | 适用场景                                  |
|--------------|-----------------------------------|-------------------------------------------|
| `datetime`   | 日期/时间的创建、运算、格式化（推荐） | 学习计划生成、学习时长统计、时间戳转换    |
| `time`       | 时间戳、睡眠、底层时间操作         | 定时执行英语学习任务、程序延时            |
| `calendar`   | 日历生成、星期/月份计算            | 月度英语学习计划、统计每周学习天数        |

### 3. 英语学习场景对应
- 生成每日英语句子推送的日期（`datetime.date`）。
- 记录单词记忆、听力练习的精确时间（`datetime.datetime`）。
- 计算累计学习时长（如“本周听力共练习 5.5 小时”，`timedelta`）。
- 解析英语新闻、学习资料的发布时间（`strptime` 字符串转日期）。
- 定时发送学习提醒（`time.sleep` 或 `datetime` 结合多线程）。
- 生成月度学习日历（`calendar` 模块）。

## 二、基础入门：`datetime` 模块（核心）
`datetime` 模块是处理日期时间的首选，提供 `date`（日期）、`time`（时间）、`datetime`（日期时间）、`timedelta`（时间差）四个核心类，用法简洁且功能强大。

### 1. 核心类：创建日期/时间对象
#### （1）`datetime.date`：处理日期（年、月、日）
```python
from datetime import date

# 1. 创建指定日期（年, 月, 日）
study_date = date(2025, 11, 20)
print("📅 英语学习日期：", study_date)  # 输出：2025-11-20

# 2. 获取当前日期
today = date.today()
print("📅 今日日期：", today)  # 输出：2025-11-19（根据实际日期）

# 3. 访问日期组件（年、月、日、星期）
print("年份：", today.year)      # 2025
print("月份：", today.month)     # 11
print("日期：", today.day)       # 19
print("星期（0=周一，6=周日）：", today.weekday())  # 2（对应周三）
print("星期（1=周一，7=周日）：", today.isoweekday())  # 3（国际标准）

# 英语学习场景：判断今天是否是学习日（周一到周五）
if today.weekday() < 5:
    print("✅ 今日是学习日，建议学习 1 小时英语！")
else:
    print("📌 周末可适当放松，推荐阅读一篇英语短文～")
```

#### （2）`datetime.time`：处理时间（时、分、秒、微秒）
```python
from datetime import time

# 1. 创建指定时间（时, 分, 秒, 微秒）
listen_time = time(19, 30, 0)  # 晚上7点30分
print("⏰ 听力练习时间：", listen_time)  # 输出：19:30:00

# 2. 访问时间组件
print("小时：", listen_time.hour)   # 19
print("分钟：", listen_time.minute) # 30
print("秒：", listen_time.second)   # 0

# 英语学习场景：根据时间安排任务
def arrange_task(current_time):
    if time(8, 0) <= current_time < time(12, 0):
        return "📚 上午适合记忆单词"
    elif time(14, 0) <= current_time < time(17, 0):
        return "📖 下午适合语法学习"
    elif time(19, 0) <= current_time < time(21, 0):
        return "🎧 晚上适合听力练习"
    else:
        return "💤 休息时间，可听英语睡前故事"

# 模拟当前时间为15:30
current_time = time(15, 30)
print("今日任务建议：", arrange_task(current_time))  # 输出：📖 下午适合语法学习
```

#### （3）`datetime.datetime`：处理日期+时间（最常用）
```python
from datetime import datetime

# 1. 创建指定日期时间
study_record = datetime(2025, 11, 19, 19, 45, 30)
print("📝 学习记录时间：", study_record)  # 输出：2025-11-19 19:45:30

# 2. 获取当前日期时间（本地时间）
now = datetime.now()
print("⏱️ 当前本地时间：", now)  # 输出：2025-11-19 15:20:30.123456（含微秒）

# 3. 获取当前 UTC 时间（全球统一时间）
utc_now = datetime.utcnow()
print("🌍 当前 UTC 时间：", utc_now)  # 输出：2025-11-19 07:20:30.123456

# 4. 访问组件（整合 date 和 time 的所有属性）
print("年：", now.year, "月：", now.month, "日：", now.day)
print("时：", now.hour, "分：", now.minute, "秒：", now.second)
```

### 2. 格式化：日期时间 ↔ 字符串（核心技能）
日期时间对象无法直接拼接字符串，需通过 `strftime()`（格式化输出）和 `strptime()`（解析输入）实现与字符串的互转，**格式符是关键**。

#### （1）`strftime()`：日期时间 → 字符串（格式化输出）
```python
from datetime import datetime

now = datetime.now()

# 常用格式符（结合英语学习场景示例）
print("格式化日期（年-月-日）：", now.strftime("%Y-%m-%d"))  # 2025-11-19
print("格式化时间（时:分:秒）：", now.strftime("%H:%M:%S"))  # 15:25:40
print("完整日期时间：", now.strftime("%Y-%m-%d %H:%M:%S"))  # 2025-11-19 15:25:40
print("英语格式日期：", now.strftime("%B %d, %Y"))  # November 19, 2025（月份英文）
print("带星期的日期：", now.strftime("%A, %Y-%m-%d"))  # Wednesday, 2025-11-19（星期英文）

# 英语学习场景：生成学习记录标题
record_title = now.strftime("[%Y-%m-%d %H:%M] 英语学习记录")
print("📄 学习记录标题：", record_title)  # [2025-11-19 15:25] 英语学习记录
```

#### （2）`strptime()`：字符串 → 日期时间（解析输入）
常用于解析 API 返回的时间字符串、用户输入的学习时间等。
```python
from datetime import datetime

# 示例1：解析英语新闻发布时间字符串
news_pub_str = "November 18, 2025 08:30 AM"
pub_datetime = datetime.strptime(news_pub_str, "%B %d, %Y %I:%M %p")
print("📰 新闻发布时间（解析后）：", pub_datetime)  # 2025-11-18 08:30:00

# 示例2：解析用户输入的学习计划时间
plan_str = "2025-11-20 20:00"
plan_datetime = datetime.strptime(plan_str, "%Y-%m-%d %H:%M")
print("📅 学习计划时间（解析后）：", plan_datetime)  # 2025-11-20 20:00:00

# 英语学习场景：计算新闻发布到现在的时间差
time_diff = datetime.now() - pub_datetime
print("⌛ 新闻发布于", time_diff.days, "天前")  # 输出：1 天前
```

#### 常用格式符对照表（必记）
| 格式符 | 含义                  | 示例          |
|--------|-----------------------|---------------|
| `%Y`   | 4位年份               | 2025          |
| `%m`   | 2位月份（01-12）      | 11            |
| `%d`   | 2位日期（01-31）      | 19            |
| `%H`   | 24小时制小时（00-23） | 19            |
| `%I`   | 12小时制小时（01-12） | 07            |
| `%M`   | 2位分钟（00-59）      | 30            |
| `%S`   | 2位秒（00-59）        | 45            |
| `%B`   | 月份英文全名          | November      |
| `%b`   | 月份英文缩写          | Nov           |
| `%A`   | 星期英文全名          | Wednesday     |
| `%a`   | 星期英文缩写          | Wed           |
| `%p`   | 上/下午（AM/PM）      | PM            |

### 3. 时间运算：`timedelta` 时间差
用于计算两个日期时间的间隔，或给日期时间加减一段时间（如“3天后的学习计划”“累计学习2小时30分钟”）。

```python
from datetime import datetime, timedelta

now = datetime.now()

# 1. 计算两个日期时间的差
study_start = datetime(2025, 11, 19, 19, 0)
study_end = datetime(2025, 11, 19, 21, 30)
study_duration = study_end - study_start
print("⏳ 本次学习时长：", study_duration)  # 2:30:00
print("小时数：", study_duration.total_seconds() / 3600)  # 2.5 小时

# 2. 给日期时间加/减时间差
# 3天后的学习计划
future_plan = now + timedelta(days=3)
print("📅 3天后的学习计划日期：", future_plan.strftime("%Y-%m-%d"))

# 1小时20分钟后的听力练习
listen_plan = now + timedelta(hours=1, minutes=20)
print("⏰ 下次听力练习时间：", listen_plan.strftime("%H:%M"))

# 减少5天（查看5天前的学习记录）
past_record = now - timedelta(days=5)
print("📝 5天前的日期：", past_record.strftime("%Y-%m-%d"))

# 英语学习场景：计算本月剩余学习天数（假设每月目标20天）
today = date.today()
# 获取本月最后一天
last_day = date(today.year, today.month + 1, 1) - timedelta(days=1)
remaining_days = (last_day - today).days
print(f"📌 本月剩余 {remaining_days} 天，还需学习 {max(0, 20 - (today.day - 1))} 天")
```

`timedelta` 支持的参数：`days`（天）、`seconds`（秒）、`microseconds`（微秒）、`milliseconds`（毫秒）、`minutes`（分钟）、`hours`（小时）、`weeks`（周）。

## 三、其他核心模块：`time` 与 `calendar`
### 1. `time` 模块：时间戳与底层操作
```python
import time

# 1. 获取当前时间戳（秒数）
timestamp = time.time()
print("⌚ 当前时间戳：", timestamp)  # 1755732600.123456

# 时间戳 → datetime 对象
timestamp_datetime = datetime.fromtimestamp(timestamp)
print("时间戳转日期时间：", timestamp_datetime)

# 2. 时间戳 → 结构化时间（便于访问组件）
struct_time = time.localtime(timestamp)
print("结构化时间（年）：", struct_time.tm_year)
print("结构化时间（月）：", struct_time.tm_mon)

# 3. 睡眠（程序暂停，用于定时任务）
print("⏳ 3秒后将显示英语学习提醒...")
time.sleep(3)  # 暂停3秒
print("✅ 提醒：该学习英语单词了！")

# 英语学习场景：每隔2秒打印一次单词（模拟记忆复习）
words = ["perseverance", "consistent", "fluency"]
for word in words:
    print(f"📝 复习单词：{word}")
    time.sleep(2)
```

### 2. `calendar` 模块：日历与日期计算
```python
import calendar
from datetime import date

# 1. 生成月度日历（英语学习计划日历）
year = 2025
month = 11
print(f"📅 2025年11月英语学习日历：")
print(calendar.month(year, month))  # 打印完整日历

# 2. 判断闰年（2月学习计划需考虑）
print("2025年是否为闰年？", calendar.isleap(2025))  # False

# 3. 计算某个月的天数
days_in_nov = calendar.monthrange(year, month)[1]
print(f"11月共有 {days_in_nov} 天，可安排 {min(25, days_in_nov)} 天学习")

# 4. 获取指定日期的星期（0=周一，6=周日）
study_date = date(2025, 11, 20)
weekday = calendar.weekday(study_date.year, study_date.month, study_date.day)
weekday_map = ["周一", "周二", "周三", "周四", "周五", "周六", "周日"]
print(f"2025-11-20 是 {weekday_map[weekday]}，适合安排口语练习")
```

## 四、进阶：时区处理（Python3.9+ 推荐 `zoneinfo`）
不同地区的时间存在偏移（如英语新闻发布于 UTC 时间，需转为北京时间），Python3.9+ 内置 `zoneinfo` 模块，无需第三方依赖即可处理时区。

### 1. 时区转换基础
```python
from datetime import datetime
from zoneinfo import ZoneInfo  # Python3.9+ 内置（低版本需安装 pytz）

# 1. 创建带时区的日期时间（UTC 时间）
utc_datetime = datetime(2025, 11, 19, 8, 0, tzinfo=ZoneInfo("UTC"))
print("🌍 UTC 时间：", utc_datetime)  # 2025-11-19 08:00:00+00:00

# 2. 转换为北京时间（UTC+8）
beijing_datetime = utc_datetime.astimezone(ZoneInfo("Asia/Shanghai"))
print("🇨🇳 北京时间：", beijing_datetime)  # 2025-11-19 16:00:00+08:00

# 3. 转换为纽约时间（UTC-5）
new_york_datetime = utc_datetime.astimezone(ZoneInfo("America/New_York"))
print("🇺🇸 纽约时间：", new_york_datetime)  # 2025-11-19 03:00:00-05:00

# 英语学习场景：解析国外英语课程的直播时间（UTC）
live_utc_str = "2025-11-20 14:00:00 UTC"
live_utc_datetime = datetime.strptime(live_utc_str, "%Y-%m-%d %H:%M:%S %Z").replace(tzinfo=ZoneInfo("UTC"))
live_beijing_datetime = live_utc_datetime.astimezone(ZoneInfo("Asia/Shanghai"))
print(f"📺 英语直播 UTC 时间：{live_utc_datetime.strftime('%Y-%m-%d %H:%M')}")
print(f"📺 英语直播北京时间：{live_beijing_datetime.strftime('%Y-%m-%d %H:%M')}")
```

### 2. 常见时区名称（参考）
- 北京时间：`Asia/Shanghai`
- UTC 时间：`UTC`
- 纽约时间：`America/New_York`
- 伦敦时间：`Europe/London`
- 东京时间：`Asia/Tokyo`

> 💡 低版本 Python（<3.9）解决方案：安装 `pytz` 库（`pip install pytz`），用法类似但语法略有差异。

## 五、实战案例：英语学习场景应用
### 案例 1：每日英语句子定时推送（结合 SMTP 教程）
```python
from datetime import datetime, timedelta
import smtplib
from email.mime.text import MIMEText
from email.header import Header
import time

def send_daily_english():
    # 邮件配置（替换为自己的邮箱信息）
    smtp_server = "smtp.qq.com"
    smtp_port = 465
    sender_email = "your_qq@qq.com"
    sender_pass = "your_auth_code"
    receiver_email = "your_target@example.com"

    # 每日英语句子（可从数据库/文件读取）
    today = datetime.now().strftime("%Y-%m-%d")
    sentences = [
        ("The only limit to our realization of tomorrow is our doubts of today.", "实现明天理想的唯一障碍是今天的疑虑。"),
        ("Learning is a lifelong journey.", "学习是一场终身的旅程。"),
        ("Practice makes perfect.", "熟能生巧。")
    ]
    english_sent, chinese_trans = sentences[datetime.now().day % len(sentences)]

    # 构建邮件内容
    mail_content = f"""
    🌟 每日英语句子 ({today}) 🌟
    英文：{english_sent}
    中文：{chinese_trans}
    📌 今日任务：用这个句子造一个英语句子，回复邮件保存学习记录～
    """

    # 发送邮件（代码来自 SMTP 教程）
    msg = MIMEText(mail_content, "plain", "utf-8")
    msg["Subject"] = Header(f"每日英语学习 · {today}", "utf-8")
    msg["From"] = Header("英语学习助手", "utf-8") + f" <{sender_email}>"
    msg["To"] = Header("英语学习者", "utf-8") + f" <{receiver_email}>"

    try:
        with smtplib.SMTP_SSL(smtp_server, smtp_port) as server:
            server.login(sender_email, sender_pass)
            server.sendmail(sender_email, receiver_email, msg.as_string())
        print(f"✅ {today} 英语句子推送成功！")
    except Exception as e:
        print(f"❌ 推送失败：{e}")

# 定时推送：每天早上8点发送（结合循环和 sleep）
def timed_push():
    while True:
        now = datetime.now()
        # 目标时间：早上8点
        target_time = now.replace(hour=8, minute=0, second=0, microsecond=0)
        # 如果当前时间已过8点，目标时间设为明天8点
        if now > target_time:
            target_time += timedelta(days=1)
        # 计算等待时间（秒）
        wait_seconds = (target_time - now).total_seconds()
        print(f"⌛ 距离下次推送还有 {wait_seconds / 3600:.1f} 小时")
        time.sleep(wait_seconds)
        # 发送邮件
        send_daily_english()

if __name__ == "__main__":
    timed_push()
```

### 案例 2：统计每周英语学习时长
```python
from datetime import datetime, timedelta
import json

# 模拟学习记录（实际可从 JSON/MySQL 读取）
study_records = [
    {"start": "2025-11-18 19:00", "end": "2025-11-18 20:30"},
    {"start": "2025-11-17 15:00", "end": "2025-11-17 16:10"},
    {"start": "2025-11-15 20:00", "end": "2025-11-15 21:20"},
    {"start": "2025-11-12 10:00", "end": "2025-11-12 11:00"},  # 上周的记录
]

def calculate_weekly_duration(records):
    """统计本周（周一到当前）的学习时长"""
    now = datetime.now()
    # 获取本周一的日期（当前日期 - 星期数（0=周一））
    monday = now - timedelta(days=now.weekday())
    monday_start = datetime(monday.year, monday.month, monday.day, 0, 0, 0)
    
    total_duration = 0.0
    weekly_records = []

    for record in records:
        # 解析开始和结束时间
        start = datetime.strptime(record["start"], "%Y-%m-%d %H:%M")
        end = datetime.strptime(record["end"], "%Y-%m-%d %H:%M")
        
        # 筛选本周的记录
        if start >= monday_start:
            duration = (end - start).total_seconds() / 3600  # 转为小时
            total_duration += duration
            weekly_records.append({
                "date": start.strftime("%Y-%m-%d"),
                "duration": round(duration, 1)
            })
    
    # 生成统计报告
    report = f"""
    📊 本周英语学习时长统计（{monday_start.strftime('%Y-%m-%d')} 至今）
    { '-' * 40 }
    总时长：{round(total_duration, 1)} 小时
    每日记录：
    """
    for rec in weekly_records:
        report += f"  - {rec['date']}：{rec['duration']} 小时\n"
    
    # 目标检查（假设每周目标 5 小时）
    target = 5.0
    remaining = max(0, target - total_duration)
    report += f"\n🎯 本周目标：{target} 小时 | 剩余需学习：{remaining} 小时"
    return report

if __name__ == "__main__":
    print(calculate_weekly_duration(study_records))
```

        "published_at": "2025-11-20T09:00:00Z",
        "difficulty": "hard"
    }
]

def sort_news_by_publish_time(news_list):
    """按发布时间排序（最新在前），并转换为北京时间"""
    sorted_news = []
    for news in news_list:
        # 解析 UTC 时间字符串（ISO 格式：2025-11-19T08:30:00Z）
        utc_datetime = datetime.fromisoformat(news["published_at"].replace("Z", "+00:00"))
        # 转换为北京时间
        beijing_datetime = utc_datetime.astimezone(ZoneInfo("Asia/Shanghai"))
        # 添加本地时间和排序键（时间戳）
        news["published_at_beijing"] = beijing_datetime.strftime("%Y-%m-%d %H:%M")
        news["sort_key"] = beijing_datetime.timestamp()
        sorted_news.append(news)
    
    # 按时间戳降序排序（最新在前）
    sorted_news.sort(key=lambda x: x["sort_key"], reverse=True)
    return sorted_news

if __name__ == "__main__":
    sorted_news = sort_news_by_publish_time(news_data)
    print("📰 按发布时间排序的英语新闻（最新在前）：")
    for idx, news in enumerate(sorted_news, 1):
        print(f"\n{idx}. 标题：{news['title']}")
        print(f"   难度：{news['difficulty']}")
        print(f"   发布时间（北京时间）：{news['published_at_beijing']}")

## 六、常见问题与解决方案
### 1. 时区转换错误（最易踩坑）
- 问题：直接给 `datetime` 对象添加时区导致偏移错误，或解析无时区字符串时默认本地时间。
- 解决方案：
  - 解析 UTC 时间字符串时，显式指定 `tzinfo=ZoneInfo("UTC")`。
  - 转换时区必须用 `astimezone()` 方法，而非直接修改 `tzinfo` 属性。
  - 示例：
    ```python
    # 错误：直接修改 tzinfo
    wrong_dt = datetime(2025, 11, 19, 8, 0).replace(tzinfo=ZoneInfo("Asia/Shanghai"))
    # 正确：先创建 UTC 时间，再转换
    correct_dt = datetime(2025, 11, 19, 8, 0, tzinfo=ZoneInfo("UTC")).astimezone(ZoneInfo("Asia/Shanghai"))
    ```

### 2. 格式化字符串错误（`ValueError`）
- 问题：`strftime()` 或 `strptime()` 中格式符与字符串不匹配（如用 `%Y` 解析两位数年份）。
- 解决方案：
  - 严格对应格式符与字符串结构（如字符串是 `25-11-19`，需用 `%y-%m-%d` 而非 `%Y-%m-%d`）。
  - 不确定时用在线工具验证（如 [strftime 验证工具](https://strftime.org/)）。
  - 示例：
    ```python
    # 错误：格式符与字符串不匹配
    datetime.strptime("25-11-19", "%Y-%m-%d")  # ValueError
    # 正确：用 %y 解析两位数年份
    datetime.strptime("25-11-19", "%y-%m-%d")  # 2025-11-19
    ```

### 3. 跨时区时间计算错误
- 问题：计算两个不同时区的时间差时，未统一时区导致结果偏差。
- 解决方案：先将所有时间转换为同一时区（如 UTC），再进行运算。
  ```python
  dt1 = datetime(2025, 11, 19, 8, 0, tzinfo=ZoneInfo("Asia/Shanghai"))
  dt2 = datetime(2025, 11, 19, 0, 0, tzinfo=ZoneInfo("America/New_York"))
  # 统一转为 UTC 后计算差值
  dt1_utc = dt1.astimezone(ZoneInfo("UTC"))
  dt2_utc = dt2.astimezone(ZoneInfo("UTC"))
  time_diff = dt1_utc - dt2_utc  # 正确差值
  ```

### 4. 闰年/闰月处理错误
- 问题：手动计算日期时未考虑闰年（如 2024 年 2 月有 29 天），导致日期越界。
- 解决方案：使用 `datetime` 或 `calendar` 模块的内置方法，避免手动计算：
  ```python
  # 正确：用 timedelta 计算下个月日期（自动处理闰月）
  feb_29_2024 = date(2024, 2, 29)
  next_month = feb_29_2024 + timedelta(days=31)
  print(next_month)  # 2024-03-31（自动跳过 2 月的特殊情况）
  ```

## 七、学习小结与进阶建议
### 核心知识点总结
1. **`datetime` 模块是核心**：

   - `timedelta`：时间差计算（学习时长、计划安排）。
2. **辅助模块**：
   - `time`：时间戳、程序睡眠（定时任务）。

3. **进阶技能**：时区处理（`zoneinfo`）、跨时区时间转换（英语学习资料解析）。

- 解析英语新闻、课程的发布时间。
- 学习记录排序与时间戳管理。
