# Python3 random 模块完全指南
random 模块是 Python 内置的 **伪随机数生成工具**，核心用于生成符合各种需求的随机数、随机选择、序列打乱等场景，适用于游戏开发、数据抽样、模拟测试、随机排序等场景。其底层基于梅森旋转算法（Mersenne Twister），生成的是“伪随机数”（可通过种子控制复现），**不适合加密场景**（加密需用 `secrets` 模块）。


## 一、模块简介与基础说明
- **无需安装**：Python 3.0+ 内置，直接 `import random` 即可使用；
- **伪随机特性**：默认以系统时间为种子（seed），每次运行生成不同序列；若手动设置种子，可复现相同随机结果（适合测试）；
- **核心定位**：满足日常“非加密”的随机需求，功能覆盖“基础随机数、序列操作、概率分布”三大类。


## 二、核心功能分类（含示例）
按使用场景分为 4 大类，涵盖 90% 高频用法，配合表格+代码示例快速上手：

### 1. 基础随机数生成（整数/浮点数）
生成指定范围的整数或浮点数，是最常用的基础功能。

| 函数                  | 功能描述                                  | 示例代码                          | 输出（示例）       |
|-----------------------|-------------------------------------------|-----------------------------------|--------------------|
| `random.random()`     | 生成 [0.0, 1.0) 左闭右开区间的浮点数       | `random.random()`                 | `0.3745401188473625` |
| `random.uniform(a, b)`| 生成 [a, b] 或 [b, a] 区间的均匀分布浮点数 | `random.uniform(1.5, 4.5)`        | `2.891739130479442` |
| `random.randint(a, b)`| 生成 [a, b] 闭区间的整数（等价于 `randrange(a, b+1)`） | `random.randint(1, 10)`       | `7`                |
| `random.randrange(start, stop=None, step=1)` | 生成 [start, stop) 区间、步长为 step 的整数 | `random.randrange(0, 10, 2)`（0-10偶数） | `6` |

#### 实用示例：生成随机验证码（6位数字）
```python
import random

code = ''.join(str(random.randint(0, 9)) for _ in range(6))
print("随机验证码：", code)  # 输出：639285（每次不同）
```


### 2. 序列随机操作（洗牌/选择/抽样）
针对列表、元组等序列的随机处理，是日常开发高频场景（如抽奖、随机排序）。

| 函数                  | 功能描述                                  | 示例代码                          | 输出（示例）       |
|-----------------------|-------------------------------------------|-----------------------------------|--------------------|
| `random.choice(seq)`  | 从非空序列中随机选择 1 个元素              | `random.choice(['a', 'b', 'c'])`  | `'b'`              |
| `random.shuffle(seq)` | 原地打乱序列（仅支持可变序列，如列表）     | `lst = [1,2,3]; random.shuffle(lst)` | `lst = [3,1,2]` |
| `random.sample(seq, k)` | 从序列中**无放回抽样** k 个元素（不重复）  | `random.sample([1,2,3,4,5], 3)`   | `[2,5,1]`          |
| `random.choices(seq, weights=None, k=1)` | 从序列中**有放回抽样** k 个元素（可重复），支持权重 | `random.choices(['a','b'], weights=[3,1], k=2)` | `['a','a']` |

#### 关键区别：`sample` vs `choices`
- `sample`：无放回，抽样数 `k` 不能超过序列长度（否则报错）；
- `choices`：有放回，抽样数 `k` 可任意，`weights` 参数可设置元素被选中的概率（如权重 `[3,1]` 表示 `'a'` 被选中概率是 `'b'` 的 3 倍）。

#### 实用示例：抽奖程序（从名单中抽 2 个幸运儿，1 个一等奖）
```python
import random

name_list = ["张三", "李四", "王五", "赵六", "孙七"]
# 无放回抽 2 个幸运儿（不重复）
lucky_ones = random.sample(name_list, 2)
# 从幸运儿中抽 1 个一等奖（有放回，也可用 choice）
first_prize = random.choice(lucky_ones)

print("幸运儿：", lucky_ones)      # 输出：['李四', '孙七']（示例）
print("一等奖：", first_prize)    # 输出：'孙七'（示例）
```


### 3. 概率分布随机数（模拟真实场景）
生成符合特定概率分布的随机数，适用于模拟、统计分析等场景（如身高分布、时间间隔）。

| 函数                  | 分布类型                                  | 功能描述                                  | 示例场景                  |
|-----------------------|-------------------------------------------|-------------------------------------------|---------------------------|
| `random.gauss(mu, sigma)` | 正态分布（高斯分布） | `mu`=均值，`sigma`=标准差，生成符合正态分布的浮点数 | 模拟身高、成绩、误差分布  |
| `random.expovariate(lambd)` | 指数分布 | `lambd`=速率参数（λ>0），生成非负浮点数 | 模拟服务响应时间、设备故障间隔 |
| `random.triangular(low, high, mode)` | 三角分布 | `low`=最小值，`high`=最大值，`mode`=众数 | 模拟用户停留时间、产品销量 |
| `random.betavariate(alpha, beta)` | Beta 分布 | 生成 [0,1] 区间浮点数，适用于概率的概率分布 | 模拟转化率、点击率分布    |

#### 实用示例：模拟班级学生成绩（正态分布，均值 75，标准差 10）
```python
import random

# 生成 50 名学生的成绩（正态分布，取整到 0-100）
scores = [int(random.gauss(mu=75, sigma=10)) for _ in range(50)]
# 修正超出 0-100 的成绩
scores = [max(0, min(100, s)) for s in scores]

print("学生成绩分布：", scores[:10])  # 输出前 10 个：[82, 78, 65, 90, 72, 68, 85, 70, 79, 83]（示例）
```


### 4. 种子与状态控制（可复现随机序列）
random 模块生成的是“伪随机数”，其序列由“种子（seed）”决定——**相同种子会生成完全相同的随机序列**，适用于测试、复现场景。

#### （1）设置种子（`random.seed()`）
```python
import random

# 设置种子为 10（整数、字符串、字节等均可）
random.seed(10)
print(random.randint(1, 100))  # 输出：74（固定）
print(random.choice(['a', 'b', 'c']))  # 输出：'a'（固定）

# 重置种子为 10，重新生成相同序列
random.seed(10)
print(random.randint(1, 100))  # 输出：74（与之前一致）
print(random.choice(['a', 'b', 'c']))  # 输出：'a'（与之前一致）
```

#### （2）保存/恢复随机状态（`random.getstate()`/`random.setstate()`）
用于中断后继续生成相同的随机序列（比种子更灵活）：
```python
import random

# 生成一个随机数，保存当前状态
random.seed(20)
print(random.randint(1, 10))  # 输出：3
state = random.getstate()  # 保存状态

# 继续生成几个随机数
print(random.randint(1, 10))  # 输出：10
print(random.randint(1, 10))  # 输出：6

# 恢复状态，从保存点继续生成
random.setstate(state)
print(random.randint(1, 10))  # 输出：10（与保存后第一次生成一致）
print(random.randint(1, 10))  # 输出：6（与保存后第二次生成一致）
```


## 三、常见问题与避坑指南
### 1. 伪随机数不能用于加密/安全场景
- 问题：random 模块的随机数可通过种子复现，安全性低；
- 解决方案：加密场景（如生成密码、令牌）用 `secrets` 模块（Python 3.6+ 内置）：
  ```python
  import secrets

  # 生成 8 位随机密码（字母+数字）
  password = ''.join(secrets.choice('abcdefghijklmnopqrstuvwxyz0123456789') for _ in range(8))
  print(password)  # 输出：x7k2p9z3（不可复现，更安全）
  ```

### 2. `shuffle()` 原地修改列表，不可用于元组
- 问题：`random.shuffle(seq)` 会直接修改原序列，且仅支持可变序列（列表），元组/字符串会报错；
- 解决方案：元组/字符串需先转为列表，或用 `sample(seq, k=len(seq))` 生成新序列：
  ```python
  tup = (1, 2, 3)
  # 错误：TypeError: 'tuple' object does not support item assignment
  # random.shuffle(tup)

  # 正确：用 sample 生成打乱后的新列表
  shuffled_tup = random.sample(tup, k=len(tup))
  print(shuffled_tup)  # 输出：[3,1,2]
  ```

### 3. `sample()` 抽样数 k 不能超过序列长度
- 问题：`random.sample(seq, k)` 中 k 若大于 `len(seq)`，会抛出 `ValueError`；
- 解决方案：需抽样数大于序列长度时，用 `random.choices()`（有放回抽样）。

### 4. 多线程环境下的随机数安全
- 问题：random 模块的全局函数（如 `random.randint`）不是线程安全的，多线程同时调用可能导致生成重复序列；
- 解决方案：每个线程使用独立的 `random.Random` 实例：
  ```python
  import random
  import threading

  def thread_func(seed):
      # 每个线程创建独立的 Random 实例
      r = random.Random(seed)
      print(f"线程 {threading.current_thread().name}：{r.randint(1, 100)}")

  # 启动两个线程，各自独立生成随机数
  threading.Thread(target=thread_func, args=(10,), name="Thread1").start()
  threading.Thread(target=thread_func, args=(20,), name="Thread2").start()
  ```


## 四、核心函数速查表（快速检索）
| 功能需求                | 推荐函数                          |
|-------------------------|-----------------------------------|
| 生成 0-1 浮点数         | `random.random()`                 |
| 生成指定范围整数        | `random.randint(a, b)`            |
| 生成指定范围浮点数      | `random.uniform(a, b)`            |
| 从序列选 1 个元素       | `random.choice(seq)`              |
| 打乱序列（原地）        | `random.shuffle(seq)`             |
| 无放回抽样 k 个元素     | `random.sample(seq, k)`           |
| 有放回抽样 k 个元素     | `random.choices(seq, k=k)`        |
| 模拟正态分布数据        | `random.gauss(mu, sigma)`         |
| 复现随机序列            | `random.seed(seed)`               |


## 五、总结
random 模块是 Python 随机处理的“瑞士军刀”，核心用法可总结为：
1. 基础随机数：`randint`（整数）、`random`/`uniform`（浮点数）；
2. 序列操作：`choice`（选1个）、`shuffle`（打乱）、`sample`（无放回抽样）、`choices`（有放回抽样）；
3. 高级场景：概率分布（`gauss` 等）、种子控制（`seed`）、线程安全（独立 `Random` 实例）；
4. 避坑关键：不用于加密（用 `secrets`）、`shuffle` 仅支持列表、抽样数不超序列长度。

掌握以上内容，即可覆盖日常开发中 95% 以上的随机处理场景。
