## 一、什么是 mysql-connector？
`mysql-connector` 是 MySQL 官方提供的 Python 驱动程序，用于在 Python 代码中连接和操作 MySQL 数据库。它完全由 Python 编写，无需依赖其他第三方库，支持 MySQL 5.1+ 版本，提供了简洁的 API 和完整的数据库操作功能，是 Python 开发者与 MySQL 交互的常用工具。

### 核心优势
- 官方维护，兼容性强，更新及时
- 纯 Python 实现，跨平台（Windows/macOS/Linux）
- 支持事务、批量操作、参数化查询等核心功能
- 语法简洁，学习成本低，适合初学者

## 三、连接 MySQL 数据库
### 1. 基础连接模板
```python
import mysql.connector
from mysql.connector import Error

try:
    # 建立数据库连接
    connection = mysql.connector.connect(
        host='localhost',        # 数据库地址（本地为 localhost）
        database='english_db',   # 数据库名（需提前创建）
        user='root',             # 数据库用户名
        password='your_password' # 数据库密码
    )

    if connection.is_connected():
        db_info = connection.get_server_info()
        print(f"Connected to MySQL Server version {db_info}")
        cursor = connection.cursor()  # 创建游标对象（用于执行SQL）
        cursor.execute("select database();")
        record = cursor.fetchone()
        print(f"You're connected to database: {record[0]}")

except Error as e:
    print(f"Error while connecting to MySQL: {e}")

finally:
    # 关闭连接（重要！避免资源泄露）
    if connection.is_connected():
        cursor.close()
        connection.close()
        print("MySQL connection is closed")
```

### 2. 关键配置参数
| 参数名       | 说明                                  | 示例值                  |
|--------------|---------------------------------------|-------------------------|
| `host`       | 数据库服务器地址                      | `localhost`/`127.0.0.1` |
| `database`   | 要连接的数据库名（需提前创建）        | `english_learning`      |
| `user`       | 数据库登录用户名                      | `root`                  |
| `password`   | 数据库登录密码                        | `123456`                |
| `port`       | 数据库端口（默认3306，无需修改）      | `3306`                  |
| `charset`    | 字符编码（建议设为 `utf8mb4` 支持 emoji） | `utf8mb4`           |

## 四、核心操作：CRUD（增删改查）
以「英语学习平台」的 `vocabulary`（生词表）为例，演示完整操作。  
### 先创建测试表（SQL）
```sql
CREATE TABLE vocabulary (
    id INT AUTO_INCREMENT PRIMARY KEY,
    word VARCHAR(50) NOT NULL,  # 英文单词
    definition TEXT,            # 释义
    example VARCHAR(200),       # 例句
    difficulty ENUM('easy', 'medium', 'hard') DEFAULT 'medium',  # 难度等级
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP  # 添加时间
);
```

### 1. 插入数据（Create）
#### 单条插入
```python
try:
    connection = mysql.connector.connect(
        host='localhost',
        database='english_db',
        user='root',
        password='your_password'
    )
    cursor = connection.cursor()

    # SQL 插入语句（参数化查询，避免 SQL 注入）
    insert_query = """
    INSERT INTO vocabulary (word, definition, example, difficulty)
    VALUES (%s, %s, %s, %s)
    """
    word_data = (
        "mindfulness",
        "The practice of being present and fully engaged with the current moment",
        "She practices mindfulness for 10 minutes every morning",
        "medium"
    )
    
    cursor.execute(insert_query, word_data)
    connection.commit()  # 提交事务（插入/更新/删除必须提交）
    print(f"New word inserted successfully! ID: {cursor.lastrowid}")

except Error as e:
    print(f"Error: {e}")
    connection.rollback()  # 出错时回滚事务

finally:
    if connection.is_connected():
        cursor.close()
        connection.close()
```

#### 批量插入（高效）
```python
# 批量插入多条生词
word_list = [
    ("intentional", "Done on purpose; deliberate", "He made an intentional choice to learn English", "medium"),
    ("meditation", "A practice to train attention and awareness", "Meditation helps improve focus", "hard"),
    ("gratitude", "The quality of being thankful", "Express gratitude every day", "easy")
]

try:
    connection = mysql.connector.connect(
        host='localhost', database='english_db', user='root', password='your_password'
    )
    cursor = connection.cursor()
    insert_query = """
    INSERT INTO vocabulary (word, definition, example, difficulty)
    VALUES (%s, %s, %s, %s)
    """
    
    cursor.executemany(insert_query, word_list)  # 批量执行
    connection.commit()
    print(f"{cursor.rowcount} words inserted successfully!")

except Error as e:
    print(f"Error: {e}")
    connection.rollback()

finally:
    # 关闭连接（同上）
```

### 2. 查询数据（Read）
#### 查询所有数据
```python
try:
    connection = mysql.connector.connect(
        host='localhost', database='english_db', user='root', password='your_password'
    )
    cursor = connection.cursor(dictionary=True)  # 返回字典格式（默认元组）

    # 查询所有中等难度的单词
    query = "SELECT word, definition, example FROM vocabulary WHERE difficulty = %s"
    cursor.execute(query, ('medium',))  # 参数传递（注意元组格式）
    
    results = cursor.fetchall()  # 获取所有结果
    print("Medium difficulty words:")
    for row in results:
        print(f"\nWord: {row['word']}")
        print(f"Definition: {row['definition']}")
        print(f"Example: {row['example']}")

except Error as e:
    print(f"Error: {e}")

finally:
    # 关闭连接（同上）
```

#### 常用查询方法
| 方法               | 说明                          |
|--------------------|-------------------------------|
| `fetchone()`       | 获取第一条结果（单次查询）    |
| `fetchall()`       | 获取所有结果（适合少量数据）  |
| `fetchmany(n)`     | 获取前 n 条结果（分页常用）   |
| `cursor.rowcount`  | 获取查询结果行数              |

### 3. 更新数据（Update）
```python
try:
```

## 五、高级特性
### 1. 事务处理
`mysql-connector` 默认开启事务，需手动 `commit()` 提交，`rollback()` 回滚：
```python
try:
    connection = mysql.connector.connect(
        host='localhost', database='english_db', user='root', password='your_password'
    )
    cursor = connection.cursor()

    # 事务操作：插入+更新
    query1 = "INSERT INTO vocabulary (word, definition) VALUES (%s, %s)"
    query2 = "UPDATE vocabulary SET difficulty = 'easy' WHERE word = %s"
    
    cursor.execute(query1, ('serene', 'Calm, peaceful, and untroubled'))
    cursor.execute(query2, ('serene',))
    
    connection.commit()  # 所有操作成功，提交
    print("Transaction completed successfully")

except Error as e:
    print(f"Transaction failed: {e}")
    connection.rollback()  # 出错时回滚所有操作

finally:
    # 关闭连接（同上）
```

### 2. 上下文管理器（自动关闭连接）
使用 `with` 语句，无需手动关闭连接和游标，更简洁：
```python
try:
    with mysql.connector.connect(
        host='localhost',
        database='english_db',
        user='root',
        password='your_password'
    ) as connection:
        with connection.cursor(dictionary=True) as cursor:
            query = "SELECT word FROM vocabulary WHERE difficulty = 'easy'"
            cursor.execute(query)
            results = cursor.fetchall()
            print("Easy words:", [row['word'] for row in results])

except Error as e:
    print(f"Error: {e}")
```

### 3. 处理大数据集（分页查询）
避免一次性加载大量数据，使用 `LIMIT` 分页：
```python
def get_words_by_page(page=1, page_size=10):
    offset = (page - 1) * page_size
    try:
        with mysql.connector.connect(
            host='localhost', database='english_db', user='root', password='your_password'
        ) as connection:
            with connection.cursor(dictionary=True) as cursor:
                query = """
                SELECT * FROM vocabulary 
                ORDER BY create_time DESC 
                LIMIT %s OFFSET %s
                """
                cursor.execute(query, (page_size, offset))
                return cursor.fetchall()
    except Error as e:
        print(f"Error: {e}")
        return []

# 获取第2页数据（每页5条）
page_2_words = get_words_by_page(page=2, page_size=5)
```

## 六、常见问题与解决方案
### 1. 连接失败 `Access denied for user`
- 检查用户名/密码是否正确
- 确认 MySQL 服务已启动（`systemctl start mysql` 或 服务面板启动）
- 若远程连接，需授权用户访问：`GRANT ALL ON english_db.* TO 'root'@'%' IDENTIFIED BY 'password';`

### 2. 字符编码乱码
- 连接时指定 `charset='utf8mb4'`
- 数据库表字符集设为 `utf8mb4`（支持中文和 emoji）

### 3. SQL 注入风险
- 禁止直接拼接字符串：`"INSERT INTO vocabulary VALUES ('" + word + "')"`（危险！）
- 必须使用参数化查询（`%s` 占位符），如本文示例

### 4. 游标对象复用
- 每次执行新的 SQL 前，若需保留之前的结果，需先 `fetchall()` 读取
- 建议一个游标对应一组相关操作，或执行新查询前重置游标

## 七、学习小结
`mysql-connector` 是 Python 操作 MySQL 的轻量高效工具，核心流程可总结为：  
**安装驱动 → 建立连接 → 创建游标 → 执行 SQL → 处理结果 → 关闭连接**  

适合场景：
- 小型/中型项目的数据库交互
- 英语学习平台、博客系统等数据存储需求
- 快速原型开发和学习实践

进阶建议：
- 结合 `configparser` 存储数据库配置（避免硬编码密码）
- 大型项目可使用 ORM 框架（如 SQLAlchemy）简化操作
- 学习 MySQL 索引优化，提升查询性能
