Python3 的基本数据类型是构建程序的基础，它们用于存储不同类型的数据，每种类型都有其独特的特性和用途。以下是 Python3 中最核心的基本数据类型，包含详细说明和示例：


### **1. 整数（int）**  
**定义**：没有小数点的数字，包括正整数、负整数和零。  
**特点**：  
- Python 整数没有大小限制（可表示极大或极小的整数）；  
- 支持十进制、二进制（`0b` 前缀）、八进制（`0o` 前缀）、十六进制（`0x` 前缀）表示。  


**示例**：  
```python
# 十进制（默认）
a = 100
b = -50

# 二进制（0b开头）
binary = 0b1010  # 对应十进制 10
print(binary)  # 输出：10

# 八进制（0o开头）
octal = 0o12  # 对应十进制 10
print(octal)  # 输出：10

# 十六进制（0x开头）
hexadecimal = 0xa  # 对应十进制 10
print(hexadecimal)  # 输出：10
```

**常用操作**：  
- 算术运算（`+`、`-`、`*`、`/`、`//`、`%`、`**`）；  
- 类型转换：`int(3.14)` → `3`（截断小数），`int("123")` → `123`（字符串转整数，字符串需为纯数字）。  


###** 2. 浮点数（float）**  
**定义**：带小数点的数字，或用科学计数法表示的数字（`e` 或 `E` 表示指数，如 `3e2` 即 `3×10²`）。  
**特点**：  
- 浮点数有精度限制（受计算机二进制存储方式影响，可能存在微小误差）；  
- 科学计数法常用于表示极大或极小的数。  


**示例**：  
```python
# 普通浮点数
x = 3.14
y = -0.001

# 科学计数法
sci1 = 2.5e3  # 2.5 × 10³ = 2500.0
sci2 = 1e-2   # 1 × 10⁻² = 0.01
print(sci1, sci2)  # 输出：2500.0 0.01
```

**注意**：浮点数的精度问题（避免直接比较）：  
```python
print(0.1 + 0.2)  # 输出：0.30000000000000004（而非 0.3）
```

**常用操作**：  
- 算术运算（与整数类似）；  
- 类型转换：`float(5)` → `5.0`，`float("3.14")` → `3.14`。  


### **3. 字符串（str）**  
**定义**：由字符组成的文本，用单引号（`'`）、双引号（`"`）或三引号（`'''` 或 `"""`）包裹。  
**特点**：  
- **不可变**：定义后不能直接修改单个字符（需通过切片或拼接生成新字符串）；  
- 支持索引（从 `0` 开始）和切片（截取子串）；  
- 三引号可用于表示多行字符串。  


**示例**：  
```python
# 单/双引号（单行字符串）
s1 = 'hello'
s2 = "Python"

# 三引号（多行字符串）
s3 = '''第一行
第二行
第三行'''
print(s3)
# 输出：
# 第一行
# 第二行
# 第三行

# 转义字符（表示特殊字符）
s4 = "He said: \"Hello!\""  # 转义双引号
s5 = "Line1\nLine2"  # \n 表示换行
print(s5)
# 输出：
# Line1
# Line2
```

**核心操作**：  
- **索引**：`s = "abc"; print(s[0])` → `'a'`（取第1个字符）；  
- **切片**：`s[1:3]` → `'bc'`（从索引1到2，左闭右开）；  
- **长度**：`len(s)` → 3（字符串长度）；  
- **常用方法**：  
  - `s.upper()` → 转大写；`s.lower()` → 转小写；  
  - `s.split(',')` → 按 `,` 分割为列表；  
  - `s.strip()` → 去除首尾空白字符。  


### **4. 布尔值（bool）**  
**定义**：表示“真”或“假”，仅两个值：`True`（真）和 `False`（假）（首字母必须大写）。  
**特点**：  
- 常用于条件判断（`if`、`while` 等语句）；  
- 支持逻辑运算（`and`、`or`、`not`）。  


**示例**：  
```python
# 直接赋值
is_true = True
is_false = False

# 比较运算的结果（返回布尔值）
print(5 > 3)  # 输出：True
print(2 == 2)  # 输出：True
print("a" == "b")  # 输出：False

# 逻辑运算
print(True and False)  # 输出：False（and：都真才真）
print(True or False)   # 输出：True（or：一真就真）
print(not True)        # 输出：False（not：取反）
```

**类型转换**：  
- 其他类型转布尔值时，`0`、`0.0`、空字符串（`""`）、空列表（`[]`）、`None` 等“空/零”值会转为 `False`；  
- 其余值（如 `1`、`"a"`、`[1]`）均转为 `True`。  
```python
print(bool(0))      # 输出：False
print(bool(""))     # 输出：False
print(bool([1,2]))  # 输出：True
```


### **5. 空值（None）**  
**定义**：表示“无值”或“空”，是一个特殊的常量（首字母大写）。  
**特点**：  
- 不同于 `0`、空字符串（`""`）或空列表（`[]`），它表示“不存在”；  
- 常用作函数的默认返回值（若函数无 `return` 语句，默认返回 `None`），或初始化变量（表示暂未赋值）。  


**示例**：  
```python
# 定义空值变量
empty_var = None

# 函数默认返回 None
def func():
    print("Hello")

result = func()
print(result)  # 输出：None（函数无 return，默认返回 None）

# 判断是否为 None（用 is 而非 ==）
print(empty_var is None)  # 输出：True
```


### **类型检查与转换**  
- **检查类型**：用 `type()` 或 `isinstance()`  
  ```python
  print(type(10))  # 输出：<class 'int'>
  print(isinstance(3.14, float))  # 输出：True（判断是否为指定类型）
  ```  

- **类型转换**：用 `int()`、`float()`、`str()`、`bool()` 等  
  ```python
  print(int("123"))    # 输出：123（字符串→整数）
  print(float(5))      # 输出：5.0（整数→浮点数）
  print(str(3.14))     # 输出："3.14"（浮点数→字符串）
  print(bool(""))      # 输出：False（空字符串→布尔值）
  ```  


### 总结  
Python3 的基本数据类型是：整数（`int`）、浮点数（`float`）、字符串（`str`）、布尔值（`bool`）、空值（`None`）。它们是构建复杂数据结构（如列表、字典）和逻辑的基础，理解每种类型的特性和操作是入门 Python 的关键。
