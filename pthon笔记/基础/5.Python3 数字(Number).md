在Python3中，**数字（Number）** 是最基础的数据类型之一，用于表示数值。Python3支持三种主要的数字类型：**整数（int）**、**浮点数（float）**、**复数（complex）**，此外布尔值（`bool`）本质上是整数的子类（`True`对应`1`，`False`对应`0`）。


### 一、整数（int）
整数是没有小数部分的数字，可正可负，且**Python3的整数没有大小限制**（理论上可无限大，仅受内存限制）。


#### 1. 基本表示
- 十进制：日常使用的数字（如`123`、`-456`）；  
- 二进制：以`0b`或`0B`开头（如`0b101`表示十进制的`5`）；  
- 八进制：以`0o`或`0O`开头（如`0o12`表示十进制的`10`）；  
- 十六进制：以`0x`或`0X`开头（如`0x1A`表示十进制的`26`）。  


#### 2. 示例
```python
# 十进制整数
a = 100
b = -200
print(a, type(a))  # 100 <class 'int'>

# 二进制（0b开头）
c = 0b1010  # 二进制1010 → 十进制10
print(c)    # 10

# 八进制（0o开头）
d = 0o17    # 八进制17 → 十进制15（1×8 + 7 = 15）
print(d)    # 15

# 十六进制（0x开头，0-9、a-f/A-F）
e = 0x1f    # 十六进制1f → 十进制31（1×16 + 15 = 31）
print(e)    # 31
```


### 二、浮点数（float）
浮点数是带小数部分的数字，也可通过**科学计数法**表示（用`e`或`E`表示10的幂）。


#### 1. 基本表示
- 直接带小数点：如`3.14`、`-0.001`；  
- 科学计数法：如`1.23e4`（表示`1.23×10⁴ = 12300`）、`5e-3`（表示`5×10⁻³ = 0.005`）。  


#### 2. 注意：浮点数的精度问题
由于计算机存储浮点数的方式（二进制近似），部分十进制小数无法精确表示，可能导致计算误差。

```python
print(0.1 + 0.2)  # 0.30000000000000004（而非0.3）
print(0.1 + 0.2 == 0.3)  # False（因精度误差）
```

**解决方法**：如需精确计算，可使用`decimal`模块（高精度小数）或`fractions`模块（分数）。


#### 3. 示例
```python
f = 3.14
g = -0.5
h = 1.2e3  # 1.2×10³ = 1200.0
i = 2e-2   # 2×10⁻² = 0.02

print(f, type(f))  # 3.14 <class 'float'>
print(h)           # 1200.0
print(i)           # 0.02
```


### 三、复数（complex）
复数由**实部**和**虚部**组成，虚部以字母`j`或`J`结尾（数学中常用`i`表示虚数单位，Python中用`j`）。


#### 1. 基本表示
格式：`实部 + 虚部j`，例如`3+4j`、`-2.5-1j`。  
- 实部：复数中的实数部分（如`3+4j`的实部是`3`）；  
- 虚部：复数中与`j`相乘的部分（如`3+4j`的虚部是`4`）。  


#### 2. 访问实部和虚部
通过复数的`real`属性获取实部，`imag`属性获取虚部。

```python
z1 = 3 + 4j
z2 = -1.5 - 2j

print(z1.real)  # 3.0（实部，返回float类型）
print(z1.imag)  # 4.0（虚部，返回float类型）
print(z2.real)  # -1.5
print(z2.imag)  # -2.0
```


#### 3. 复数运算
复数支持加减乘除等基本运算：
```python
z1 = 3 + 4j
z2 = 1 + 2j
print(z1 + z2)  # (4+6j)
print(z1 * z2)  # (3×1 - 4×2) + (3×2 + 4×1)j = (-5+10j)
```


### 四、布尔值（bool）
布尔值是特殊的整数类型（`bool`继承自`int`），仅包含两个值：`True`（真）和`False`（假）。  
- `True`等价于整数`1`；  
- `False`等价于整数`0`。  


#### 1. 示例
```python
print(True == 1)   # True
print(False == 0)  # True
print(True + 1)    # 2（True当作1参与运算）
print(False * 5)   # 0（False当作0参与运算）
```

布尔值主要用于逻辑判断（如条件语句、循环控制），后续会详细讲解。


### 五、数字类型的常用操作
#### 1. 类型转换
通过`int()`、`float()`、`complex()`函数实现类型转换（详见“数据类型转换”章节）：
```python
print(int(3.9))     # 3（浮点数转整数，截断小数）
print(float(10))    # 10.0（整数转浮点数）
print(complex(2))   # (2+0j)（整数转复数，虚部为0）
```


#### 2. 内置函数
Python提供了多个用于数字操作的内置函数：
- `abs(x)`：返回x的绝对值；  
- `round(x, n)`：将x四舍五入，保留n位小数（默认n=0）；  
- `max(x1, x2, ...)`：返回多个数中的最大值；  
- `min(x1, x2, ...)`：返回多个数中的最小值；  
- `pow(x, y)`：返回x的y次方（等价于`x** y`）。  


#### 3. 示例
```python
print(abs(-5))        # 5（绝对值）
print(round(3.1415, 2))  # 3.14（保留2位小数）
print(max(1, 3, 2))   # 3（最大值）
print(pow(2, 3))      # 8（2的3次方）
```


#### 4. `math`模块
如需更复杂的数学运算（如开方、三角函数等），可导入`math`模块：
```python
import math

print(math.sqrt(16))  # 4.0（开平方）
print(math.sin(math.pi/2))  # 1.0（正弦函数，pi是π）
print(math.floor(3.7))  # 3（向下取整）
```


### 六、总结
- **整数（int）**：无小数部分，支持多进制表示，无大小限制；  
- **浮点数（float）**：带小数部分，可能有精度误差，支持科学计数法；  
- **复数（complex）**：含实部和虚部（虚部以`j`结尾），用于复杂数学运算；  
- **布尔值（bool）**：`True`（1）和`False`（0），用于逻辑判断。  

掌握数字类型的特性和操作，是进行Python数值计算的基础。
