在Python3中，**字符串（String）** 是由字符组成的不可变序列，用于表示文本数据。字符串是Python中最常用的数据类型之一，支持丰富的操作和方法。


### 一、字符串的定义
字符串可以用**单引号（`'`）**、**双引号（`"`）** 或**三引号（`'''` 或 `"""`）** 定义，其中三引号可用于表示多行字符串。


#### 1. 基本定义方式
```python
# 单引号定义
s1 = 'hello'

# 双引号定义（与单引号无本质区别，可嵌套对方）
s2 = "world"

# 三引号定义（支持多行）
s3 = '''这是第一行
这是第二行
这是第三行'''

# 嵌套引号（单引号内可直接用双引号，反之亦然）
s4 = 'He said "Python is great"'
s5 = "I'm a student"

print(s1, type(s1))  # hello <class 'str'>
print(s3)  # 输出多行内容
```


#### 2. 注意：字符串的不可变性
字符串一旦创建，其内容**不可修改**（修改会创建新字符串，原字符串不变）。
```python
s = 'hello'
# s[0] = 'H'  # 报错：TypeError（字符串不可修改）

# 若需"修改"，需创建新字符串
new_s = 'H' + s[1:]  # 拼接新字符串
print(new_s)  # Hello
```


### 二、字符串的索引与切片
字符串是序列，支持通过**索引（Index）** 访问单个字符，通过**切片（Slice）** 获取子串。


#### 1. 索引（访问单个字符）
- 正向索引：从左到右，第一个字符索引为 `0`，第二个为 `1`，以此类推；  
- 反向索引：从右到左，最后一个字符索引为 `-1`，倒数第二个为 `-2`，以此类推。  

```python
s = 'python'
# 正向索引
print(s[0])  # 'p'（第1个字符）
print(s[2])  # 't'（第3个字符）

# 反向索引
print(s[-1])  # 'n'（最后1个字符）
print(s[-3])  # 'h'（倒数第3个字符）
```


#### 2. 切片（获取子串）
语法：`str[start:end:step]`  
- `start`：起始索引（包含，默认0）；  
- `end`：结束索引（不包含，默认字符串长度）；  
- `step`：步长（每次跳过的字符数，默认1，负数表示反向切片）。  

```python
s = 'abcdefgh'

# 基本切片（start到end-1）
print(s[2:5])  # 'cde'（索引2、3、4的字符）

# 省略start（从开头开始）
print(s[:4])   # 'abcd'（索引0到3）

# 省略end（到结尾结束）
print(s[5:])   # 'fgh'（索引5到最后）

# 步长为2（每隔1个字符取1个）
print(s[::2])  # 'aceg'（0、2、4、6索引）

# 反向切片（步长为-1，倒序）
print(s[::-1])  # 'hgfedcba'（整个字符串倒序）

# 反向切片（指定范围）
print(s[5:1:-1])  # 'fedc'（索引5到2，倒序）
```


### 三、字符串的常用操作
#### 1. 拼接与重复
- 拼接：用 `+` 连接多个字符串；  
- 重复：用 `*` 重复字符串多次。  

```python
a = 'Hello'
b = 'World'
print(a + ' ' + b)  # 'Hello World'（拼接）
print(a * 3)        # 'HelloHelloHello'（重复3次）
```


#### 2. 成员运算
用 `in` 或 `not in` 判断字符/子串是否在字符串中（返回布尔值）。
```python
s = 'python'
print('p' in s)      # True（'p'是s的字符）
print('py' in s)     # True（'py'是s的子串）
print('xyz' not in s)  # True（'xyz'不在s中）
```


#### 3. 长度计算
用 `len()` 函数获取字符串的长度（字符个数）。
```python
s = 'hello world'
print(len(s))  # 11（包含空格）
```


### 四、字符串的常用内置方法
Python字符串提供了大量内置方法，以下是最常用的几类：


#### 1. 大小写转换
- `upper()`：转为全大写；  
- `lower()`：转为全小写；  
- `title()`：每个单词首字母大写（其余小写）；  
- `capitalize()`：仅首字母大写（其余小写）。  

```python
s = 'Hello World'
print(s.upper())    # 'HELLO WORLD'
print(s.lower())    # 'hello world'
print(s.title())    # 'Hello World'（每个单词首字母大写）
print(s.capitalize())  # 'Hello world'（仅第一个单词首字母大写）
```


#### 2. 查找与替换
- `find(sub)`：返回子串 `sub` 第一次出现的索引（未找到返回 `-1`）；  
- `index(sub)`：与 `find` 类似，但未找到会报错；  
- `replace(old, new, count)`：将 `old` 子串替换为 `new`（`count` 可选，指定替换次数）。  

```python
s = 'abracadabra'
print(s.find('a'))    # 0（第一个'a'的索引）
print(s.find('x'))    # -1（'x'不存在）
print(s.index('bra')) # 1（'bra'第一次出现的索引）

# 替换
print(s.replace('a', 'A'))       # 'AbrAcAdAbrA'（所有'a'替换为'A'）
print(s.replace('a', 'A', 2))    # 'AbrAcadabra'（仅替换前2个'a'）
```


#### 3. 去除空白
- `strip()`：去除两端的空白字符（空格、`\n`、`\t` 等）；  
- `lstrip()`：仅去除左端空白；  
- `rstrip()`：仅去除右端空白。  

```python
s = '  \thello world\n  '
print(s.strip())   # 'hello world'（去除两端空白）
print(s.lstrip())  # 'hello world\n  '（仅去左端）
print(s.rstrip())  # '  \thello world'（仅去右端）
```


#### 4. 分割与连接
- `split(sep, maxsplit)`：按 `sep` 分割字符串为列表（`sep` 默认为任意空白，`maxsplit` 限制分割次数）；  
- `join(iterable)`：将可迭代对象（如列表）的元素用字符串连接起来。  

```python
# 分割
s = 'apple,banana,orange'
print(s.split(','))        # ['apple', 'banana', 'orange']（按','分割）
print(s.split(',', 1))     # ['apple', 'banana,orange']（仅分割1次）

# 连接（常用！比用'+'拼接更高效）
fruits = ['apple', 'banana', 'orange']
print(','.join(fruits))    # 'apple,banana,orange'（用','连接列表元素）
print(' '.join(fruits))    # 'apple banana orange'（用空格连接）
```


#### 5. 判断类方法（返回布尔值）
- `startswith(prefix)`：是否以 `prefix` 开头；  
- `endswith(suffix)`：是否以 `suffix` 结尾；  
- `isdigit()`：是否全为数字字符；  
- `isalpha()`：是否全为字母字符；  
- `isalnum()`：是否全为字母或数字；  
- `islower()`/`isupper()`：是否全为小写/大写。  

```python
s = 'Python123'
print(s.startswith('Py'))  # True（以'Py'开头）
print(s.endswith('3'))     # True（以'3'结尾）
print(s.isdigit())         # False（不全是数字）
print(s.isalpha())         # False（含数字）
print(s.isalnum())         # True（仅字母和数字）
```


### 五、字符串格式化
字符串格式化用于将变量/值嵌入到字符串中，常用方式有3种：


#### 1. 百分号（`%`）格式化
语法：`"格式字符串" % (值1, 值2, ...)`  
常用占位符：`%s`（字符串）、`%d`（整数）、`%f`（浮点数）。  

```python
name = 'Alice'
age = 18
score = 95.5

# 基本用法
print('Name: %s, Age: %d, Score: %.1f' % (name, age, score))  
# 输出：Name: Alice, Age: 18, Score: 95.5（%.1f表示保留1位小数）
```


#### 2. `format()` 方法
语法：`"格式字符串".format(值1, 值2, ...)`  
通过 `{}` 占位，支持位置参数、关键字参数和格式控制。  

```python
# 位置参数（按顺序）
print('Name: {}, Age: {}, Score: {:.2f}'.format(name, age, score))  
# 输出：Name: Alice, Age: 18, Score: 95.50（.2f保留2位小数）

# 关键字参数（更清晰）
print('Name: {n}, Age: {a}, Score: {s}'.format(n=name, a=age, s=score))  
# 输出：Name: Alice, Age: 18, Score: 95.5
```


#### 3. f-字符串（Python3.6+，推荐）
语法：`f"字符串内容 {变量/表达式}"`（在字符串前加 `f`，直接嵌入变量或表达式），简洁高效。  

```python
# 直接嵌入变量
print(f'Name: {name}, Age: {age}, Score: {score:.1f}')  
# 输出：Name: Alice, Age: 18, Score: 95.5

# 支持表达式
print(f'Next year, age will be {age + 1}')  # 输出：Next year, age will be 19
```


### 六、转义字符与原始字符串
- **转义字符**：用 `\` 表示特殊字符（如换行、制表符等）；  
- **原始字符串**：在字符串前加 `r`，忽略转义（原样输出所有字符）。  

```python
# 转义字符
print('Hello\nWorld')  # 换行（\n）
# 输出：
# Hello
# World

print('Name: \tAlice')  # 制表符（\t，相当于按Tab）
# 输出：Name: 	Alice

print('He said \'Hi\'')  # 转义单引号（避免与外层冲突）
# 输出：He said 'Hi'

# 原始字符串（忽略转义）
print(r'C:\Users\name')  # 原样输出，不解析\U或\n
# 输出：C:\Users\name
```


### 总结
- 字符串是不可变序列，可用单引号、双引号、三引号定义；  
- 支持索引（访问单个字符）和切片（获取子串）；  
- 提供丰富的内置方法（大小写转换、查找替换、分割连接等）；  
- 格式化方式推荐使用 f-字符串（简洁高效）；  
- 转义字符用于表示特殊符号，原始字符串可忽略转义。  

掌握字符串操作是处理文本数据的基础，在日常开发中应用广泛。
