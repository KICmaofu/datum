在Python3中，运算符是用于执行各种操作（如算术计算、比较、逻辑判断等）的特殊符号。根据功能不同，可分为以下几类：


### 一、算术运算符
用于数值之间的加减乘除等基本运算，返回计算结果。

| 运算符 | 描述                 | 示例                     | 结果   |
|--------|----------------------|--------------------------|--------|
| `+`    | 加法                 | `3 + 5`                  | `8`    |
| `-`    | 减法                 | `10 - 4`                 | `6`    |
| `*`    | 乘法                 | `2 * 6`                  | `12`   |
| `/`    | 除法（返回浮点数）   | `7 / 2`                  | `3.5`  |
| `%`    | 取余（模运算）       | `7 % 2`                  | `1`    |
| `**`   | 幂运算（x的y次方）   | `2 **3`                  | `8`    |
| `//`   | 取整除（向下取整）   | `7 // 2` → `3`；`-7 // 2` → `-4` |        |


**注意**：  
- `//` 是向下取整（向负无穷方向），如负数 `(-7) // 2` 结果为 `-4`（而非 `-3`）；  
- `%` 取余结果的符号与除数一致，如 `7 % -2` 结果为 `-1`（因除数是 `-2`）。


### 二、比较运算符
用于比较两个值的关系，返回布尔值（`True` 或 `False`）。

| 运算符 | 描述               | 示例                  | 结果    |
|--------|--------------------|-----------------------|---------|
| `==`   | 等于（值相等）     | `5 == 5`              | `True`  |
| `!=`   | 不等于             | `5 != 3`              | `True`  |
| `>`    | 大于               | `10 > 8`              | `True`  |
| `<`    | 小于               | `3 < 1`               | `False` |
| `>=`   | 大于等于           | `5 >= 5`              | `True`  |
| `<=`   | 小于等于           | `2 <= 0`              | `False` |


**注意**：  
- `==` 比较的是“值是否相等”，而 `is`（身份运算符）比较的是“对象是否为同一个”（内存地址是否相同），二者不同。例如：  
  ```python
  a = [1, 2]
  b = [1, 2]
  print(a == b)  # True（值相等）
  print(a is b)  # False（不同对象，内存地址不同）
  ```


### 三、赋值运算符
用于给变量赋值，基础赋值为 `= `，还有简化运算的**复合赋值运算符**。

| 运算符 | 描述（等价于）               | 示例                     | 结果（执行后a的值） |
|--------|------------------------------|--------------------------|--------------------|
| `=`    | 基础赋值                     | `a = 5`                  | `5`                |
| `+=`   | `a = a + b`                  | `a = 3; a += 2`          | `5`                |
| `-=`   | `a = a - b`                  | `a = 5; a -= 3`          | `2`                |
| `*=`   | `a = a * b`                  | `a = 2; a *= 4`          | `8`                |
| `/=`   | `a = a / b`                  | `a = 8; a /= 2`          | `4.0`              |
| `%=`   | `a = a % b`                  | `a = 7; a %= 3`          | `1`                |
| `**=`  | `a = a **b`                 | `a = 2; a**= 3`          | `8`                |
| `//=`  | `a = a // b`                 | `a = 7; a //= 2`         | `3`                |


### 四、逻辑运算符
用于连接多个条件，进行逻辑判断，返回结果可为布尔值或实际值（因“短路特性”）。

| 运算符 | 描述                 | 逻辑规则                                  | 示例                     | 结果    |
|--------|----------------------|-------------------------------------------|--------------------------|---------|
| `and`  | 逻辑与（且）         | 两边都为真则真；有一个假则假（短路：左边假则不看右边） | `3 > 2 and 5 > 1`        | `True`  |
| `or`   | 逻辑或（或）         | 有一个为真则真；都假则假（短路：左边真则不看右边） | `3 > 5 or 5 > 1`         | `True`  |
| `not`  | 逻辑非（取反）       | 真变假，假变真                            | `not (3 > 5)`            | `True`  |


**关键特性**：  
- **短路计算**：  
  - `and`：若左边为“假”，直接返回左边值，不再计算右边；  
  - `or`：若左边为“真”，直接返回左边值，不再计算右边。  
  例如：  
  ```python
  print(0 and 5)   # 0（左边0是假，短路返回0）
  print(5 or 0)    # 5（左边5是真，短路返回5）
  print(not 0)     # True（0是假，取反为真）
  ```  
- 这里的“真/假”遵循 `bool()` 转换规则（非零数字、非空对象为真，0、空对象、`None` 为假）。


### 五、位运算符
对整数的二进制位进行直接操作（需先将整数转为二进制）。

| 运算符 | 描述（二进制操作）               | 示例（十进制）       | 二进制计算过程（以8位为例）          | 结果 |
|--------|----------------------------------|----------------------|---------------------------------------|------|
| `&`    | 按位与（同1则1，否则0）          | `5 & 3`              | `00000101 & 00000011 = 00000001`      | `1`  |
| `|`    | 按位或（有1则1，否则0）          | `5 | 3`              | `00000101 | 00000011 = 00000111`      | `7`  |
| `^`    | 按位异或（不同则1，相同则0）      | `5 ^ 3`              | `00000101 ^ 00000011 = 00000110`      | `6`  |
| `~`    | 按位取反（0变1，1变0，含符号位）  | `~5`                 | `~00000101 = 11111010`（十进制 `-6`） | `-6` |
| `<<`   | 左移（左移n位=乘2ⁿ）             | `5 << 1`             | `00000101 << 1 = 00001010`            | `10` |
| `>>`   | 右移（右移n位=整除2ⁿ，保留符号） | `5 >> 1`             | `00000101 >> 1 = 00000010`            | `2`  |


### 六、成员运算符
用于检查“元素是否在序列中”（序列：列表、元组、字符串、集合等），返回布尔值。

| 运算符   | 描述                     | 示例                          | 结果    |
|----------|--------------------------|-------------------------------|---------|
| `in`     | 元素在序列中则为真       | `"a" in "abc"`                | `True`  |
| `not in` | 元素不在序列中则为真     | `3 not in [1, 2, 4]`          | `True`  |


### 七、身份运算符
用于检查“两个对象是否为同一个”（即内存地址是否相同），返回布尔值。

| 运算符    | 描述                     | 示例                                  | 结果    |
|-----------|--------------------------|---------------------------------------|---------|
| `is`      | 两个对象是同一个则为真   | `a = [1]; b = a; a is b`              | `True`  |
| `is not`  | 两个对象不是同一个则为真 | `a = [1]; b = [1]; a is not b`        | `True`  |


**注意**：  
- 对于小整数（-5到256）和短字符串，Python会缓存对象，可能出现 `a is b` 为 `True` 但实际是不同赋值的情况（属于优化机制），但不建议依赖此特性。


### 八、运算符优先级（简化版）
当多个运算符同时出现时，优先级高的先执行（可通过括号 `()` 强制改变优先级）。  
从高到低大致顺序：  
1. 幂运算 `**`  
2. 位取反 `~`、正负号 `+x`/`-x`  
3. 算术运算符（`*`/`/`/`//`/`%` 高于 `+`/`-`）  
4. 位运算符（`<<`/`>>` 高于 `&` 高于 `^` 高于 `|`）  
5. 比较运算符（`==`/`!=`/`>`/`<`/`>=`/`<=`）  
6. 身份运算符（`is`/`is not`）  
7. 成员运算符（`in`/`not in`）  
8. 逻辑非 `not`  
9. 逻辑与 `and`  
10. 逻辑或 `or`  


**示例**：  
```python
print(2 + 3 * 4)       # 14（先乘后加）
print((2 + 3) * 4)     # 20（括号改变优先级）
print(3 > 1 and 5 < 2) # False（先比较后逻辑与）
```


通过合理使用运算符，可实现Python中几乎所有的操作逻辑，掌握其特性和优先级能避免很多错误。
