在Python中，**条件控制**通过判断条件的真假来决定执行不同的代码块，核心语法是 `if` 语句，包括 `if`、`if-else`、`if-elif-else` 三种形式。条件控制是实现程序分支逻辑的基础（即“如果满足A就做X，满足B就做Y”）。


### 一、基本语法结构
Python的条件控制通过**缩进**（通常是4个空格）来区分代码块（无需使用大括号 `{}`），缩进相同的代码属于同一代码块。


#### 1. 单分支：`if` 语句
当条件为 `True` 时，执行缩进的代码块；条件为 `False` 时，不执行。  

**语法**：  
```python
if 条件表达式:
    代码块（条件为True时执行）
```

**示例**：判断一个数是否为正数  
```python
num = 5
if num > 0:  # 条件：num大于0
    print(f"{num}是正数")  # 条件为True，执行此句
# 输出：5是正数

num = -3
if num > 0:
    print(f"{num}是正数")  # 条件为False，不执行
# 无输出
```


#### 2. 双分支：`if-else` 语句
当条件为 `True` 时执行 `if` 后的代码块，为 `False` 时执行 `else` 后的代码块（二选一）。  

**语法**：  
```python
if 条件表达式:
    代码块1（条件为True时执行）
else:
    代码块2（条件为False时执行）
```

**示例**：判断一个数是正数还是非正数  
```python
num = 5
if num > 0:
    print(f"{num}是正数")
else:
    print(f"{num}不是正数")
# 输出：5是正数

num = -3
if num > 0:
    print(f"{num}是正数")
else:
    print(f"{num}不是正数")
# 输出：-3不是正数
```


#### 3. 多分支：`if-elif-else` 语句
用于判断多个条件，依次检查每个条件：  
- 若第一个条件为 `True`，执行对应代码块，后续条件不再判断；  
- 若所有 `if` 和 `elif` 条件都为 `False`，则执行 `else` 代码块（`else` 可选）。  

**语法**：  
```python
if 条件1:
    代码块1（条件1为True时执行）
elif 条件2:
    代码块2（条件1为False，条件2为True时执行）
elif 条件3:
    代码块3（条件1、2为False，条件3为True时执行）
...
else:
    代码块n（所有条件都为False时执行）
```

**示例**：根据分数判断等级（0-100分）  
```python
score = 85
if score >= 90:
    print("优秀")
elif score >= 80:  # 仅当score <90时才会检查此条件
    print("良好")
elif score >= 60:  # 仅当score <80时才会检查此条件
    print("及格")
else:  # 所有条件都不满足（score <60）
    print("不及格")
# 输出：良好（因85 >=80且<90）
```

**注意**：`elif` 是“else if”的缩写，必须跟在 `if` 之后；`else` 是可选的，且最多只能有一个。


### 二、条件表达式的类型
条件表达式的结果可以是**布尔值**（`True`/`False`），也可以是**可转换为布尔值的其他类型**（Python会自动用 `bool()` 转换）。


#### 1. 布尔值直接作为条件
```python
flag = True
if flag:  # 等价于 if True
    print("条件为真")  # 执行
```


#### 2. 比较表达式（返回布尔值）
结合比较运算符（`==`、`!=`、`>`、`<`、`>=`、`<=`）使用：  
```python
a = 10
b = 20
if a < b:  # 10 < 20 → True
    print(f"{a}小于{b}")  # 执行
```


#### 3. 逻辑表达式（返回布尔值）
结合逻辑运算符（`and`、`or`、`not`）组合多个条件：  
```python
age = 18
score = 90
# 同时满足两个条件（and：两边都为真才为真）
if age >= 18 and score >= 80:
    print("成年且成绩优秀")  # 执行（18>=18为真，90>=80为真）

# 满足任一条件（or：有一个为真就为真）
if age < 18 or score < 60:
    print("未成年或不及格")  # 不执行（两个条件都为假）

# 条件取反（not：真变假，假变真）
if not (age < 18):  # 等价于 age >= 18
    print("已成年")  # 执行
```


#### 4. 其他类型自动转换为布尔值
根据 `bool()` 转换规则，以下值会被视为 `False`，其他值视为 `True`：  
- `0`、`0.0`（零值）；  
- 空字符串 `""`、空列表 `[]`、空元组 `()`、空字典 `{}`、空集合 `set()`；  
- `None`。  

**示例**：  
```python
s = "hello"
if s:  # 非空字符串 → True
    print("字符串非空")  # 执行

lst = []
if lst:  # 空列表 → False
    print("列表非空")  # 不执行
else:
    print("列表为空")  # 执行
```


### 三、嵌套条件控制
条件语句内部可以再包含条件语句（嵌套），用于处理更复杂的分支逻辑。  

**示例**：先判断是否为正数，再判断是否为偶数  
```python
num = 6
if num > 0:  # 外层条件：正数
    print(f"{num}是正数")
    if num % 2 == 0:  # 内层条件：偶数（仅当外层条件为真时才检查）
        print(f"{num}是偶数")
    else:
        print(f"{num}是奇数")
else:
    print(f"{num}不是正数")
# 输出：
# 6是正数
# 6是偶数
```


### 四、三元表达式（简洁的条件判断）
三元表达式是 `if-else` 语句的简洁写法，用于将简单的条件判断和赋值合并为一行。  

**语法**：  
```python
变量 = 表达式1 if 条件 else 表达式2
```
- 若条件为 `True`，变量赋值为 `表达式1`；  
- 若条件为 `False`，变量赋值为 `表达式2`。  


**示例**：  
```python
num = 5
# 普通if-else
if num % 2 == 0:
    result = "偶数"
else:
    result = "奇数"
print(result)  # 奇数

# 三元表达式简化
result = "偶数" if num % 2 == 0 else "奇数"
print(result)  # 奇数
```

**注意**：三元表达式仅适合简单逻辑，复杂逻辑建议用普通 `if-else` 以保证可读性。


### 五、常见错误与注意事项
1. **缩进错误**：Python通过缩进区分代码块，缩进不一致会导致 `IndentationError`。  
   ```python
   # 错误示例：缩进不一致
   if 5 > 3:
       print("5>3")
     print("缩进错误")  # 缩进与上一行不同，报错
   ```

2. **条件判断的顺序**：`if-elif-else` 中，条件是**从上到下依次判断**的，前面的条件满足后，后面的条件不再执行。  
   ```python
   score = 95
   # 错误示例：条件顺序错误
   if score >= 60:
       print("及格")  # 会执行（95>=60为真），但实际应判断为优秀
   elif score >= 90:
       print("优秀")  # 不会执行
   ```
   正确做法：**从严格到宽松**排列条件（如先判断高分段，再判断低分段）。

3. **混淆 `==` 和 `=`**：`==` 是比较运算符（判断相等），`=` 是赋值运算符，条件中误用 `=` 会报错或逻辑错误。  
   ```python
   # 错误示例：用=代替==
   if num = 5:  # 报错：SyntaxError（条件中不能用赋值）
       print("num是5")
   ```


### 总结
条件控制是Python实现分支逻辑的核心，通过 `if`、`if-else`、`if-elif-else` 实现不同场景的判断：  
- 单分支（`if`）：满足条件才执行；  
- 双分支（`if-else`）：二选一执行；  
- 多分支（`if-elif-else`）：多选一执行；  
- 嵌套条件：处理更复杂的逻辑；  
- 三元表达式：简化简单的 `if-else` 赋值。  

掌握条件控制能让程序根据不同情况做出不同响应，是编写灵活程序的基础。
