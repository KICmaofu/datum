Python 的 `os` 模块提供了与操作系统交互的功能，尤其擅长处理文件和目录操作（如创建、删除、重命名、查询等）。它封装了操作系统的底层接口，支持跨平台（Windows、Linux、macOS 等）使用。以下是 `os` 模块中常用的文件/目录方法及示例：


### 一、基础准备
使用 `os` 模块前需先导入：
```python
import os
```


### 二、目录操作方法
#### 1. 创建目录
- **`os.mkdir(path, mode=0o777)`**  
  创建**单级目录**（父目录必须已存在，否则报错）。  
  - `path`：目录路径。  
  - `mode`：权限模式（默认 `0o777`，受系统 umask 限制）。  

  ```python
  # 创建单级目录 "test_dir"
  os.mkdir("test_dir")  # 若目录已存在，会报错 FileExistsError
  ```

- **`os.makedirs(name, mode=0o777, exist_ok=False)`**  
  创建**多级目录**（若父目录不存在，自动创建，类似 `mkdir -p`）。  
  - `exist_ok`：若为 `True`，目录已存在时不报错（默认 `False`，会报错）。  

  ```python
  # 创建多级目录 "a/b/c"
  os.makedirs("a/b/c")  # 自动创建 a、a/b、a/b/c

  # 允许目录已存在（不报错）
  os.makedirs("a/b/c", exist_ok=True)
  ```


#### 2. 删除目录
- **`os.rmdir(path)`**  
  删除**空的单级目录**（目录非空或不存在时报错）。  

  ```python
  os.rmdir("test_dir")  # 目录必须为空，否则报错 OSError
  ```

- **`os.removedirs(name)`**  
  递归删除空目录（从子目录向上删除，直到遇到非空目录为止）。  

  ```python
  # 删除 "a/b/c"（需确保 a/b/c、a/b、a 均为空）
  os.removedirs("a/b/c")  # 先删 c，再删 b，最后删 a
  ```


#### 3. 遍历与查看目录
- **`os.listdir(path='.')`**  
  返回指定目录下的**所有文件和子目录名称**（列表形式，不包含递归子目录）。  

  ```python
  # 查看当前目录内容
  print(os.listdir("."))  # 输出：['test.txt', 'test_dir', ...]
  ```

- **`os.scandir(path='.')`**  
  返回目录的迭代器（`DirEntry` 对象），包含文件/目录的详细信息（如是否为文件、大小等），比 `listdir` 更高效（尤其处理大量文件时）。  

  ```python
  with os.scandir(".") as entries:
      for entry in entries:
          print(entry.name, "是否为文件：", entry.is_file())
  ```


#### 4. 切换与获取目录
- **`os.chdir(path)`**  
  切换当前工作目录（类似 `cd` 命令）。  

  ```python
  os.chdir("test_dir")  # 切换到 test_dir 目录
  ```

- **`os.getcwd()`**  
  获取当前工作目录的绝对路径。  

  ```python
  print(os.getcwd())  # 输出：/home/user/test_dir（Linux）或 C:\user\test_dir（Windows）
  ```


### 三、文件操作方法
#### 1. 删除文件
- **`os.remove(path)`**  
  删除指定文件（文件不存在时报错）。  

  ```python
  os.remove("test.txt")  # 删除 test.txt 文件
  ```


#### 2. 重命名文件/目录
- **`os.rename(src, dst)`**  
  重命名文件或目录（若目标路径已存在，Windows 会报错，Linux 会覆盖）。  

  ```python
  # 重命名文件
  os.rename("old.txt", "new.txt")

  # 重命名目录
  os.rename("old_dir", "new_dir")
  ```

- **`os.replace(src, dst)`**  
  类似 `rename`，但即使目标路径已存在也会强制替换（跨平台一致性更好）。  


#### 3. 获取文件/目录信息
- **`os.stat(path)`**  
  返回文件/目录的元信息（如大小、创建时间、修改时间等）。  

  ```python
  stat_info = os.stat("test.txt")
  print("文件大小（字节）：", stat_info.st_size)
  print("最后修改时间：", stat_info.st_mtime)  # 时间戳格式
  ```

  时间戳可通过 `datetime` 转换为可读格式：  
  ```python
  from datetime import datetime
  print("最后修改时间：", datetime.fromtimestamp(stat_info.st_mtime))
  ```


### 四、路径处理（`os.path` 子模块）
`os.path` 是 `os` 的子模块，专注于路径字符串的处理（跨平台兼容，自动处理 `/` 和 `\` 分隔符）。

#### 1. 路径拼接与拆分
- **`os.path.join(path1, path2, ...)`**  
  拼接多个路径片段（自动处理分隔符，推荐使用，避免手动拼接）。  

  ```python
  # 跨平台路径拼接
  full_path = os.path.join("a", "b", "test.txt")
  print(full_path)  # Linux: a/b/test.txt；Windows: a\b\test.txt
  ```

- **`os.path.split(path)`**  
  拆分路径为 `(目录, 文件名)` 元组。  

  ```python
  dir_name, file_name = os.path.split("/a/b/test.txt")
  print(dir_name)  # /a/b
  print(file_name)  # test.txt
  ```

- **`os.path.splitext(path)`**  
  拆分路径为 `(路径, 扩展名)` 元组（用于分离文件名和后缀）。  

  ```python
  name, ext = os.path.splitext("test.txt")
  print(name)  # test
  print(ext)   # .txt
  ```


#### 2. 路径属性判断
- **`os.path.exists(path)`**：判断路径（文件或目录）是否存在。  
- **`os.path.isfile(path)`**：判断是否为文件。  
- **`os.path.isdir(path)`**：判断是否为目录。  
- **`os.path.isabs(path)`**：判断是否为绝对路径。  

  ```python
  path = "test.txt"
  print(os.path.exists(path))  # True（若文件存在）
  print(os.path.isfile(path))  # True
  print(os.path.isdir(path))   # False
  ```


#### 3. 绝对路径与规范化
- **`os.path.abspath(path)`**：返回路径的绝对路径。  

  ```python
  print(os.path.abspath("test.txt"))  # /home/user/test.txt（绝对路径）
  ```

- **`os.path.normpath(path)`**：规范化路径（处理 `.` 和 `..` 等相对路径符号）。  

  ```python
  print(os.path.normpath("/a/b/../c/test.txt"))  # /a/c/test.txt（处理 .. 回到上级目录）
  ```


#### 4. 目录与文件名提取
- **`os.path.dirname(path)`**：返回路径中的目录部分。  
- **`os.path.basename(path)`**：返回路径中的文件名（含扩展名）。  

  ```python
  path = "/a/b/test.txt"
  print(os.path.dirname(path))  # /a/b
  print(os.path.basename(path)) # test.txt
  ```


### 五、其他常用方法
- **`os.environ`**：获取系统环境变量（字典形式）。  
  ```python
  print(os.environ.get("PATH"))  # 打印 PATH 环境变量
  ```

- **`os.system(command)`**：执行系统命令（返回命令退出码，`0` 表示成功）。  
  ```python
  os.system("ls -l")  # Linux：列出目录详情；Windows 可替换为 "dir"
  ```

- **`os.sep`**：当前系统的路径分隔符（Linux 为 `/`，Windows 为 `\`）。  
- **`os.linesep`**：当前系统的换行符（Linux 为 `\n`，Windows 为 `\r\n`）。  


### 六、错误处理
文件/目录操作可能因权限不足、路径不存在等原因报错，需用 `try-except` 捕获异常：
```python
try:
    os.mkdir("test_dir")
except FileExistsError:
    print("目录已存在")
except PermissionError:
    print("权限不足")
except Exception as e:
    print("错误：", e)
```


### 七、常用场景总结
| 需求                | 推荐方法                          |
|---------------------|-----------------------------------|
| 创建多级目录        | `os.makedirs(path, exist_ok=True)` |
| 遍历目录文件        | `os.scandir(path)` 或 `os.listdir(path)` |
| 路径拼接            | `os.path.join(...)`              |
| 判断文件/目录是否存在 | `os.path.exists(path)` + `os.path.isfile`/`isdir` |
| 重命名/移动文件     | `os.rename(src, dst)`            |
| 删除空目录/文件     | `os.rmdir(dir)` / `os.remove(file)` |


### 总结
`os` 模块是 Python 处理文件和目录的核心工具，结合 `os.path` 子模块可高效完成路径处理、目录管理、文件操作等任务。使用时需注意跨平台兼容性（优先用 `os.path.join` 拼接路径），并通过异常处理确保代码健壮性。对于更复杂的操作（如复制文件），可结合 `shutil` 模块（`os` 的补充，提供更高层的文件操作）。
