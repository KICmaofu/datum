在Python3中，**输入（Input）** 指从外部（如用户输入、文件、网络等）获取数据，**输出（Output）** 指将数据展示给用户或写入外部存储（如屏幕、文件）。这是程序与外界交互的核心方式，以下是详细介绍：


### 一、输入：获取外部数据
最常用的输入方式是通过 `input()` 函数获取用户从控制台的输入，此外还可以从文件、命令行参数等获取输入。


#### 1. `input()` 函数：获取用户控制台输入
`input()` 函数会暂停程序执行，等待用户输入，用户按下回车后，返回输入的内容（**默认是字符串类型**）。

**基本用法**：
```python
# 提示用户输入，并将结果保存到变量
name = input("请输入你的名字：")
print(f"你好，{name}！")  # 输出：你好，XXX！（XXX为用户输入）
```

**注意**：  
`input()` 无论用户输入什么（数字、符号等），返回的都是字符串。如果需要其他类型（如整数、浮点数），需手动转换：
```python
# 输入数字并转换类型
age_str = input("请输入你的年龄：")
age = int(age_str)  # 转换为整数
print(f"明年你将是 {age + 1} 岁")

# 简写：直接在输入后转换
height = float(input("请输入你的身高（米）："))  # 输入"1.75" → 转换为1.75
print(f"身高的厘米数：{height * 100}")  # 175.0
```

**处理输入错误**：如果用户输入的内容无法转换（如输入字母却想转整数），会报错。可以用 `try-except` 捕获错误：
```python
try:
    num = int(input("请输入一个数字："))
    print(f"你输入的数字是 {num}")
except ValueError:
    print("输入错误！请输入有效的数字。")
```


#### 2. 从命令行参数获取输入
通过 `sys` 模块的 `argv` 列表，可以获取运行脚本时传入的命令行参数（适合批量输入或自动化场景）。

**示例**：创建 `args_demo.py` 脚本：
```python
import sys

# sys.argv 是一个列表，第一个元素是脚本名，后续是传入的参数
print("脚本名：", sys.argv[0])
print("参数列表：", sys.argv[1:])  # 切片获取除脚本名外的参数
```

运行脚本时传入参数：
```bash
python args_demo.py 100 hello world
```

输出：
```
脚本名： args_demo.py
参数列表： ['100', 'hello', 'world']
```

**注意**：命令行参数默认是字符串类型，需按需转换（如 `int(sys.argv[1])`）。


### 二、输出：展示或保存数据
最常用的输出方式是通过 `print()` 函数向控制台打印内容，也可以将数据写入文件。


#### 1. `print()` 函数：控制台输出
`print()` 用于将数据打印到控制台，支持多种参数控制输出格式。

**基本用法**：
```python
# 打印字符串、变量、表达式
name = "Alice"
age = 20
print("姓名：", name)  # 打印多个值（默认用空格分隔）
print(f"年龄：{age}")  # 用f-字符串嵌入变量
print("明年年龄：", age + 1)  # 打印表达式结果
```

**常用参数**：
- `sep`：指定多个值之间的分隔符（默认是空格）。
- `end`：指定输出结束时的字符（默认是换行符 `\n`）。
- `file`：指定输出到的文件对象（默认是控制台）。

**示例**：
```python
# 自定义分隔符
print("apple", "banana", "orange", sep=", ")  # 输出：apple, banana, orange

# 自定义结束符（不换行）
print("Hello", end=" ")
print("World")  # 输出：Hello World（两行合并为一行）

# 输出到文件（需先打开文件）
with open("output.txt", "w", encoding="utf-8") as f:
    print("这行内容会写入文件", file=f)  # 内容被写入output.txt
```


#### 2. 格式化输出：控制字符串样式
当需要按特定格式输出（如对齐、保留小数、拼接变量）时，可使用以下三种方式：


##### （1）f-字符串（Python3.6+，推荐）
在字符串前加 `f` 或 `F`，用 `{变量/表达式}` 嵌入内容，简洁直观。

**示例**：
```python
name = "Bob"
score = 95.567

# 基本用法
print(f"姓名：{name}，分数：{score}")  # 姓名：Bob，分数：95.567

# 格式化数字（保留2位小数）
print(f"分数：{score:.2f}")  # 分数：95.57

# 表达式计算
print(f"分数翻倍：{score * 2:.1f}")  # 分数翻倍：191.1

# 对齐（左对齐<，右对齐>，居中^，宽度为10）
print(f"左对齐：{name:<10}结束")  # 左对齐：Bob       结束
print(f"右对齐：{name:>10}结束")  # 右对齐：       Bob结束
```


##### （2）`str.format()` 方法
通过 `{}` 占位符，用 `format()` 传入参数，兼容性好（支持Python3.5及以下）。

**示例**：
```python
name = "Bob"
score = 95.567

# 按位置占位
print("姓名：{0}，分数：{1:.2f}".format(name, score))  # 姓名：Bob，分数：95.57

# 按名称占位
print("姓名：{n}，分数：{s:.1f}".format(n=name, s=score))  # 姓名：Bob，分数：95.6

# 对齐（宽度10，居中）
print("居中：{:^10}".format(name))  # 居中：   Bob    
```


##### （3）% 格式化（旧式，了解即可）
类似C语言的 `printf`，用 `%` 作为占位符，现已逐渐被f-字符串替代。

**示例**：
```python
name = "Bob"
score = 95.567

# %s（字符串）、%f（浮点数）、%d（整数）
print("姓名：%s，分数：%.2f" % (name, score))  # 姓名：Bob，分数：95.57
```


### 三、文件输入输出：读写外部文件
除了控制台，程序常需从文件读取数据或写入数据到文件（如文本、图片、日志等）。核心是通过 `open()` 函数操作文件。


#### 1. 打开文件：`open()` 函数
`open(file, mode='r', encoding=None)` 用于打开文件，返回**文件对象**。  
- `file`：文件路径（绝对路径或相对路径）。  
- `mode`：打开模式（常用模式如下）：  
  | 模式 | 含义                                  | 适用文件类型 |
  |------|---------------------------------------|--------------|
  | `r`  | 只读（默认），文件不存在则报错        | 文本/二进制  |
  | `w`  | 只写，覆盖原有内容；文件不存在则创建  | 文本/二进制  |
  | `a`  | 追加，在文件末尾添加内容；不存在则创建| 文本/二进制  |
  | `r+` | 读写                                  | 文本/二进制  |
  | 加 `b` | 二进制模式（如 `rb`、`wb`）           | 非文本文件   |
- `encoding`：文本文件编码（如 `utf-8`，二进制文件无需指定）。


#### 2. 关闭文件：`close()` 与 `with` 语句
文件使用后必须关闭，否则会占用系统资源。推荐用 `with` 语句（自动关闭文件，无需手动调用 `close()`）。

**最佳实践（`with` 语句）**：
```python
# 读取文件
with open("test.txt", "r", encoding="utf-8") as f:  # 打开文件
    content = f.read()  # 读取内容
print("文件内容：", content)  # 退出with块后，文件自动关闭

# 写入文件
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("这是写入的第一行\n")  # 写入字符串（需手动加换行符）
    f.writelines(["第二行\n", "第三行\n"])  # 写入字符串列表
```


#### 3. 读取文件的常用方法
- `read(size)`：读取指定字节数（默认读取全部内容）。  
- `readline()`：读取一行内容（包括换行符）。  
- `readlines()`：读取所有行，返回列表（每行作为一个元素）。  

**示例**：
```python
with open("test.txt", "r", encoding="utf-8") as f:
    # 方法1：读取全部
    print("全部内容：", f.read())

    # 方法2：逐行读取（适合大文件，节省内存）
    f.seek(0)  # 重置文件指针到开头
    line = f.readline()
    while line:
        print("行内容：", line.strip())  # strip()去除换行符
        line = f.readline()

    # 方法3：读取所有行到列表
    f.seek(0)
    lines = f.readlines()
    print("所有行：", lines)  # ['第一行\n', '第二行\n']
```


#### 4. 二进制文件读写（如图片、音频）
二进制文件需用 `b` 模式打开（如 `rb`、`wb`），读写的是字节流（`bytes` 类型）。

**示例：复制图片**
```python
# 读取原图（二进制）
with open("source.jpg", "rb") as src:
    data = src.read()  # 读取字节流

# 写入新图（二进制）
with open("copy.jpg", "wb") as dst:
    dst.write(data)  # 写入字节流
```


### 四、总结
- **输入**：  
  - 用 `input()` 获取用户控制台输入（返回字符串，需手动转换类型）。  
  - 用 `sys.argv` 获取命令行参数。  
- **输出**：  
  - 用 `print()` 打印到控制台，通过 `sep`、`end` 控制格式。  
  - 格式化输出优先用 f-字符串（简洁），其次用 `str.format()`。  
- **文件操作**：  
  - 用 `with open(...)` 打开文件（自动关闭，安全）。  
  - 文本文件指定 `encoding="utf-8"` 避免乱码；二进制文件加 `b` 模式。  

掌握输入输出是编写交互式程序、处理数据的基础，灵活运用可满足大部分场景需求。
