在Python3中，**集合（Set）** 是一种无序、可变、元素唯一的容器类型，用于存储不重复的元素。集合的核心特性是**去重**和**支持数学集合运算**（如交集、并集等），适合用于元素去重、关系判断等场景。


### 一、集合的定义
集合用**大括号 `{}`** 表示（元素之间用逗号分隔），或通过 `set()` 函数创建。  
- 集合中的元素必须是**可哈希的不可变类型**（如整数、字符串、元组等），列表、字典等可变类型不能作为集合元素；  
- 集合会自动去除重复元素（相同元素仅保留一个）；  
- 集合是无序的，**不能通过索引访问元素**（与列表、元组不同）。  


#### 1. 基本定义方式
```python
# 非空集合（用{}）
nums = {1, 2, 3, 4, 4}  # 重复元素会被自动去重
print(nums)  # {1, 2, 3, 4}（顺序可能不同，因无序）

# 混合类型集合（元素为可哈希类型）
mixed = {1, "hello", (2, 3)}  # 整数、字符串、元组（均为可哈希类型）
print(mixed)  # {1, 'hello', (2, 3)}（顺序不定）

# 用set()函数创建集合（接收可迭代对象，如列表、字符串）
set_from_list = set([1, 2, 2, 3])  # 列表转集合（去重）
set_from_str = set("hello")        # 字符串转集合（每个字符为元素，去重）
print(set_from_list)  # {1, 2, 3}
print(set_from_str)   # {'h', 'e', 'l', 'o'}（'l'重复，仅保留一个）

# 空集合（必须用set()，不能用{}，因{}表示空字典）
empty_set = set()
print(empty_set, type(empty_set))  # set() <class 'set'>
```


#### 2. 核心特性总结
- **无序性**：元素没有固定顺序，每次打印可能不同；  
- **唯一性**：自动去重，集合中不存在重复元素；  
- **可变性**：可添加、删除元素（但元素本身必须不可变）；  
- **不可索引**：不能通过 `[index]` 访问元素（因无序）。  


### 二、集合的基本操作
#### 1. 添加元素
- `add(x)`：向集合中添加单个元素 `x`（若 `x` 已存在，则不做任何操作）；  
- `update(iterable)`：向集合中添加**可迭代对象**（如列表、字符串、集合）的所有元素（自动去重）。  

```python
s = {1, 2, 3}

# add()：添加单个元素
s.add(4)
print(s)  # {1, 2, 3, 4}（添加新元素）
s.add(2)  # 元素2已存在，不操作
print(s)  # {1, 2, 3, 4}

# update()：添加可迭代对象的所有元素
s.update([4, 5, 6])  # 添加列表元素（4已存在，自动去重）
print(s)  # {1, 2, 3, 4, 5, 6}

s.update("abc")  # 添加字符串的每个字符
print(s)  # {1, 2, 3, 4, 5, 6, 'a', 'b', 'c'}
```


#### 2. 删除元素
- `remove(x)`：删除元素 `x`（若 `x` 不存在，则**报错** `KeyError`）；  
- `discard(x)`：删除元素 `x`（若 `x` 不存在，**不报错**，无操作）；  
- `pop()`：随机删除并返回集合中的一个元素（因无序，无法确定删除哪个）；  
- `clear()`：清空集合（删除所有元素，保留空集合）。  

```python
s = {1, 2, 3, 4}

# remove()：删除存在的元素
s.remove(3)
print(s)  # {1, 2, 4}

# remove()删除不存在的元素 → 报错
# s.remove(5)  # 报错：KeyError: 5

# discard()：删除存在的元素
s.discard(2)
print(s)  # {1, 4}

# discard()删除不存在的元素 → 无操作
s.discard(5)
print(s)  # {1, 4}

# pop()：随机删除一个元素
deleted = s.pop()
print(deleted)  # 可能是1或4（随机）
print(s)        # 剩余元素

# clear()：清空集合
s.clear()
print(s)  # set()
```


### 三、集合的运算（数学集合操作）
集合支持数学中的交集、并集、差集等运算，可通过运算符或方法实现。假设有两个集合 `a` 和 `b`：

| 运算类型   | 运算符 | 对应方法                | 描述                                  |
|------------|--------|-------------------------|---------------------------------------|
| 并集       | `a \| b` | `a.union(b)`            | 包含 `a` 和 `b` 中所有的元素（去重）  |
| 交集       | `a & b` | `a.intersection(b)`     | 包含 `a` 和 `b` 中**共同的元素**      |
| 差集       | `a - b` | `a.difference(b)`       | 包含 `a` 中有但 `b` 中**没有的元素**  |
| 对称差集   | `a ^ b` | `a.symmetric_difference(b)` | 包含 `a` 和 `b` 中**不同时存在的元素**（即并集减去交集） |


#### 示例
```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# 并集（a和b所有元素）
print(a | b)  # {1, 2, 3, 4, 5, 6}
print(a.union(b))  # 同上

# 交集（a和b共同元素）
print(a & b)  # {3, 4}
print(a.intersection(b))  # 同上

# 差集（a有b没有的元素）
print(a - b)  # {1, 2}
print(a.difference(b))  # 同上

# 对称差集（a和b不同时有的元素）
print(a ^ b)  # {1, 2, 5, 6}
print(a.symmetric_difference(b))  # 同上
```


### 四、集合的关系判断
- `a.issubset(b)` 或 `a <= b`：判断 `a` 是否是 `b` 的**子集**（`a` 的所有元素都在 `b` 中）；  
- `a.issuperset(b)` 或 `a >= b`：判断 `a` 是否是 `b` 的**超集**（`b` 的所有元素都在 `a` 中）；  
- `a.isdisjoint(b)`：判断 `a` 和 `b` 是否**不相交**（没有共同元素）。  


#### 示例
```python
a = {1, 2}
b = {1, 2, 3, 4}
c = {5, 6}

# 子集判断（a是b的子集）
print(a.issubset(b))  # True
print(a <= b)         # True

# 超集判断（b是a的超集）
print(b.issuperset(a))  # True
print(b >= a)           # True

# 不相交判断（a和c没有共同元素）
print(a.isdisjoint(c))  # True
print(a.isdisjoint(b))  # False（a和b有共同元素1、2）
```


### 五、集合推导式（Set Comprehension）
集合推导式是创建集合的简洁语法，格式为：  
`{表达式 for 变量 in 可迭代对象 if 条件}`  
作用：遍历可迭代对象，对符合条件的元素应用表达式，生成新集合（自动去重）。


#### 示例
```python
# 生成1-10中偶数的集合
evens = {x for x in range(1, 11) if x % 2 == 0}
print(evens)  # {2, 4, 6, 8, 10}

# 字符串去重（提取不重复的字符）
chars = {c for c in "hello world"}
print(chars)  # {'h', 'e', 'l', 'o', ' ', 'w', 'r', 'd'}（自动去重'l'）
```


### 六、冻结集合（frozenset）
**冻结集合**是一种**不可变的集合**（`frozenset` 类型），创建后不能添加、删除元素，但其元素仍需可哈希。  
- 语法：`frozenset(可迭代对象)`；  
- 用途：可作为字典的键（因不可变、可哈希），或作为其他集合的元素（普通集合的元素必须可哈希）。  


#### 示例
```python
# 创建冻结集合
fs = frozenset([1, 2, 3])
print(fs, type(fs))  # frozenset({1, 2, 3}) <class 'frozenset'>

# 冻结集合不可修改（添加/删除元素会报错）
# fs.add(4)  # 报错：AttributeError: 'frozenset' object has no attribute 'add'

# 冻结集合可作为字典的键
d = {fs: "value"}  # 合法
print(d)  # {frozenset({1, 2, 3}): 'value'}

# 普通集合不能作为集合的元素（因不可哈希），但冻结集合可以
s = {fs, frozenset([4, 5])}  # 合法
print(s)  # {frozenset({1, 2, 3}), frozenset({4, 5})}
```


### 七、集合与其他数据类型的区别
| 数据类型 | 核心特点                          | 核心用途                     |
|----------|-----------------------------------|------------------------------|
| 列表     | 有序、可变、可重复                | 存储有序、可修改的序列       |
| 元组     | 有序、不可变、可重复              | 存储固定不变的序列           |
| 字典     | 键值对、可通过键访问、无序（3.7+有序） | 存储关联关系数据             |
| 集合     | 无序、可变、不可重复、支持集合运算 | 元素去重、集合关系判断（交集/并集等） |
| 冻结集合 | 无序、不可变、不可重复            | 作为字典键或集合元素         |


### 八、注意事项
1. **元素必须可哈希**：列表、字典等可变类型不能作为集合元素（因不可哈希），否则报错。  
   ```python
   # 错误示例：列表作为集合元素
   # s = {[1, 2], 3}  # 报错：TypeError: unhashable type: 'list'
   ```

2. **无序性导致不可索引**：不能通过 `s[0]` 访问元素，需通过循环或转换为列表后访问。  
   ```python
   s = {1, 2, 3}
   # print(s[0])  # 报错：TypeError: 'set' object is not subscriptable

   # 转为列表后可索引（但顺序不固定）
   lst = list(s)
   print(lst[0])  # 可能是1、2或3
   ```

3. **集合的去重特性**：适合快速去除列表、字符串中的重复元素（转换为集合后再转回原类型）。  
   ```python
   # 列表去重
   lst = [1, 2, 2, 3, 3, 3]
   unique_lst = list(set(lst))  # 先转集合去重，再转列表
   print(unique_lst)  # [1, 2, 3]（顺序可能不同）
   ```


### 总结
集合是Python中用于存储唯一元素的无序容器，核心优势是**自动去重**和**支持数学集合运算**。通过 `add()`、`update()` 可添加元素，通过 `remove()`、`discard()` 等可删除元素，配合集合推导式可简洁创建集合。冻结集合作为不可变版本，可用于字典键或集合元素。集合在去重、元素关系判断（如交集、并集）等场景中应用广泛，是处理无重复元素集合的高效工具。
