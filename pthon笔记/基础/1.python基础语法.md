Python3 是一种简洁、可读性强的编程语言，其基础语法注重直观性和易用性。以下是 Python3 核心基础语法的梳理，附带示例帮助理解：


### **1. 注释**
注释用于解释代码，不参与程序执行。  
- 单行注释：用 `#` 开头  
- 多行注释：用三个单引号 `'''` 或三个双引号 `"""` 包裹  


```python
# 这是单行注释

'''
这是
多行注释
'''

"""
这也是
多行注释
"""
```


### **2. 变量与数据类型**
Python 是动态类型语言，变量无需声明类型，直接赋值即可。


#### 基本数据类型：
- **整数（int）**：无小数点的数字（如 `10`、`-5`、`0`）  
- **浮点数（float）**：带小数点的数字（如 `3.14`、`-0.5`）  
- **字符串（str）**：文本，用单引号 `'` 或双引号 `"` 包裹（如 `'hello'`、`"Python"`）  
- **布尔值（bool）**：仅 `True`（真）或 `False`（假）  
- **空值（None）**：表示“无”，不同于 `0` 或空字符串  


```python
# 变量赋值
age = 18  # int
height = 1.75  # float
name = "Alice"  # str
is_student = True  # bool
empty = None  # NoneType
```


### **3. 变量命名规则**
- 只能包含字母、数字、下划线（`_`）  
- 不能以数字开头  
- 不能使用 Python 关键字（如 `if`、`for`、`def` 等）  
- 区分大小写（`name` 和 `Name` 是两个不同变量）  
- 建议使用小写字母，多个单词用下划线连接（如 `user_name`）  


### **4. 运算符**
#### （1）算术运算符
| 运算符 | 描述       | 示例          |
|--------|------------|---------------|
| `+`    | 加         | `3 + 2 = 5`   |
| `-`    | 减         | `5 - 1 = 4`   |
| `*`    | 乘         | `2 * 3 = 6`   |
| `/`    | 除（浮点数）| `5 / 2 = 2.5` |
| `//`   | 整除（取整）| `5 // 2 = 2`  |
| `%`    | 取余       | `5 % 2 = 1`   |
| `**`   | 幂运算     | `2**3 = 8`    |


#### （2）比较运算符（返回布尔值 `True`/`False`）
| 运算符 | 描述       | 示例               |
|--------|------------|--------------------|
| `==`   | 等于       | `3 == 3 → True`    |
| `!=`   | 不等于     | `3 != 5 → True`    |
| `>`    | 大于       | `5 > 2 → True`     |
| `<`    | 小于       | `1 < 0 → False`    |
| `>=`   | 大于等于   | `4 >= 4 → True`    |
| `<=`   | 小于等于   | `3 <= 2 → False`   |


#### （3）逻辑运算符（组合条件）
| 运算符 | 描述       | 示例                          |
|--------|------------|-------------------------------|
| `and`  | 与（都真才真）| `True and False → False`      |
| `or`   | 或（一真就真）| `True or False → True`        |
| `not`  | 非（取反）  | `not True → False`            |


#### （4）赋值运算符
基础赋值：`=`（如 `a = 10`）  
复合赋值：`+=`、`-=`、`*=` 等（如 `a += 5` 等价于 `a = a + 5`）  


### **5. 控制流语句**
#### （1）条件语句：`if-elif-else`
根据条件执行不同代码块（**缩进是 Python 的语法规则，通常用 4 个空格**）。

```python
score = 85

if score >= 90:
    print("优秀")
elif score >= 80:  # 否则如果
    print("良好")
else:  # 以上条件都不满足
    print("继续努力")

# 输出：良好
```


#### （2）循环语句
- **`for` 循环**：遍历序列（如列表、字符串）或范围。  
  语法：`for 变量 in 序列: 代码块`

```python
# 遍历字符串
for char in "Python":
    print(char)  # 依次输出 P、y、t、h、o、n

# 遍历范围（range(开始, 结束, 步长)，左闭右开）
for i in range(1, 5):  # 1,2,3,4
    print(i)
```


- **`while` 循环**：当条件为 `True` 时重复执行。  
  语法：`while 条件: 代码块`

```python
count = 0
while count < 3:
    print("count:", count)
    count += 1  # 等价于 count = count + 1

# 输出：
# count: 0
# count: 1
# count: 2
```


- 循环控制：  
  - `break`：跳出当前循环  
  - `continue`：跳过当前迭代，进入下一次循环  


```python
# 用 break 跳出循环
for i in range(10):
    if i == 3:
        break  # 当 i=3 时跳出循环
    print(i)  # 输出 0,1,2

# 用 continue 跳过迭代
for i in range(5):
    if i == 2:
        continue  # 跳过 i=2
    print(i)  # 输出 0,1,3,4
```


### **6. 基础数据结构**
#### （1）列表（list）：可变序列，用 `[]` 表示
可存储多个元素，支持增删改查。

```python
fruits = ["apple", "banana", "orange"]

# 访问元素（索引从 0 开始）
print(fruits[0])  # 输出：apple

# 修改元素
fruits[1] = "grape"
print(fruits)  # 输出：['apple', 'grape', 'orange']

# 新增元素
fruits.append("pear")  # 末尾添加
print(fruits)  # 输出：['apple', 'grape', 'orange', 'pear']

# 删除元素
fruits.pop(2)  # 删除索引 2 的元素
print(fruits)  # 输出：['apple', 'grape', 'pear']
```


#### （2）元组（tuple）：不可变序列，用 `()` 表示
元素一旦定义不能修改，适合存储固定数据。

```python
point = (3, 4)  # 坐标 (x,y)
print(point[0])  # 输出：3

# 元组不可修改（以下代码会报错）
# point[0] = 5  # 报错：'tuple' object does not support item assignment
```


#### （3）字典（dict）：键值对集合，用 `{}` 表示
通过“键”访问“值”，键唯一且不可变。

```python
student = {
    "name": "Bob",
    "age": 15,
    "is_male": True
}

# 访问值（通过键）
print(student["name"])  # 输出：Bob

# 修改值
student["age"] = 16
print(student["age"])  # 输出：16

# 新增键值对
student["score"] = 90
print(student)  # 输出：{'name': 'Bob', 'age': 16, 'is_male': True, 'score': 90}
```


#### （4）集合（set）：无序不重复元素，用 `{}` 或 `set()` 表示
适合去重或集合运算（交集、并集等）。

```python
nums = {1, 2, 2, 3}  # 自动去重
print(nums)  # 输出：{1, 2, 3}

# 交集（共同元素）
a = {1, 2, 3}
b = {2, 3, 4}
print(a & b)  # 输出：{2, 3}
```


### **7. 函数**
用 `def` 定义函数，封装可复用的代码。  
语法：`def 函数名(参数): 代码块; return 返回值`


```python
# 定义函数：计算两数之和
def add(a, b):
    result = a + b
    return result  # 返回结果

# 调用函数
sum = add(3, 5)
print(sum)  # 输出：8

# 带默认参数的函数
def greet(name="Guest"):  # name 默认值为 "Guest"
    print(f"Hello, {name}!")

greet("Alice")  # 输出：Hello, Alice!
greet()  # 输出：Hello, Guest!
```


### **8. 输入与输出**
- **`input()`**：获取用户输入（返回字符串类型）。  
- **`print()`**：输出内容（可多个参数，默认用空格分隔）。  


```python
# 获取输入（注意：input() 返回字符串，需转换类型）
age_str = input("请输入年龄：")  # 假设用户输入 "18"
age = int(age_str)  # 转换为整数

# 输出
print("你的年龄是：", age)  # 输出：你的年龄是： 18
print("姓名：", "Bob", "年龄：", 15)  # 输出：姓名： Bob 年龄： 15
```
