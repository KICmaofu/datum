Python 是一门支持**面向对象编程（OOP）** 的语言，其核心思想是将数据（属性）和操作数据的行为（方法）封装到“对象”中，通过类的定义实现代码的复用和逻辑的清晰化。


### 一、核心概念
- **类（Class）**：对一类事物的抽象描述（如“汽车”），包含该类事物的**属性**（特征，如颜色、型号）和**方法**（行为，如行驶、刹车）。  
- **对象（Object）**：类的具体实例（如“我的红色特斯拉”），是类的实际载体。  
- **封装（Encapsulation）**：将属性和方法包裹在类中，隐藏内部实现细节，仅通过公开接口交互。  
- **继承（Inheritance）**：子类可以继承父类的属性和方法，实现代码复用，并可扩展新功能。  
- **多态（Polymorphism）**：不同类的对象可以通过相同的接口（方法名）表现出不同的行为。  


### 二、类与对象的定义和使用
#### 1. 定义类
用 `class` 关键字定义类，类名通常采用“驼峰命名法”（首字母大写，如 `Person`）。  
类中至少包含一个 `__init__` 方法（构造方法），用于初始化对象的属性。


#### 2. 创建对象
通过“类名()”创建对象（实例化），会自动调用 `__init__` 方法初始化属性。


#### 3. 实例属性与实例方法
- **实例属性**：每个对象独有的属性（如每个人的姓名、年龄），通过 `self.属性名` 定义。  
- **实例方法**：类中定义的函数，第一个参数必须是 `self`（代表当前对象），用于操作实例属性。  


**示例：定义 `Person` 类并使用**
```python
# 定义类
class Person:
    # 构造方法：初始化实例属性
    def __init__(self, name, age):
        self.name = name  # 实例属性：姓名
        self.age = age    # 实例属性：年龄

    # 实例方法：对象的行为
    def say_hello(self):
        print(f"你好，我叫{self.name}，今年{self.age}岁")

    def grow_up(self):
        self.age += 1  # 修改实例属性
        print(f"{self.name}长大了一岁，现在{self.age}岁")


# 创建对象（实例化）
p1 = Person("Alice", 20)  # 自动调用__init__，传入name="Alice"，age=20
p2 = Person("Bob", 18)

# 访问实例属性
print(p1.name)  # 输出：Alice
print(p2.age)   # 输出：18

# 调用实例方法
p1.say_hello()  # 输出：你好，我叫Alice，今年20岁
p2.grow_up()    # 输出：Bob长大了一岁，现在19岁
```


### 三、类属性与类方法
除了实例独有的属性和方法，类还可以有**类属性**（所有实例共享）和**类方法**（操作类属性）。


#### 1. 类属性
定义在类中、`__init__` 方法外的属性，所有实例共享同一个值（如“人类的总数”）。  
通过“类名.属性名”或“实例.属性名”访问，推荐用类名访问。


#### 2. 类方法
用 `@classmethod` 装饰的方法，第一个参数是 `cls`（代表当前类），用于操作类属性。


**示例：类属性与类方法**
```python
class Person:
    # 类属性：所有实例共享，记录总人数
    total = 0

    def __init__(self, name):
        self.name = name
        Person.total += 1  # 每创建一个实例，总人数+1

    # 类方法：操作类属性
    @classmethod
    def show_total(cls):
        print(f"总人数：{cls.total}")  # cls.total 等价于 Person.total


# 创建实例
p1 = Person("Alice")
p2 = Person("Bob")

# 访问类属性
print(Person.total)  # 输出：2（通过类名访问）
print(p1.total)      # 输出：2（通过实例访问，不推荐）

# 调用类方法
Person.show_total()  # 输出：总人数：2
```


#### 3. 静态方法
用 `@staticmethod` 装饰的方法，**不需要 `self` 或 `cls` 参数**，更像类中的“工具函数”，与类或实例的状态无关。

```python
class MathUtils:
    @staticmethod
    def add(a, b):  # 无self/cls参数
        return a + b

    @staticmethod
    def multiply(a, b):
        return a * b

# 调用静态方法（通过类名或实例）
print(MathUtils.add(2, 3))      # 输出：5
print(MathUtils.multiply(2, 3)) # 输出：6
```


### 四、封装：隐藏与暴露
封装的核心是“隐藏内部细节，只暴露必要接口”。Python 没有严格的“私有”关键字，但通过命名约定实现：  
- 用**单下划线 `_`** 开头的属性/方法：约定为“私有”，不建议外部直接访问（但可强制访问）。  
- 用**双下划线 `__`** 开头的属性/方法：会触发“名称修饰”（实际变为 `_类名__属性名`），更难直接访问（伪私有）。  


**示例：封装**
```python
class BankAccount:
    def __init__(self, balance):
        self.__balance = balance  # 双下划线：伪私有属性（余额不希望被直接修改）

    # 暴露公开接口：查询余额
    def get_balance(self):
        return self.__balance

    # 暴露公开接口：存款
    def deposit(self, amount):
        if amount > 0:
            self.__balance += amount
            print(f"存款{amount}元，当前余额：{self.__balance}")

    # 私有方法：内部校验（不暴露）
    def __check_amount(self, amount):
        return amount > 0


account = BankAccount(1000)

# 不能直接访问__balance（会报错）
# print(account.__balance)  # AttributeError: 'BankAccount' object has no attribute '__balance'

# 通过公开接口访问
print(account.get_balance())  # 输出：1000
account.deposit(500)          # 输出：存款500元，当前余额：1500

# 强制访问（不推荐）：名称修饰后实际为 _BankAccount__balance
print(account._BankAccount__balance)  # 输出：1500
```


### 五、继承：代码复用与扩展
继承允许子类（派生类）继承父类（基类）的属性和方法，并可添加新属性/方法或重写父类方法。  
语法：`class 子类名(父类名):`  


#### 1. 基本继承
```python
# 父类：动物
class Animal:
    def __init__(self, name):
        self.name = name

    def eat(self):
        print(f"{self.name}在吃东西")

# 子类：狗（继承Animal）
class Dog(Animal):
    # 新增方法：狗的特有行为
    def bark(self):
        print(f"{self.name}在汪汪叫")

# 子类：猫（继承Animal）
class Cat(Animal):
    # 重写父类方法：猫的吃行为与动物不同
    def eat(self):
        print(f"{self.name}在优雅地吃鱼")

    # 新增方法
    def meow(self):
        print(f"{self.name}在喵喵叫")


# 创建子类实例
dog = Dog("旺财")
dog.eat()   # 继承父类方法 → 输出：旺财在吃东西
dog.bark()  # 子类新增方法 → 输出：旺财在汪汪叫

cat = Cat("咪咪")
cat.eat()   # 重写父类方法 → 输出：咪咪在优雅地吃鱼
cat.meow()  # 子类新增方法 → 输出：咪咪在喵喵叫
```


#### 2. 调用父类方法：`super()`
子类重写 `__init__` 时，需通过 `super()` 调用父类的 `__init__`，否则父类属性无法初始化。

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

# 子类Student继承Person
class Student(Person):
    def __init__(self, name, age, student_id):
        # 调用父类的__init__初始化name和age
        super().__init__(name, age)
        self.student_id = student_id  # 子类新增属性

    def introduce(self):
        print(f"我叫{self.name}，{self.age}岁，学号{self.student_id}")


s = Student("Alice", 20, "2023001")
s.introduce()  # 输出：我叫Alice，20岁，学号2023001
```


#### 3. 多继承
Python 支持多继承（一个子类继承多个父类），语法：`class 子类(父类1, 父类2):`。  
但需注意**方法解析顺序（MRO）**：当多个父类有同名方法时，按 MRO 顺序查找（可通过 `类名.__mro__` 查看）。

```python
class A:
    def show(self):
        print("A的show方法")

class B:
    def show(self):
        print("B的show方法")

class C(A, B):  # 继承A和B
    pass

c = C()
c.show()  # 输出：A的show方法（因A在B前面，MRO中A优先）
print(C.__mro__)  # 查看MRO：(<class '__main__.C'>, <class '__main__.A'>, <class '__main__.B'>, <class 'object'>)
```


### 六、多态：同一接口，不同实现
多态指“不同对象调用同名方法时，表现出不同行为”，依赖继承和方法重写实现。

```python
# 父类：形状
class Shape:
    def area(self):
        # 父类方法可定义为抽象方法（无实现，强制子类重写）
        raise NotImplementedError("子类必须重写area方法")

# 子类：圆形
class Circle(Shape):
    def __init__(self, radius):
        self.radius = radius

    def area(self):  # 重写area方法
        return 3.14 * self.radius **2

# 子类：矩形
class Rectangle(Shape):
    def __init__(self, width, height):
        self.width = width
        self.height = height

    def area(self):  # 重写area方法
        return self.width * self.height


# 多态体现：用父类类型接收不同子类对象，调用同名方法
def print_area(shape):  # shape参数可接收任何Shape子类对象
    print(f"面积：{shape.area()}")

# 传入Circle对象
print_area(Circle(2))    # 输出：面积：12.56
# 传入Rectangle对象
print_area(Rectangle(3, 4))  # 输出：面积：12
```


### 七、魔术方法（特殊方法）
Python 类中以 `__` 开头和结尾的方法（如 `__init__`）称为“魔术方法”，用于实现类的特殊行为（如字符串表示、运算符重载等）。


#### 常用魔术方法：
- `__str__(self)`：返回对象的“用户友好”字符串表示（`print()` 时调用）。  
- `__repr__(self)`：返回对象的“开发者友好”字符串表示（控制台直接输出对象时调用）。  
- `__add__(self, other)`：重载 `+` 运算符。  
- `__len__(self)`：重载 `len()` 函数。  


**示例：魔术方法**
```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    # 自定义print()输出
    def __str__(self):
        return f"《{self.title}》（{self.pages}页）"

    # 自定义控制台直接输出
    def __repr__(self):
        return f"Book(title='{self.title}', pages={self.pages})"

    # 重载len()：返回页数
    def __len__(self):
        return self.pages


book = Book("Python入门", 300)

print(book)      # 调用__str__ → 输出：《Python入门》（300页）
print(repr(book))# 调用__repr__ → 输出：Book(title='Python入门', pages=300)
print(len(book)) # 调用__len__ → 输出：300
```


### 八、面向对象总结
- **类与对象**：类是模板，对象是实例；通过 `class` 定义类，`__init__` 初始化属性。  
- **属性与方法**：实例属性/方法属于对象，类属性/方法属于类；静态方法与类/实例无关。  
- **三大特性**：  
  - 封装：通过命名约定隐藏内部细节，暴露公开接口。  
  - 继承：子类复用父类代码，可扩展或重写方法（`super()` 调用父类）。  
  - 多态：不同子类重写父类方法，通过统一接口表现不同行为。  
- **魔术方法**：自定义类的特殊行为（如字符串表示、运算符）。  

面向对象编程适合构建复杂程序，通过抽象和封装提高代码的复用性和可维护性。
