在 Django 中创建第一个项目非常简单，只需几步即可完成。以下是详细步骤（假设你已安装 Python，若未安装，需先安装 Python 3.6+）：


### **步骤 1：安装 Django**
首先确保已安装 Django。打开终端（Windows 用 cmd 或 PowerShell，Mac/Linux 用终端），执行以下命令：  
```bash
pip install django
```  
若需指定版本（如最新稳定版）：  
```bash
pip install django==4.2  # 4.2 是长期支持版本（LTS），推荐新手使用
```  

安装完成后，验证是否成功：  
```bash
django-admin --version  # 输出类似 4.2.7 即表示安装成功
```  


### **步骤 2：创建 Django 项目**
使用 `django-admin` 工具创建项目。假设项目名为 `myfirstproject`，执行：  
```bash
django-admin startproject myfirstproject
```  

执行后，会生成一个名为 `myfirstproject` 的文件夹，目录结构如下：  
```
myfirstproject/          # 项目根目录（可自定义名称）
├── myfirstproject/      # 项目核心配置目录（与项目名同名）
│   ├── __init__.py      # 标识该目录为 Python 包
│   ├── settings.py      # 项目全局配置（数据库、应用、中间件等）
│   ├── urls.py          # 项目主 URL 路由配置
│   ├── asgi.py          # 异步 Web 服务器入口（用于生产环境）
│   └── wsgi.py          # 同步 Web 服务器入口（用于生产环境）
└── manage.py            # 项目管理脚本（启动服务器、创建应用等）
```  


### **步骤 3：运行开发服务器**
进入项目根目录（`myfirstproject`），执行以下命令启动 Django 内置的开发服务器：  
```bash
cd myfirstproject  # 进入项目根目录
python manage.py runserver
```  

若终端显示类似以下内容，说明服务器启动成功：  
```
Starting development server at http://127.0.0.1:8000/
Quit the server with CONTROL-C.
```  

此时，打开浏览器访问 `http://127.0.0.1:8000/`，会看到 Django 的默认欢迎页面（一个火箭图标），表示项目创建成功！  


### **（可选）自定义服务器端口**
默认端口是 `8000`，若端口被占用，可指定其他端口（如 `8080`）：  
```bash
python manage.py runserver 8080
```  

也可允许其他设备访问（需替换为自己的 IP，如 `192.168.1.100`）：  
```bash
python manage.py runserver 192.168.1.100:8000
```  


### **步骤 4：创建第一个应用（App）**
Django 项目由多个「应用（App）」组成（如博客项目可分为 `article`、`user` 等应用）。创建一个名为 `myapp` 的应用：  
```bash
python manage.py startapp myapp
```  

生成的应用目录结构：  
```
myapp/
├── __init__.py
├── admin.py       # 配置 Admin 后台管理
├── apps.py        # 应用配置
├── migrations/    # 数据库迁移文件（自动生成）
├── models.py      # 数据模型（数据库表定义）
├── tests.py       # 测试代码
├── urls.py        # 应用内部 URL 路由（需手动创建）
└── views.py       # 视图（处理业务逻辑）
```  


### **步骤 5：注册应用到项目**
创建应用后，需在项目配置中注册，否则 Django 无法识别。打开 `myfirstproject/settings.py`，找到 `INSTALLED_APPS` 列表，添加应用名 `myapp`：  
```python
INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    # ... 其他默认应用 ...
    'myapp',  # 新增：注册我们的应用
]
```  


### **总结**
到这里，你已经完成了 Django 第一个项目的创建，包括：  
1. 安装 Django  
2. 创建项目（`django-admin startproject`）  
3. 启动开发服务器（`manage.py runserver`）  
4. 创建并注册应用（`startapp` + 修改 `settings.py`）  

接下来，你可以开始编写视图、配置 URL、设计数据模型等，逐步构建你的 Web 应用。
